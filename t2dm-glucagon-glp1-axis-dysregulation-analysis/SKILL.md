# T2DM胰高血糖素与GLP-1轴失调机制分析

## 适用对象
- 已确诊T2DM患者
- 餐后高血糖显著者
- 胰岛素治疗效果不佳者
- GLP-1受体激动剂治疗候选者

## 前提条件
- 已排除胰腺炎、胰腺切除等器质性病变
- 具备检测空腹及餐后胰高血糖素、GLP-1水平的能力

## 禁忌与限制
- 不适用于胰高血糖素瘤或其他α细胞肿瘤患者
- 严重肝肾功能不全者需谨慎解读GLP-1代谢数据

## 执行步骤

1. **确认正常调控机制**：
   - 进餐后血糖升高 → 刺激β细胞分泌胰岛素 + 肠道L细胞分泌GLP-1 → 抑制α细胞分泌胰高血糖素 → 减少肝糖输出 → 防止餐后高血糖。

2. **识别T2DM中的失调表现**：
   - 胰岛素分泌不足 → 对α细胞抑制减弱；
   - 胰岛α/β细胞比例升高（因β细胞丢失）；
   - α细胞对葡萄糖敏感性下降 → 无法感知高血糖而持续分泌胰高血糖素；
   - GLP-1分泌缺陷：糖负荷后GLP-1释放曲线低于正常。

3. **评估病理后果**：
   - 胰高血糖素分泌增多 → 肝糖异生和糖原分解增加 → 空腹及餐后高血糖加重。

4. **分析GLP-1缺失的多重影响**：
   - 胰岛素分泌减少（葡萄糖依赖性）；
   - 胰高血糖素抑制减弱；
   - 胃排空加速；
   - 食欲抑制减弱；
   - β细胞保护作用丧失；
   - 心血管保护效应下降。

5. **判断干预指征**：
   - 若确认存在GLP-1分泌缺陷或胰高血糖素失控，应优先考虑GLP-1受体激动剂治疗，以恢复血糖稳态调控。

## 触发条件
当T2DM患者存在空腹或餐后高血糖且胰岛素治疗效果不佳时，启动本技能进行机制评估。
