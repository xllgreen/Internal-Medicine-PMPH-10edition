# 肿瘤治疗相关心功能不全（CTRCD）的监测与诊断

## 适用对象
- 正在接受或已完成蒽环类、HER-2靶向药物、免疫检查点抑制剂（ICIs）等具有明确心血管毒性的抗肿瘤治疗的患者
- 已完成基线心脏评估（包括LVEF、GLS和心脏生物标志物）

## 触发条件
当出现以下任一情况时启动本技能：
- LVEF较基线下降 >10%
- LVEF <50%
- GLS较基线下降 >15%
- 心脏生物标志物（cTnI/TnT 或 BNP/NT-proBNP）升高

## 执行步骤

### 1. 辅助检查评估
- **超声心动图**：评估LVEF和GLS变化
- **心脏磁共振（CMR）**：适用于疑似ICIs相关心肌炎，评估心肌水肿或纤维化
- **心脏生物标志物检测**：cTnI/TnT提示心肌损伤；BNP/NT-proBNP辅助心衰诊断
- **心内膜心肌活检**（必要时）：用于确诊ICIs相关心肌炎，可见炎症细胞浸润伴心肌坏死

### 2. CTRCD分级诊断
根据临床症状与客观指标，按以下标准分级：

#### 有症状CTRCD（心衰）
- **极重度**：需正性肌力药、机械循环支持或心脏移植
- **重度**：需住院治疗
- **中度**：需门诊强化利尿和心衰治疗
- **轻度**：症状轻微，无需强化治疗

#### 无症状CTRCD
- **重度**：新发LVEF <40%
- **中度**：满足以下任一
  - LVEF下降 ≥10% 且 40% ≤ LVEF ≤ 49%
  - LVEF下降 <10% 但 40% ≤ LVEF ≤ 49%，同时 GLS下降 >15% 或 生物标志物升高
- **轻度**：LVEF ≥50%，但 GLS下降 >15% 和/或 生物标志物升高

### 3. 病因特异性注意事项
- **蒽环类药物**：剂量依赖性，可表现为急性、慢性或迟发性心功能不全，常不可逆
- **HER-2靶向药**：与剂量无关，停药后多可逆
- **免疫检查点抑制剂（ICIs）**：以心肌炎为主，死亡率高（39.7%~66.0%），需高度警惕并尽早干预

## 输出结果
返回CTRCD分级诊断结果（如“无症状中度CTRCD”）及对应的初步管理建议。
