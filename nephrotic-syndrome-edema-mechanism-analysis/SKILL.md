# 肾病综合征水肿形成的双重机制分析

## 适用场景
- 肾病综合征患者出现全身性水肿
- 已确认存在低白蛋白血症（通常<30g/L）
- 需制定个体化利尿或扩容方案

## 执行步骤
1. **评估主要机制**：
   - 若血浆白蛋白显著降低 → 血浆胶体渗透压下降 → 水分从血管内转移至组织间隙 → 形成水肿
2. **评估次要/辅助机制**：
   - 检查有效循环血容量状态：
     - 若血容量不足 → 激活RAAS → 远端肾小管水钠重吸收增加 → 加重水肿
     - 若血容量正常或增加 → 提示存在**原发性水钠潴留**（肾小管钠重吸收异常增加，不依赖RAAS）
3. **判断主导机制**：
   - 测定血浆肾素水平、中心静脉压或临床容量状态
   - 低肾素 + 容量正常/增多 → 支持原发性水钠潴留
4. **制定干预策略**：
   - **低蛋白主导型**：补充白蛋白 + 联合利尿剂
   - **原发水钠潴留型**：强化利尿，避免过度扩容以防心衰

## 注意事项
- 不适用于心源性、肝源性或其他非肾病性水肿
- 水肿严重程度不仅取决于白蛋白水平，也与肾脏水钠调节异常密切相关
