# 类风湿关节炎阶梯式药物治疗

## 适用场景
- 患者已确诊类风湿关节炎
- 已评估疾病活动度（如DAS28）、预后不良因素（如ACPA阳性、早期骨破坏）及合并症（如结核、乙肝、心血管病）

## 执行步骤
1. **启动传统DMARDs（第一步）**：
   - 确诊后尽早启用，首选甲氨蝶呤（MTX）
   - 可单用或联合（如MTX+来氟米特/柳氮磺吡啶/羟氯喹）
   - 起效需1–3个月，无显著镇痛作用，但可延缓骨破坏

2. **未达标或存在预后不良因素时升级治疗（第二步）**：
   - 加用生物DMARDs（如TNF-α抑制剂、IL-6拮抗剂、CD20单抗）：
     • 使用前筛查结核、乙肝、肿瘤
     • 推荐与MTX联用
   - 或加用靶向DMARDs（JAK抑制剂，如托法替布）：
     • 心血管高危患者慎用

3. **辅助用药管理**：
   - NSAIDs：仅缓解症状，必须与DMARDs联用；避免联用两种以上
   - 糖皮质激素：小剂量、短疗程“桥梁治疗”
     • 全身使用：用于急性期或关节外表现
     • 关节腔注射：单关节炎症，每年≤3次
     • 所有使用者补充钙剂和维生素D
   - 植物药（如雷公藤多苷）：监测性腺抑制、肝损、骨髓抑制

4. **监测与调整**：
   - 定期评估疾病活动度
   - 根据疗效和不良反应调整治疗

## 禁忌与警示
- TNF-α抑制剂禁用于活动性感染、未治疗结核、中重度心衰
- JAK抑制剂慎用于老年、吸烟、心血管高危人群
