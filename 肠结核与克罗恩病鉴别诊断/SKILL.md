# 肠结核与克罗恩病鉴别诊断

## 适用场景
- 患者表现为右下腹慢性腹痛
- 影像学或内镜发现回盲部病变
- 存在肠梗阻或腹泻与便秘交替症状
- 已完成结肠镜、影像学检查及结核相关实验室检查
- 尚未确诊为其他肠道疾病（如淋巴瘤、阿米巴病）

## 执行步骤

1. **评估肠外表现**
   - 若存在肺结核或其他肠外结核病灶，倾向肠结核；克罗恩病通常无肠外结核表现。

2. **观察病程特点**
   - 单次发病、复发少见 → 肠结核
   - 病程长、缓解与复发交替 → 克罗恩病

3. **检查并发症**
   - 瘘管、腹腔脓肿、肛周病变常见 → 克罗恩病
   - 上述并发症少见 → 肠结核

4. **分析内镜/影像学溃疡形态**
   - 环形、不规则溃疡 → 肠结核
   - 纵行溃疡、裂沟状 → 克罗恩病

5. **判断病变分布**
   - 局限于回盲部、无节段性 → 肠结核
   - 多节段性分布 → 克罗恩病

6. **解读实验室与病理检查**
   - 结核菌素试验强阳性、抗酸染色阳性、病理见干酪样肉芽肿 → 肠结核
   - 上述结果阴性或弱阳性、无干酪样坏死 → 克罗恩病

7. **实施诊断性治疗（必要时）**
   - 给予标准抗结核治疗2～6周：
     - 若症状明显改善，且2～3个月后肠镜显示病变好转，支持肠结核诊断
     - 若无反应，则更倾向克罗恩病

8. **注意事项**
   - 极少数情况下两者可共存，最终确诊依赖手术后病理。
   - 若诊断仍不确定，建议多学科会诊或重复活检。

> **输出提示**：根据上述评估，提示肠结核或克罗恩病的可能性，并建议下一步处理路径（如继续抗结核治疗、启动免疫抑制治疗、或安排手术病理确诊）。
