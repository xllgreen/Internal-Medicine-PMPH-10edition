# 功能性便秘的饮食与心理干预策略

## 何时使用
当满足以下条件时启动本干预：
- 患者已明确诊断为功能性便秘或慢性便秘；
- 已排除器质性病变（如肿瘤、肠梗阻、严重狭窄）；
- 无报警症状（如便血、黑粪、体重下降、贫血、腹部包块）；
- 患者具备基本生活自理能力。

## 执行步骤
1. **饮食调整**：
   - 推荐增加膳食纤维摄入（如麦麸、蔬菜、水果）；
   - 强调同步增加饮水量，以支持纤维吸水膨胀、软化粪便。

2. **行为干预**：
   - 建立每日定时排便习惯（建议晨起或餐后）；
   - 鼓励适度体能活动，避免久坐或长期卧床；
   - 明确告知避免滥用泻药，防止依赖。

3. **心理支持**：
   - 提供疾病教育，消除患者对“严重肠道疾病”的疑虑；
   - 关注焦虑、抑郁或睡眠障碍等心理因素，因其可通过自主神经影响排便节律；
   - 增强治疗信心与依从性。

4. **禁忌提醒**：
   - 若新出现或存在报警症状，立即暂停本干预，转诊进行器质性病因排查。

> **输出结果**：生成包含高纤维饮食、足量饮水、规律排便训练及心理疏导的个性化干预计划。
