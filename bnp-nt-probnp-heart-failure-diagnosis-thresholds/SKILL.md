# BNP/NT-proBNP在心力衰竭诊断与排除中的阈值应用

## 触发条件
当满足以下条件时启用本技能：
- 患者出现呼吸困难、水肿、乏力等疑似心力衰竭症状；
- 已采集血液样本并获得BNP或NT-proBNP检测结果。

## 执行步骤

### 1. 区分临床场景
- **急性心衰**：BNP/NT-proBNP具有高阴性预测价值。若结果低于排除阈值，基本可排除急性心衰。
- **慢性心衰**：BNP < 35 ng/L 或 NT-proBNP < 125 ng/L 可用于排除诊断，但敏感度和特异度低于急性情况。

### 2. 应用诊断/排除阈值
- **排除心衰**：BNP < 35 ng/L 或 NT-proBNP < 125 ng/L（慢性场景）；急性场景下阈值更低，具体参考最新指南。
- **支持心衰诊断**：BNP ≥ 35 ng/L 或 NT-proBNP ≥ 125 ng/L，但必须结合临床表现及其他检查综合判断。

### 3. 校正影响因素
- **年龄与性别**：老年人和女性BNP/NT-proBNP基线水平偏高。
- **肥胖**：可导致BNP/NT-proBNP水平偏低。
- **肾功能不全**：显著升高NT-proBNP，轻度升高BNP。
- **其他假阳性原因**：左心室肥厚、心动过速、心肌缺血、肺栓塞、感染、败血症等。

### 4. 特殊注意事项
- **HFpEF患者**：约20%的射血分数保留型心衰（HFpEF）患者BNP/NT-proBNP可处于正常范围，不可仅凭此排除诊断。
- **动态监测**：治疗后BNP/NT-proBNP下降提示预后改善；持续升高提示病情恶化或治疗无效。

## 输出建议
- 若BNP/NT-proBNP低于阈值且临床表现不符，可排除心衰；
- 若高于阈值，需结合影像学、病史和其他检查确认诊断；
- 若为HFpEF高危人群（如老年女性、高血压、房颤），即使标志物正常也不应完全排除心衰可能。
