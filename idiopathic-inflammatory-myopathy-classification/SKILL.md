# 特发性炎症性肌病ACR/EULAR 2017分类

## 适用场景
- 患者有对称性近端肌无力
- 已完成肌力评估、肌酶（CK/AST/ALT/LDH）和自身抗体检测
- 可选肌肉活检
- 已排除感染性、药物性、代谢性或遗传性肌病

## 执行步骤
1. **选择评分路径**：
   - **无活检路径**：基于临床、血清学、肌电图
   - **有活检路径**：额外加入病理特征

2. **关键评分项示例**：
   - 向阳疹：3.1（无活检）或3.2（有活检）
   - Gottron征：3.3或3.7
   - 抗Jo-1抗体阳性：3.9或3.8
   - 下肢近端肌无力：0.8
   - 镶边空泡（仅活检路径）：3.1
   - 肌酶升高：1.3或1.4

3. **计算总分并判定**：
   - **无活检路径**：
     • ≥7.5：确诊IIM（概率≥90%）
     • 5.5–<7.5：拟诊
     • 5.3–<5.5：可疑
     • <5.3：排除
   - **有活检路径**：
     • ≥8.7：确诊
     • 6.7–<8.7：拟诊
     • 6.5–<6.7：可疑
     • <6.5：排除

4. **后续分型**：根据特征分为皮肌炎（DM）、多发性肌炎（PM）、包涵体肌炎（IBM）等

## 注意事项
- 不适用于继发性肌炎
- 评分需严格遵循定义，避免主观偏差
