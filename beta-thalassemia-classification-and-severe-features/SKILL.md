# β珠蛋白生成障碍性贫血的分型与重型特征识别

## 使用前提
- 已确认存在β珠蛋白基因缺陷
- 已完成血红蛋白电泳（含HbA、HbA₂、HbF定量）
- 已评估贫血严重程度及器官受累情况

## 分型标准与操作步骤

### 1. 轻型
- **临床表现**：无症状或仅有轻度贫血，偶见轻度脾大。
- **实验室指标**：
  - HbA₂ > 3.5%（通常为4%～8%）
  - HbF正常或轻度升高（<5%）

### 2. 中间型
- **临床表现**：中度贫血、脾大；少数有轻度骨骼改变或性发育延迟。
- **实验室指标**：
  - 小细胞低色素性贫血
  - 靶形红细胞阳性
  - HbF可达10%

### 3. 重型（Cooley贫血，纯合子）
- **临床表现**：
  - 出生后半年起进行性苍白、贫血加重，伴黄疸、肝脾大
  - 生长发育迟缓
  - 骨质疏松甚至病理性骨折
  - 特殊面容（额隆、鼻梁凹陷、眼距增宽）
- **实验室指标**：
  - Hb < 60 g/L
  - 小细胞低色素性贫血
  - 靶形红细胞占10%～35%
  - 骨髓红系显著增生，铁沉积增多
  - 红细胞渗透脆性明显减低
  - 血红蛋白电泳：HbF 30%～90%，HbA < 40% 或为0
- **影像学特征**：X线显示颅骨板增厚、皮质变薄、骨小梁呈“短发直立状”

## 病理机制简述
β链合成减少 → α链过剩 → 未结合α链在红系前体细胞内沉淀 → 细胞膜损伤 → 骨髓内无效红细胞生成 + 外周红细胞被肝脾清除 → EPO代偿性升高 → 髓外造血 → 骨骼畸形与代谢紊乱。

## 输出行动建议
若患者符合重型标准，立即评估以下事项：
- 输血依赖状态
- 铁过载风险
- 是否具备造血干细胞移植条件
