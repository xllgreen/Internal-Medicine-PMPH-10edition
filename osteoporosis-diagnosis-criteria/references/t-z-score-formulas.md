# T值与Z值计算公式

## T值（适用于绝经后女性和≥50岁男性）

T值 = (实测骨密度 - 同种族同性别正常青年人峰值骨密度) / 同种族同性别正常青年人峰值骨密度的标准差

## Z值（适用于儿童、绝经前女性和<50岁男性）

Z值 = (骨密度测定值 - 同种族同性别同龄人骨密度均值) / 同种族同性别同龄人骨密度标准差

Z值 ≤ -2.0 被定义为“低于同年龄段预期范围”，但**不能单独用于诊断骨质疏松症**。

# 继发性骨质疏松常见病因排查清单

在确诊原发性骨质疏松前，需排除以下继发性原因：
- 内分泌疾病：甲亢、库欣综合征、甲状旁腺功能亢进、性腺功能减退等
- 胃肠疾病：乳糜泻、炎症性肠病、胃切除术后
- 血液系统疾病：多发性骨髓瘤、白血病
- 药物因素：长期使用糖皮质激素（≥3个月，剂量≥7.5 mg/d泼尼松）、过量甲状腺素、抗癫痫药、芳香化酶抑制剂、促性腺激素释放激素激动剂等
- 其他：慢性肾病、类风湿关节炎、长期制动、营养不良、吸烟酗酒等

建议结合病史、实验室检查（如血钙、磷、碱性磷酸酶、25-羟维生素D、PTH、睾酮/雌二醇、甲状腺功能等）进行综合评估。
