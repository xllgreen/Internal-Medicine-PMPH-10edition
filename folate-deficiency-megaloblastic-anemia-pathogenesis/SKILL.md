# 叶酸缺乏致巨幼细胞贫血发病机制分析

## 适用场景
- 患者疑似因叶酸缺乏导致贫血
- 出现“核幼浆老”形态的巨幼变细胞
- 伴有舌炎、食欲减退、腹泻等黏膜上皮萎缩表现
- 骨髓检查显示红系、粒系或巨核系细胞发育异常

## 执行步骤
1. **确认叶酸缺乏状态**：评估是否存在摄入不足、吸收障碍（如小肠疾病）、需求增加（如妊娠）、药物干扰（如甲氨蝶呤）或排泄增多。
2. **识别关键代谢中断点**：叶酸缺乏 → N⁵,N¹⁰-亚甲基四氢叶酸（N⁵,N¹⁰-亚甲基FH₄）减少 → 胸苷酸合成酶活性下降 → dTMP及dTTP合成不足。
3. **判断DNA合成障碍**：dTTP不足导致DNA复制延迟，而RNA合成相对正常，造成RNA/DNA比值升高。
4. **识别巨幼变特征**：细胞体积增大，胞核发育滞后于胞质（“核幼浆老”），主要累及增殖活跃组织（骨髓造血细胞、黏膜上皮）。
5. **关联临床表现**：骨髓内无效造血 → 全血细胞减少；黏膜上皮更新障碍 → 舌炎、腹泻等症状。
6. **排除混杂因素**：若同时存在维生素B₁₂缺乏，需另行评估，因其机制重叠但治疗不同。

## 输出建议
提示存在DNA合成障碍相关贫血，应检测血清及红细胞叶酸水平，并结合骨髓形态学与临床表现综合判断。
