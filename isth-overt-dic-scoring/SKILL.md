# ISTH显性DIC诊断积分系统

## 适用条件
- 患者存在与DIC相关的潜在疾病（如感染、肿瘤、产科并发症等）
- 已完成以下实验室检查：血小板计数、凝血酶原时间（PT）、纤维蛋白原、FDP或D-二聚体

## 不适用情况
- 无明确基础疾病的出血患者
- 慢性DIC（本系统主要适用于急性显性DIC）

## 执行步骤
1. **确认基础疾病**：核实是否存在DIC相关诱因（如脓毒症、恶性肿瘤、胎盘早剥等）。若无，则不适用本评分系统。
2. **四项指标评分**：
   - **血小板计数（×10⁹/L）**：
     - >100 → 0分
     - ≤100 → 1分
     - ≤50 → 2分
   - **纤溶标志物（FDP或可溶性纤维蛋白单体升高倍数 vs 正常值）**：
     - 无增加 → 0分
     - 中度增加（5～10倍）→ 2分
     - 显著增加（>10倍）→ 3分
   - **PT延长（秒，较正常对照）**：
     - <3秒 → 0分
     - 3～6秒 → 1分
     - >6秒 → 2分
   - **纤维蛋白原浓度（g/L）**：
     - >1.0 → 0分
     - ≤1.0 → 1分
3. **计算总分并判断**：
   - 总分 ≥5 → 符合显性DIC诊断，需紧急干预
   - 总分 <5 → 不支持显性DIC，但应在1～2天内重复评分以动态监测

## 注意事项
- 纤维蛋白原为急性时相反应蛋白，早期可能正常甚至升高，不能单独用于排除DIC。
- D-二聚体特异性高于FDP，推荐优先使用作为纤溶标志物。
- 该评分系统综合反映凝血因子消耗、血小板消耗及纤溶亢进三大病理环节。
