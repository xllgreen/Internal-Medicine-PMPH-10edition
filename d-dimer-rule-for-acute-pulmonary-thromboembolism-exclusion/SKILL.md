# D-二聚体用于急性肺血栓栓塞症排除诊断规则

## 何时使用
- 患者被评估为**低度临床可能性**的急性PTE疑诊病例
- 已采用**高敏感度检测方法**（如酶联免疫吸附试验、酶联免疫荧光分析、高敏感度定量微粒凝集法或免疫化学发光分析）
- 需快速决定是否可安全排除PTE，避免不必要的影像学检查

## 执行步骤
1. **确认临床可能性**：确保患者经临床评估（如Wells评分或修订版Geneva评分）判定为低度可能性。
2. **执行D-二聚体检测**：必须使用高敏感度检测方法。
3. **判断结果**：
   - 若患者年龄 ≤ 50岁，且 D-二聚体 < 500 μg/L → 可基本排除急性PTE。
   - 若患者年龄 > 50岁，使用年龄调整临界值：**临界值 = 年龄 × 10 μg/L**；若实测值低于该值 → 可基本排除急性PTE。
4. **警惕假阳性**：若存在恶性肿瘤、炎症、出血、创伤、手术或组织坏死等情况，即使D-二聚体升高，也不能确诊PTE；此时应结合临床动态观察，必要时安排影像学检查（如CT肺动脉造影）。
5. **输出结论**：
   - 满足排除条件 → 输出“可基本排除急性PTE”
   - 不满足 → 提示“需进一步影像学检查以明确诊断”

> 注意：本规则仅适用于低度临床可能性患者。中/高度可能性患者无论D-二聚体结果如何，均应直接进行影像学确诊。
