# 多发性骨髓瘤R-ISS分期

## 何时使用
- 患者已确诊为**活动性多发性骨髓瘤**（非冒烟性骨髓瘤）
- 已完成以下四项检测：
  - 血清β2微球蛋白
  - 血清白蛋白
  - 血清LDH水平
  - FISH细胞遗传学检测（评估del(17p)、t(4;14)、t(14;16)）

## 执行步骤
1. **先按传统ISS标准确定基础分期**：
   - ISS I期：β2微球蛋白 < 3.5 mg/L 且 白蛋白 ≥ 35 g/L
   - ISS III期：β2微球蛋白 ≥ 5.5 mg/L
   - ISS II期：不符合I期或III期标准

2. **结合细胞遗传学高危状态和LDH水平升级为R-ISS分期**：
   - R-ISS I期：ISS I期 + **无**细胞遗传学高危异常 + LDH正常
   - R-ISS III期：ISS III期 + (**存在**细胞遗传学高危异常 **或** LDH升高)
   - R-ISS II期：其余所有情况

3. **确认细胞遗传学高危定义**（需通过间期FISH检测）：
   - del(17p)
   - t(4;14)
   - t(14;16)

4. **输出最终R-ISS分期**（I、II或III期），用于预后评估和治疗决策。

> 注意：若任一必要参数缺失，无法准确完成R-ISS分期。
