# 感染性心内膜炎影像学阳性诊断标准

## 何时使用
- 患者有发热、心脏杂音、栓塞表现等提示感染性心内膜炎的临床征象；
- 已完成初步病史采集与体格检查；
- 需要确认是否满足改良Duke标准中的“影像学阳性证据”这一主要诊断标准。

> 注意：本技能仅用于诊断评估，不用于治疗决策。若影像学阴性但临床高度怀疑IE，仍需结合微生物学及其他临床标准综合判断。

## 如何执行

依次评估以下影像学检查结果，**任一技术检出至少一项下列病变即可判定为“影像学阳性”**：

1. **超声心动图（TTE/TEE）阳性表现**：
   - 赘生物；
   - 脓肿、假性动脉瘤或心脏内瘘；
   - 瓣膜穿孔或动脉瘤；
   - 新发人工瓣膜部分破裂。

2. **心脏CT或CTA阳性表现**：
   - 在自体瓣膜或人工瓣膜周围发现明确病灶（如脓肿、瘘管等）。

3. **¹⁸F-FDG PET-CT阳性表现**：
   - 在自体或人工瓣膜周围区域显示异常高代谢活动（提示活动性炎症/感染）。

4. **放射标记白细胞SPECT-CT阳性表现**：
   - 在自体或人工瓣膜周围区域显示局灶性放射性浓聚（提示活动性感染）。

### 特别提示
- 对于**人工瓣膜心内膜炎（PVE）患者**，优先依赖**经食管超声心动图（TEE）** 或 **心脏CT/CTA**，因其对瓣周并发症的敏感性更高。
- 影像学阳性结果可直接作为改良Duke诊断标准中的一项**主要标准**。

## 输出
若满足上述任一条件，输出：“影像学阳性，符合感染性心内膜炎主要诊断标准之一”。否则输出：“未见明确影像学阳性证据，需结合其他标准综合评估”。
