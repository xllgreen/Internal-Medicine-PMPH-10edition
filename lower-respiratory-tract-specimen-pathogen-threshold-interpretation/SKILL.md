# 下呼吸道标本病原菌判定阈值解读

## 何时使用
- 已采集下呼吸道标本并完成细菌培养
- 需区分致病菌与口咽部定植/污染菌
- 患者存在呼吸道感染症状但病原学证据不明确

## 执行步骤

### 1. 标本质量控制（仅适用于痰标本）
- **合格标准（满足任一即可）**：
  - 每低倍视野鳞状上皮细胞＜10个且白细胞＞25个
  - 鳞状上皮细胞：白细胞比例＜1:2.5
- 若不合格，结果不可靠，应重新采集。

### 2. 按标本类型应用致病菌判定阈值
- **普通痰标本**：
  - ≥10⁵ cfu/ml → 致病菌
  - ≤10⁴ cfu/ml → 污染菌
  - 10⁴～10⁵ cfu/ml → 建议重复培养；若连续2次以上分离相同细菌且浓度达10⁵～10⁶ cfu/ml，可认定为致病菌
- **经支气管镜或人工气道吸引物**：
  - ≥10⁵ cfu/ml → 致病菌
  - ＜10⁵ cfu/ml → 多为污染菌
- **防污染毛刷（PSB）**：
  - ≥10³ cfu/ml → 致病菌
- **支气管肺泡灌洗液（BAL）**：
  - 普通BAL：≥10⁴ cfu/ml → 致病菌
  - 防污染BAL：≥10³ cfu/ml → 致病菌

### 3. 关键注意事项
- 必须结合临床表现、影像学及其他检查综合判断
- 抗菌药物使用后可能抑制细菌生长，影响结果准确性
- 老年人、危重患者及慢性气道疾病者定植菌多，需更严格判读
- 本方法仅适用于细菌性病原体，不适用于病毒或非典型病原体
