# 白血病分类框架（急慢性与细胞系列）

## 触发条件
当满足以下条件时启用本技能：
- 患者已确诊为白血病
- 已获得骨髓或外周血细胞形态学数据

## 执行步骤

1. **判断急慢性类型**：
   - **急性白血病（AL）**：细胞分化停滞在早期阶段（以原始细胞及早期幼稚细胞为主），自然病程仅数月
   - **慢性白血病（CL）**：细胞分化停滞在较晚阶段（以较成熟幼稚细胞和成熟细胞为主），自然病程为数年

2. **细分急性白血病亚型**：
   - **急性淋巴细胞白血病（ALL）**：起源于淋巴系
   - **急性髓系白血病（AML）**：起源于髓系

3. **识别慢性白血病主要类型**：
   - 慢性髓系白血病（CML）
   - 慢性淋巴细胞白血病（CLL）
   - 少见类型：毛细胞白血病、幼淋巴细胞白血病等

4. **后续行动**：此分类仅为临床初步框架，必须结合WHO或FAB分型及MICM（形态学、免疫学、细胞遗传学、分子生物学）进一步细化。

## 输出结果
输出白血病大类（AL 或 CL）及其亚型（如 AML、ALL、CML、CLL），用于指导后续精准分型和治疗规划。
