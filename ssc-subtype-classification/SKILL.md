# 系统性硬化症亚型分类

## 触发条件
当患者存在以下任一表现时启动分类流程：
- 雷诺现象
- 皮肤硬化
- 内脏器官受累（如肺动脉高压、指端溃疡、消化道运动障碍等）
- 血清学异常（如自身抗体阳性）

## 分类步骤

1. **评估皮肤病变范围**
   - 若皮肤病变局限于肘、膝远端（不超过肘膝关节），可累及颜面和颈部 → 考虑**局限皮肤型 SSc**
   - 若皮肤病变广泛，累及肢体近端、胸部或腹部 → 考虑**弥漫皮肤型 SSc**
   - 若**无任何临床可见的皮肤硬化**，但具备雷诺现象、特征性内脏表现及血清学异常 → 考虑**无皮肤硬化的 SSc（sine scleroderma）**

2. **检测血清学抗体**
   - 抗着丝点抗体（ACA）阳性 → 支持局限皮肤型
   - 抗 Scl-70 抗体阳性 → 支持弥漫皮肤型
   - 其他抗体（如抗RNA聚合酶Ⅲ）可辅助判断

3. **评估内脏受累情况**
   - 局限皮肤型常合并肺动脉高压、严重消化道病变
   - 弥漫皮肤型多伴有快速进展的内脏病变（肺、心、肾等）

4. **排查重叠综合征**
   - 若患者同时符合另一种结缔组织病（如类风湿关节炎、系统性红斑狼疮、炎症性肌病）的诊断标准 → 归类为**SSc 重叠综合征**，可出现在任一亚型中

5. **输出最终亚型**
   - 明确归入以下四类之一：
     - 局限皮肤型 SSc
     - 弥漫皮肤型 SSc
     - 无皮肤硬化的 SSc
     - SSc 重叠综合征

> 注意：CREST 综合征（钙化、雷诺现象、食管运动障碍、硬指、毛细血管扩张）主要见于局限皮肤型，但也可出现在弥漫型，不单独作为分类依据。

## 使用限制
- 不适用于仅表现为皮肤硬化的非 SSc 疾病（如硬斑病、糖尿病性硬肿病等）
- 必须结合临床表现、抗体谱和内脏评估综合判断，不可仅凭单一指标分类
