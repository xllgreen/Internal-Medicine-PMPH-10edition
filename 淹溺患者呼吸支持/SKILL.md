# 淹溺患者呼吸支持

## 触发条件
患者为淹溺者，经高流量吸氧后SpO₂<90%或PaO₂<60mmHg。

## 执行步骤
1. **初始评估**：
   - 对所有淹溺患者评估呼吸状态和氧合情况
   - 清醒且无严重呼吸窘迫者：使用CPAP

2. **升级呼吸支持指征**：
   - SpO₂低于90%
   - PaO₂低于60mmHg

3. **机械通气指征**：
   - PaCO₂超过50mmHg
   - 严重或进行性呼吸窘迫
   - 缺乏气道反射保护（昏迷、呕吐反射消失）
   - 合并头胸部损伤

4. **干预阶梯**：
   - 普通吸氧 → CPAP → 气管插管+机械通气

5. **综合评估**：
   - 所有干预措施需结合临床整体评估，避免延误气道管理
