# ACTH兴奋试验诊断肾上腺皮质功能减退

## 适用场景
- 疑似原发性或继发性肾上腺皮质功能减退
- 患者病情较重但尚未接受充分替代治疗
- 急性危象期需同步诊断与治疗

## 执行步骤
1. **静脉注射人工合成ACTH（1-24）250μg**
2. **分别于注射前、30分钟、60分钟采集血样测皮质醇**
3. **判断结果**：
   - 血皮质醇峰值＜500nmol/L → 提示肾上腺皮质功能减退
   - 非患者应有明显皮质醇上升
4. **重症患者处理**：可同步静脉给予地塞米松和ACTH，既诊断又预防肾上腺危象
5. **结合基础ACTH水平鉴别病因**：
   - 原发性：基础ACTH显著升高（88–440 pmol/L）
   - 继发性：基础ACTH降低（<18 pmol/L）

## 输出要求
返回血皮质醇峰值数值及是否低于500nmol/L的判断，并结合基础ACTH水平明确区分原发性（ACTH↑）与继发性（ACTH↓）肾上腺皮质功能减退。
