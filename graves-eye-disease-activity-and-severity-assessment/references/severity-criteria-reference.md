# GO严重度分级参考标准

## EUGOGO 2021标准操作定义

### 轻度GO
- **眼睑退缩 < 2 mm**：上睑缘位于角膜缘上方 < 2 mm（睁眼平视）。
- **轻度软组织受累**：晨起轻度眼睑水肿，日间缓解，无明显红斑。
- **突眼 ≤ 正常上限 + 3 mm**：亚洲人群正常上限通常为16–18 mm，高加索人为20–21 mm。
- **一过性复视**：仅在极端注视方向出现，不影响日常阅读或行走。
- **角膜暴露可缓解**：人工泪液每日 ≤ 4 次即可控制干眼症状。

### 中重度GO
- **眼睑退缩 ≥ 2 mm**：上睑缘 ≥ 角膜缘上方2 mm。
- **中/重度软组织受累**：持续性眼睑红肿、结膜充血，影响外观或舒适度。
- **突眼 ≥ 正常上限 + 3 mm**：例如亚洲患者 ≥ 21 mm。
- **间歇性/持续性复视**：在日常活动中（如阅读、下楼梯）出现复视。

### 极重度GO（威胁视力）
- **DON诊断标准**：
  - 视力下降（≥1行）
  - 色觉异常（Ishihara测试错误 ≥ 2 片）
  - 相对性瞳孔传入障碍（RAPD阳性）
  - 视野缺损（中心或周边）
  - OCT显示视神经纤维层变薄
- **严重暴露性角膜病变**：
  - 持续性角膜上皮缺损
  - 角膜溃疡或即将穿孔
  - 需频繁润滑（>每小时1次）或睑裂缝合

## ATA 0–6级标准补充说明
- **3级（眼球突出）**：突眼 > 18 mm（高加索人）或 > 16 mm（亚洲人）
- **4级（眼外肌受累）**：CT/MRI显示眼外肌增粗，伴复视
- **5级（角膜受累）**：角膜点状染色、丝状角膜炎
- **6级（视力丧失）**：因DON导致不可逆视力损害
