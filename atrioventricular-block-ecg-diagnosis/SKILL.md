# 房室传导阻滞的心电图分度诊断

## 适用条件
- 心电图记录中存在可识别的P波和QRS波群
- 已明确测量PR间期
- 可判断P波与QRS波群的传导关系

> **注意**：本技能不适用于窦房传导阻滞，也不适用于无P波的心律失常（如未控制的心房颤动）。

## 诊断流程

### 1. 判断是否为一度房室传导阻滞
- 检查每个P波后是否均有QRS波群（无脱漏）
- 测量PR间期是否 > 0.20秒
- 确认QRS波群形态与时限正常

### 2. 若存在QRS脱漏，判断是否为二度房室传导阻滞
- **若PR间期逐搏进行性延长，直至一个P波后QRS脱漏** → 诊断为**二度Ⅰ型（文氏阻滞）**
- **若PR间期恒定，但部分P波后突然无QRS波群** → 诊断为**二度Ⅱ型**

### 3. 若连续两个或以上P波不能下传
- 判断是否为**高度房室传导阻滞**（常见传导比例如2:1或3:1）

### 4. 若P波与QRS波群完全无关
- 确认心房率 > 心室率
- 判断是否存在房室分离
- 若满足，则诊断为**三度（完全性）房室传导阻滞**

### 5. 推测阻滞部位
- **QRS波群正常**：阻滞可能位于房室结
- **QRS波群增宽、畸形**：阻滞位于希氏束-浦肯野系统
- **三度阻滞时心室率 < 40次/分**：提示起搏点在希氏束以下

## 输出结果
输出房室传导阻滞的具体分度（一度、二度Ⅰ型、二度Ⅱ型、高度、三度）及其阻滞部位推测（房室结 vs 希氏束以下），用于指导临床治疗决策。
