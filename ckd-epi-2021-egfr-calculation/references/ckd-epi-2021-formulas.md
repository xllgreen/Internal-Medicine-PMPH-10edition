# CKD-EPI 2021 公式详细说明

## 一、仅使用肌酐的公式

### 女性
- 若 Scr ≤ 0.7 mg/dl：
  ```
  eGFR = 144 × (Scr/0.7)^(-0.241) × 0.994^(Age)
  ```
- 若 Scr > 0.7 mg/dl：
  ```
  eGFR = 144 × (Scr/0.7)^(-1.200) × 0.994^(Age)
  ```

### 男性
- 若 Scr ≤ 0.9 mg/dl：
  ```
  eGFR = 142 × (Scr/0.9)^(-0.302) × 0.994^(Age)
  ```
- 若 Scr > 0.9 mg/dl：
  ```
  eGFR = 142 × (Scr/0.9)^(-1.200) × 0.994^(Age)
  ```

## 二、联合肌酐与Cystatin C的公式

### 女性
- Scr ≤ 0.7 且 Scys ≤ 0.8：
  ```
  eGFR = 130 × (Scr/0.7)^(-0.219) × (Scys/0.8)^(-0.323) × 0.996^(Age)
  ```
- Scr ≤ 0.7 且 Scys > 0.8：
  ```
  eGFR = 130 × (Scr/0.7)^(-0.219) × (Scys/0.8)^(-0.778) × 0.996^(Age)
  ```
- Scr > 0.7 且 Scys ≤ 0.8：
  ```
  eGFR = 130 × (Scr/0.7)^(-0.548) × (Scys/0.8)^(-0.323) × 0.996^(Age)
  ```
- Scr > 0.7 且 Scys > 0.8：
  ```
  eGFR = 130 × (Scr/0.7)^(-0.544) × (Scys/0.8)^(-0.778) × 0.996^(Age)
  ```

### 男性
- Scr ≤ 0.9 且 Scys ≤ 0.8：
  ```
  eGFR = 135 × (Scr/0.9)^(-0.144) × (Scys/0.8)^(-0.323) × 0.996^(Age)
  ```
- Scr ≤ 0.9 且 Scys > 0.8：
  ```
  eGFR = 135 × (Scr/0.9)^(-0.144) × (Scys/0.8)^(-0.778) × 0.996^(Age)
  ```
- Scr > 0.9 且 Scys ≤ 0.8：
  ```
  eGFR = 135 × (Scr/0.9)^(-0.544) × (Scys/0.8)^(-0.323) × 0.996^(Age)
  ```
- Scr > 0.9 且 Scys > 0.8：
  ```
  eGFR = 135 × (Scr/0.9)^(-0.444) × (Scys/0.8)^(-0.778) × 0.996^(Age)
  ```

## 注意事项
- 所有Scr必须经IDMS标准化方法检测
- Age为整数（向下取整）
- 该公式于2021年由CKD-EPI工作组发布，移除了种族校正因子
- 联合公式在eGFR < 60 ml/(min·1.73m²)时准确性显著优于单一肌酐公式
