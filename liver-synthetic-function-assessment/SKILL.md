# 肝脏合成功能评估

## 触发条件
当患者出现以下情况之一时启动本技能：
- 已知或疑似慢性肝病、肝硬化
- 出现腹水、黄疸、凝血障碍等肝功能失代偿表现
- 实验室检查提示肝功能异常

## 执行步骤

1. **评估血清白蛋白**
   - 获取当前血清白蛋白值（单位：g/L）
   - 注意：该指标反映长期合成功能，但受营养状态及急性应激（如感染、出血、手术）显著影响；若近期有应激事件，即使数值正常也不能排除合成功能受损

2. **评估凝血因子合成状态**
   - 检查凝血酶原时间（PT）、部分活化凝血酶原时间（APTT）、凝血酶时间（TT）
   - 重点关注PT延长秒数（对照值基础上的延长值）
   - 注意：维生素K依赖性凝血因子（Ⅱ、Ⅶ、Ⅸ、Ⅹ）半衰期短，可在白蛋白仍正常时已显著减少，是早期肝合成功能受损的敏感指标

3. **评估胆固醇水平**
   - 获取总胆固醇值
   - 注意：约70%内源性胆固醇由肝脏合成，合成功能严重受损时可下降

4. **整合其他相关指标**
   - 获取总胆红素（μmol/L），虽主要反映代谢/排泄功能，但在Child-Pugh评分中作为综合指标使用

5. **综合判断原则**
   - 不得单独依赖任一指标判断整体肝功能
   - 必须结合临床症状、体征、影像学及病理结果
   - 将上述指标输入Child-Pugh评分系统，确定肝功能分级（A/B/C级），指导治疗决策（如手术风险评估、肝移植指征等）

> **注意**：急性应激事件可导致白蛋白骤降，不代表慢性合成功能衰竭；此时应优先评估凝血指标以判断真实肝功能状态。
