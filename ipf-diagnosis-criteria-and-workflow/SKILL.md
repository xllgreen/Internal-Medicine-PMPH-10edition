# 特发性肺纤维化(IPF)的诊断标准与流程

## 触发条件
当满足以下条件时启动本流程：
- 患者表现为进行性呼吸困难伴限制性通气障碍
- HRCT显示间质性肺改变
- 已完成胸部HRCT检查
- 已排除结缔组织病、药物性肺损伤、职业暴露等已知病因

## 执行步骤

### 1. 确认初步条件
- 确诊为间质性肺疾病（ILD）
- 排除其他已知病因（如环境暴露、药物、结缔组织病等）

### 2. 根据HRCT表现分类处理

**A. 典型UIP表现（满足全部）：**
- 网格影、蜂窝样改变 ± 牵拉性支气管扩张
- 病变以胸膜下、基底部分布为主
→ 与病理UIP符合率 >90%，可临床诊断IPF，无需活检

**B. 可能UIP表现（满足以下）：**
- 网格改变伴牵拉支气管扩张
- 缺乏蜂窝样改变
- 胸膜下、基底部分布为主
→ 与病理UIP符合率70%–89%，需结合临床判断，建议多学科讨论

**C. 不确定UIP或非UIP表现：**
→ 必须进行外科肺活检（SLB）或经支气管冷冻肺活检（TBLC）

### 3. 若行活检，应用病理UIP标准
- 致密纤维化伴肺结构破坏（蜂窝/瘢痕）
- 胸膜下和/或间隔旁分布为主的纤维化
- 斑片状肺实质纤维化
- 成纤维细胞灶

### 4. 最终诊断
- 必须经过多学科讨论（MDD），整合临床、影像、病理资料
- 仅当符合上述标准且排除其他ILD时，方可诊断为IPF

### 5. 注意事项
- UIP模式也可见于石棉肺、过敏性肺炎、CTD-ILD等，必须鉴别
- BALF中性粒细胞/嗜酸粒细胞增多支持IPF，但无特异性
- 经支气管肺活检（TBLB）对IPF无诊断价值；TBLC在有经验中心可替代SLB

## 输出建议
若符合IPF诊断标准，应启动抗纤维化治疗并评估肺移植指征；若诊断不确定，需进一步检查或转诊至ILD专科中心。
