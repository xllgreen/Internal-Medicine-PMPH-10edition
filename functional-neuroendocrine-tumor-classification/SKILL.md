# 功能性神经内分泌肿瘤与异位激素综合征鉴别分类

## 适用场景
- 存在神经内分泌肿瘤（胰腺、胃肠道、肺等）
- 出现无法用原发病解释的内分泌综合征
- 激素水平异常升高

## 分类与诊断流程

### 第一步：判断是否为功能性神经内分泌肿瘤（F-NEN）
1. 检测肿瘤是否分泌激素或其产物。
   - 若否 → 归为非功能性神经内分泌肿瘤（NF-NEN）
   - 若是 → 进入下一步
2. 判断该激素是否具有生物学活性并引发典型临床综合征（如胰岛素瘤致低血糖、胃泌素瘤致ZES）。
   - 若激素无活性（仅为前体或片段）→ 仍为NF-NEN
   - 若有活性且伴典型症状 → 归为F-NEN
3. 注意：部分NF-NEN可在疾病进展中转化为F-NEN，需定期监测

### 第二步：若激素来源于非内分泌组织，启动异位激素分泌综合征诊断
确认以下五项依据全部满足：
1. 肿瘤来源于**非正常分泌该激素的组织**（如肺癌分泌ACTH）
2. 激素水平异常升高，且该组织正常状态下**不合成或分泌该激素**
3. 激素分泌呈**自主性**，不能被正常负反馈抑制（如异位ACTH不受地塞米松抑制）
4. 肿瘤经特异性治疗后，激素水平下降，症状缓解
5. **排除其他原因**（如原发内分泌疾病、药物等）

## 输出结果
- 返回分类：'功能性神经内分泌肿瘤（F-NEN）'、'非功能性神经内分泌肿瘤（NF-NEN）' 或 '异位激素分泌综合征'

## 关键变量
- `激素活性`（boolean）：分泌的激素是否具有生物学活性
- `临床症状`（string）：是否存在典型内分泌综合征
- `肿瘤来源`（string）：是否起源于非内分泌组织
- `激素反馈抑制`（boolean）：是否可被正常反馈机制抑制
- `治疗反应`（boolean）：肿瘤治疗后激素是否下降
