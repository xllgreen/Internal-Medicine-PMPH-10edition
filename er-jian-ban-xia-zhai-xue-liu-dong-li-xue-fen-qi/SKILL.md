# 二尖瓣狭窄的血流动力学分期与症状演变

## 使用前提
- 患者已确诊为风湿性二尖瓣狭窄
- 已通过心导管或经食道超声测量二尖瓣口面积（MVA）

## 禁忌场景
- 先天性或退行性二尖瓣狭窄（机制不同）
- 合并显著二尖瓣反流（血流动力学更复杂，需额外评估）

## 执行步骤

1. **确认二尖瓣口面积（MVA）分级**：
   - 轻度狭窄：MVA > 2.5 cm²
   - 中度狭窄：MVA 1.6～2.5 cm²
   - 重度狭窄：MVA ≤ 1.5 cm²

2. **根据MVA和临床表现判断血流动力学阶段**：
   - **早期（轻度狭窄）**：仅在心率增快（如运动、感染、妊娠、房颤）时左心房压升高，静息状态正常。
   - **中期（中度狭窄）**：静息状态下左心房压持续升高 → 肺静脉压增高 → 劳力性呼吸困难。
   - **晚期（重度狭窄）**：肺静脉高压 → 肺动脉高压 → 右心室肥厚扩张 → 右心衰竭。
   - **终末期**：右心输出减少致肺淤血减轻，但体循环淤血（肝大、腹水、下肢水肿）加重。

3. **识别关键并发症警示信号**：
   - 咯血：提示肺静脉压骤升
   - 房颤：左心房扩大诱发，进一步降低心排血量
   - 静息呼吸困难、右心衰体征：提示可能已进入不可逆阶段

4. **决定干预时机**：
   - 若 MVA ≤ 1.5 cm² 且有症状，应考虑介入或手术
   - 若无症状，需密切监测肺动脉压和右心功能

> **注意**：一旦出现静息呼吸困难、咯血或右心衰体征，提示疾病可能已进入不可逆阶段，需紧急评估。
