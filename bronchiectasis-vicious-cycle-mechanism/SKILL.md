# 支气管扩张症恶性循环机制与干预策略

## 适用场景
当面对以下情况时应用本技能：
- 患者有慢性咳嗽、大量脓痰
- 反复发生呼吸道感染
- 影像学（如高分辨率CT）显示支气管异常扩张
- 已排除急性、自限性支气管炎及无结构性气道改变的咳嗽

## 核心执行流程
1. **识别初始诱因**：确认是否存在感染（细菌、病毒、真菌、分枝杆菌）、气道阻塞（异物、肿瘤）、免疫缺陷或遗传病（如纤毛不动综合征）。
2. **评估恶性循环环节**：
   - 感染是否引发持续气道炎症？
   - 黏液高分泌与清除障碍是否导致分泌物积聚？
   - 是否存在铜绿假单胞菌、流感嗜血杆菌等关键病原体定植？
   - 气道结构（软骨、肌肉、弹性组织）是否已被纤维组织替代？
3. **判断肺功能影响**：确定是阻塞性还是限制性通气功能障碍（约50%为阻塞性，限制性更常见）。
4. **制定多靶点干预策略**：
   - **促进引流**：实施体位引流、使用黏液溶解剂（如乙酰半胱氨酸）
   - **控制感染**：根据病原体选择长期口服或吸入抗生素（如吸入妥布霉素对抗铜绿假单胞菌）
   - **抑制炎症**：考虑吸入糖皮质激素（尤其合并哮喘或嗜酸粒细胞增高者）
   - **增强防御**：推荐流感疫苗、肺炎球菌疫苗，必要时评估免疫调节治疗
   - **预防急性加重**：教育患者识别早期症状，及时干预
5. **监测关键风险**：关注新生血管相关咯血风险，尤其当小动脉被侵蚀时。

## 注意事项
- 本机制不适用于无结构性气道病变的患者。
- 合并肺气肿或铜绿假单胞菌定植者预后更差，需强化管理。
- 治疗目标不仅是缓解症状，更在于阻断恶性循环以延缓肺功能恶化。
