# SOFA评分详细标准

## 1. 呼吸系统（基于PaO₂/FiO₂，单位：mmHg）
- 0分：≥400
- 1分：<400
- 2分：<300
- 3分：<200
- 4分：<100

## 2. 凝血系统（血小板计数，单位：×10⁹/L）
- 0分：≥150
- 1分：<150
- 2分：<100
- 3分：<50
- 4分：<20

## 3. 肝脏功能（总胆红素，单位：μmol/L）
- 0分：<20.5
- 1分：20.5–32.5
- 2分：34.2–100.9
- 3分：102.6–203.5
- 4分：>205.2

## 4. 循环系统（依据MAP及血管活性药物剂量，单位：μg/(kg·min)）
- 0分：MAP ≥70 mmHg，且未使用血管活性药
- 1分：MAP <70 mmHg
- 2分：多巴胺 ≤5 或 多巴酚丁胺（任意剂量）
- 3分：多巴胺 >5 或 肾上腺素 ≤0.1 或 去甲肾上腺素 ≤0.1
- 4分：多巴胺 >15 或 肾上腺素 >0.1 或 去甲肾上腺素 >0.1

> 注：血管活性药物必须持续输注≥1小时方可计入评分。

## 5. 神经系统（基于GCS评分）
- 0分：15
- 1分：13–14
- 2分：10–12
- 3分：6–9
- 4分：<6

## 6. 肾脏功能（满足任一即可，取高分）
### 肌酐（μmol/L）
- 0分：<106.1
- 1分：106.1–168
- 2分：176.8–300.6
- 3分：309.4–433.2
- 4分：>442.1

### 尿量（ml/24h）
- 0–2分：≥500
- 3分：<500
- 4分：<200

## 总分范围
- 最低：0分（无器官功能障碍）
- 最高：24分（所有系统严重衰竭）

## 临床意义
- 总分≥2提示存在器官功能障碍
- 入ICU时SOFA≥10与高死亡率显著相关
- 动态评分变化可用于评估治疗反应
