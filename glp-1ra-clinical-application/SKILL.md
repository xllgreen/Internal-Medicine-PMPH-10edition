# GLP-1受体激动剂临床应用规范

## 触发条件
当满足以下任一条件时启用本技能：
- 患者确诊为2型糖尿病（T2DM）；
- 同时存在肥胖、显著胰岛素抵抗、ASCVD或其高危因素；
- 或当前血糖控制不达标需强化治疗。

## 执行步骤

### 1. 确认适用性与禁忌证
- **确认前提条件**：
  - 已确诊T2DM；
  - eGFR符合所选制剂要求；
  - 无绝对禁忌证。
- **排除绝对禁忌**：
  - 1型糖尿病（T1DM）；
  - 糖尿病酮症酸中毒（DKA）；
  - 甲状腺髓样癌个人或家族史；
  - 多发性内分泌腺瘤病2型（MEN2）。
- **评估相对慎用情况**：
  - 胰腺炎或胆囊疾病史；
  - 胃轻瘫或既往胃部手术；
  - 增生型糖尿病视网膜病变（尤其使用司美格鲁肽时）；
  - eGFR低于制剂推荐阈值。

### 2. 选择合适制剂并制定给药方案
- **日制剂**：
  - 艾塞那肽：起始5μg 每日2次，可增至10μg 每日2次；
  - 利拉鲁肽：起始0.6mg 每日1次，逐步滴定至1.2–1.8mg；
  - 贝那鲁肽：起始0.1mg 每日3次。
- **周制剂**：
  - 司美格鲁肽：起始0.25mg 每周1次，最大剂量1mg/周；
  - 度拉糖肽：起始0.75mg 每周1次，最大1.5mg；
  - 洛塞那肽：起始0.1mg 每周1次，最大0.2mg；
  - 艾塞那肽缓释剂：每周1次。
- **新型制剂**：
  - 口服司美格鲁肽片剂；
  - 替尔泊肽（GLP-1/GIP双受体激动剂）可用于T2DM。

### 3. 监测疗效与安全性
- **预期疗效**：HbA1c降低1.0%–1.5%，显著减重，部分制剂具心血管保护作用。
- **常见不良反应管理**：
  - 胃肠道反应（恶心、呕吐、腹泻等）通常随用药时间减轻；
  - 建议缓慢滴定以提高耐受性。
- **罕见但重要风险监测**：
  - 胰腺炎症状（持续腹痛）；
  - 注射部位反应；
  - 心率增快；
  - 司美格鲁肽治疗早期可能加重糖尿病视网膜病变，需眼科随访。

### 4. 临床定位与联合用药
- 优先用于肥胖T2DM、需减重或合并ASCVD/高危患者；
- 可单用或与二甲双胍、SGLT2抑制剂等其他降糖药联用。

> 注意：具体剂量递增周期、eGFR阈值及药物相互作用详见参考文件。
