# 慢阻肺稳定期吸入糖皮质激素使用指征评估

## 触发条件
仅在同时满足以下条件时执行本技能：
- 患者处于慢阻肺稳定期；
- 已充分使用长效支气管扩张剂（如LABA/LAMA）进行维持治疗；
- 急性加重仍未得到有效控制。

## 执行步骤

1. **确认基础治疗状态**：核实患者是否已规律使用长效支气管扩张剂（单药或双联）且依从性良好。

2. **收集关键临床变量**：
   - 外周血嗜酸性粒细胞计数（cells/μl）；
   - 过去一年内中度或重度急性加重次数；
   - 是否有哮喘病史或哮喘特征；
   - 是否存在反复肺炎或分枝杆菌感染史。

3. **根据以下标准判断ICS使用推荐等级**：
   - **强烈推荐加用ICS**（满足任一）：
     - 有慢阻肺急性加重住院史；
     - 每年≥2次中度急性加重；
     - 嗜酸性粒细胞计数≥300个/μl；
     - 有哮喘病史或伴有哮喘特征。
   - **可考虑加用ICS**（需同时满足）：
     - 每年1次中度急性加重；
     - 嗜酸性粒细胞计数为100～300个/μl。
   - **不建议使用ICS**（满足任一）：
     - 嗜酸性粒细胞计数＜100个/μl；
     - 反复发生肺炎；
     - 合并分枝杆菌感染。

4. **制定治疗方案**：
   - 若推荐加用ICS，优先选择含ICS的三联吸入制剂（如LABA/LAMA/ICS）；
   - 常用ICS成分包括布地奈德、氟替卡松、倍氯米松；
   - 明确随访计划以监测疗效与不良反应。

> 注意：无哮喘病史且无哮喘特征者需谨慎评估，避免不必要的ICS暴露。
