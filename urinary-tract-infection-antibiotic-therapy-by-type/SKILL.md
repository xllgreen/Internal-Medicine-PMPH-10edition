# 不同类型尿路感染的抗生素治疗方案

## 触发条件
- 已确诊尿路感染类型（急性膀胱炎、肾盂肾炎等）
- 已评估患者类别（如孕妇、儿童、糖尿病、留置导尿）
- 已留取尿培养标本（经验性用药前）
- 已评估肝肾功能

## 执行步骤

### 1. 急性膀胱炎
- **非复杂性女性**（推荐短程疗法）：
  - 呋喃妥因 50–100mg tid ×5天
  - TMP-SMZ 160mg/800mg bid ×3天
  - 匹美西林 0.4g bid ×3–5天
  - 备选：左氧氟沙星 0.5g qd ×3天 或 二代头孢
- **特殊人群**（妊娠、老年、糖尿病、男性）：7天疗法
- **疗效评估**：停药7天后复查尿培养；若阳性，延长至2周

### 2. 肾盂肾炎
- **轻症**（口服）：
  - 喹诺酮类（环丙沙星 0.5g bid 或 左氧氟沙星 0.5g qd）
  - 阿莫西林-克拉维酸钾 0.457g bid
  - 头孢呋辛酯 0.25g bid
  - TMP-SMZ bid
  - 疗程：7–14天
- **重症**（静脉）：
  - 环丙沙星 0.4g bid、左氧氟沙星 0.5g qd
  - 头孢噻肟钠 2g tid、头孢曲松钠 1–2g qd、头孢吡肟 2g bid
  - 哌拉西林/他唑巴坦 2.25–4.5g tid
  - 革兰阳性球菌：考虑万古霉素
  - **避免氨基糖苷类**（肾毒性）
  - 疗程：体温正常后继续静脉3天，再转口服至总疗程2周
- **72小时无效**：按药敏结果换药

### 3. 特殊人群处理
- **妊娠期**：
  - 膀胱炎：呋喃妥因或头孢，3–7天
  - 肾盂肾炎：静脉三代头孢，疗程2周
- **儿童**：禁用喹诺酮类；按体重调整剂量；排查泌尿畸形
- **导管相关UTI**：无症状者不治疗；有全身症状按复杂性UTI处理
- **糖尿病**：严格控糖；严重者联合静脉抗生素；警惕肾乳头坏死
- **反复发作**：急性期治疗同首次；后续可长程低剂量抑菌（如呋喃妥因）或性生活后单次预防

### 4. 用药原则
- 初选用对革兰阴性杆菌有效的药物
- 优先选择尿液和肾脏浓度高、肾毒性小的药物
- 根据肝肾功能调整剂量
- 单药失败、严重感染、混合感染或耐药时联合用药

## 关键变量
- `infection_type`：急性膀胱炎 / 急性肾盂肾炎 / 复杂性UTI
- `patient_category`：非复杂性女性 / 孕妇 / 儿童 / 糖尿病 / 留置导尿等
- `severity`：轻症（口服） / 重症（静脉）
