# 医疗诊疗技术术语标准化映射

## 何时使用
- 需将中文影像学检查、内镜技术、实验室检测或介入治疗名称转换为标准英文术语
- 编写英文病历、科研论文或国际交流材料时需确保术语一致性
- 验证某项技术的官方缩写是否符合国际惯例

## 执行步骤
1. **识别技术类型**：判断输入属于以下哪一类：
   - 影像学检查（如磁共振胆胰管成像）
   - 内镜技术（如经内镜逆行胆胰管造影术）
   - 实验室检测（如丙氨酸转氨酶）
   - 评分系统或介入治疗（如APACHE Ⅱ评分、经皮冠状动脉介入术）

2. **提取标准英文全称与缩写**：
   - 使用权威来源提供的英文全称
   - 确认缩写遵循首字母缩略规则（如ERCP）或保留关键识别词（如HFNC）
   - 缩写统一使用大写字母（如PCI、TAVR）

3. **验证术语唯一性**：
   - 注意区分相关但不同的技术（如PCI 与 CAG）
   - 避免混淆具有相似中文描述的不同英文术语

4. **输出格式**：
   - 返回“中文名称 + 英文全称, 标准缩写”格式
   - 示例：`经内镜逆行胆胰管造影术 endoscopic retrograde cholangiopancreatography, ERCP`

> ⚠️ 此技能仅提供术语对照，不包含操作步骤、适应症或技术原理。
