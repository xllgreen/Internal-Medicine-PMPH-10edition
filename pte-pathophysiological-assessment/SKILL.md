# PTE病理生理评估

## 适用场景
- 已确诊或高度怀疑急性PTE的患者
- 出现以下任一表现：低血压（收缩压<90 mmHg）、右心室功能障碍（超声或CT证实）、严重低氧血症（PaO₂<60 mmHg）
- 肺动脉栓塞负荷估计≥30%

> **注意**：本技能不适用于慢性血栓栓塞性肺动脉高压（CTEPH）的急性期管理；小面积栓塞（<30%）通常无需启动此评估。

## 执行步骤

1. **评估机械阻塞程度**
   - 确认肺血管床阻塞比例是否≥30%
   - 判断肺血管阻力（PVR）是否显著升高
   - 观察肺动脉压力是否上升，提示右心室后负荷增加

2. **识别右心功能障碍**
   - 检查影像学（超声心动图或CT）是否显示右心室扩大
   - 评估是否存在室间隔左移、左心室充盈受限
   - 监测心排血量下降及体循环低灌注表现（如休克、晕厥）

3. **判断冠脉灌注受损风险**
   - 结合右心室内压升高与主动脉压下降，评估冠状动脉灌注压是否降低
   - 警惕右心室内膜下心肌缺血可能

4. **分析呼吸功能障碍机制**
   - 评估肺泡无效腔是否增大
   - 判断通气血流比例失调程度
   - 若存在顽固性低氧血症，排查卵圆孔再开放导致的右向左分流

5. **识别肺梗死相关表现（如适用）**
   - 观察是否在发病24小时内出现咯血
   - 检查有无胸膜炎性胸痛或胸腔积液

6. **应用关键阈值指导决策**
   - 阻塞<30%：通常无需强化干预
   - 阻塞≥50%：高度警惕猝死风险，尤其合并卵圆孔未闭者，应考虑紧急溶栓或介入治疗

## 输出目标
基于上述评估，明确患者是否属于高危PTE，并据此决定是否启动溶栓治疗、强化抗凝方案或提供血流动力学支持（如血管活性药物、机械通气等）。
