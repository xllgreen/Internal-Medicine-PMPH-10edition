# 肺曲霉病的临床分型与诊断

## 何时使用
当满足以下任一条件时启动本技能：
- 患者为免疫抑制宿主且出现呼吸道症状（如干咳、胸痛、咯血、呼吸困难）；
- 影像学发现特征性征象（如晕轮征、新月征、Monad征、中央性支气管扩张、空洞内球形结构）；
- 疑似呼吸道真菌感染需排除或确认肺曲霉病。

## 执行步骤

### 第一步：确定临床分型
根据免疫状态和临床特征将肺曲霉病分为三类：

1. **侵袭性肺曲霉病（IPA）**
   - 见于免疫力极度低下者；
   - 表现为干咳、胸痛、咯血、快速进展的呼吸衰竭；
   - 影像学：早期“晕轮征”，后期“新月征”；
   - 病理：2～4μm锐角分支、分隔、无色素菌丝。

2. **慢性肺曲霉病（CPA）**
   包含五种亚型：
   - **简单型肺曲霉球**：空洞内可移动球形结构（Monad征），增强扫描无强化，主要表现为反复咯血；
   - **慢性空洞性肺曲霉病（CCPA）**：多发空洞伴胸膜增厚，症状类似肺结核；
   - **慢性纤维化性肺曲霉病（CFPA）**：累及≥2肺叶的广泛纤维化，伴进行性呼吸困难；
   - **曲霉结节**：通常无症状，影像无特异性，需病理确诊；
   - **亚急性侵袭性肺曲霉病（SAIA）**：慢性咳嗽、发热，影像表现为空洞+实变+曲霉球。

3. **变应性支气管肺曲霉病（ABPA）**
   - 见于免疫功能正常者，常合并哮喘；
   - 表现为棕黄色脓痰、外周血嗜酸粒细胞增多；
   - 血清总IgE >1000 IU/ml；
   - 曲霉特异性IgE/IgG阳性，速发型皮试阳性；
   - 影像：中央性支气管扩张、一过性肺浸润、黏液嵌塞。

### 第二步：评估诊断证据

- **确诊标准**：组织活检培养阳性 + 病理见典型菌丝。
- **高度怀疑标准**：免疫抑制宿主 + BALF或痰液镜检/培养阳性 + 特征性CT表现。
- **辅助检查**：
  - GM试验（半乳甘露聚糖检测）；
  - PCR检测曲霉DNA；
  - 血清曲霉特异性IgE/IgG抗体；
  - 皮肤点刺试验。

### 第三步：排除干扰因素

- 呼吸道标本（痰/BALF）阳性仅提示定植或感染可能，**不能单独作为诊断依据**；
- 本流程**不适用于非曲霉属真菌感染**；
- 单纯定植（无症状、无影像异常）不构成肺曲霉病诊断。

## 输出要求
输出具体分型（IPA / CPA及其亚型 / ABPA）及支持该诊断的关键证据链（包括免疫状态、影像征象、实验室结果）。
