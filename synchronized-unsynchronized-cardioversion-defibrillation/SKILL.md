# 同步与非同步电复律除颤决策

## 触发条件
当患者出现以下情况时启动本技能：
- 快速型心律失常（如房颤、房扑、室速、室颤等）
- 伴有血流动力学不稳定（如低血压、意识障碍、肺水肿）或抗心律失常药物无效

## 决策流程
1. **评估心电图是否存在可识别的R波（QRS波清晰可见）**：
   - 若**存在R波** → 执行**同步电复律**
   - 若**无R波（如室颤、无脉室速）** → 执行**非同步电除颤**

2. **执行同步电复律（Cardioversion）**：
   - 确认除颤器“同步”模式已开启
   - 电流将在R波降支或R波起始后约30ms放电（心室绝对不应期内）
   - 适用于：房颤（尤其血流动力学不稳定或预激旁路前传）、房扑（药物难控或近1:1传导）、室上速（药物无效且恶化）、室速（伴意识障碍、低血压、肺水肿）

3. **执行非同步电除颤（Defibrillation）**：
   - 立即放电，无需等待R波
   - 适用于：心室颤动（VF）、心室扑动（无脉室速）
   - 黄金时间窗：室颤发生后1～3分钟内，越早成功率越高

4. **禁忌情况（禁止紧急电复律）**：
   - 慢-快综合征（房颤前心室率显著缓慢）
   - 洋地黄中毒引起的房颤
   - 未抗凝且心房内有血栓或近期动脉栓塞史

> **注意**：同步模式禁用于室颤/室扑；非同步模式禁用于存在R波的心律失常（如房颤、室速），以免诱发室颤。
