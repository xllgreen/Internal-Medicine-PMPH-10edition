# GERD诊断与分型标准流程

## 触发条件
当患者主诉烧心或反流症状时，启动本诊断流程。

## 执行步骤

### 1. 初步拟诊
- 对存在典型烧心和/或反流症状的患者，可初步拟诊为GERD。
- 给予质子泵抑制剂（PPI）试验性治疗（通常4–8周）。
- 若症状明显缓解，则支持GERD诊断。

### 2. 分型诊断
根据内镜结果和客观反流证据，将GERD分为两类：

#### 反流性食管炎（RE）
满足以下两项即可诊断：
- 存在烧心和/或反流症状；
- 内镜下发现食管黏膜破损（依据洛杉矶分级LA-A至D级）。

#### 非糜烂性反流病（NERD）
需同时满足以下四项：
- 存在烧心和/或反流症状；
- 内镜检查阴性（无黏膜破损）；
- 食管反流监测（pH或阻抗-pH）提供客观反流证据；
- PPI治疗有效。

### 3. 辅助检查选择
- **内镜检查**：诊断RE的金标准，用于评估黏膜损伤程度及并发症（如Barrett食管、狭窄）。
- **食管反流监测**：诊断GERD的“金标准”，尤其适用于NERD患者。
- **食管测压**：主要用于抗反流手术前的功能评估。
- **钡剂造影**：不作为常规检查，但可用于排除食管癌等结构性病变。

### 4. 鉴别诊断
必须与以下疾病相鉴别：
- 感染性食管炎
- 嗜酸细胞性食管炎
- 贲门失弛缓症
- 食管癌
- 功能性烧心
- 心源性胸痛

## 输出结果
根据分型结果制定治疗策略：
- RE患者需足疗程抑酸治疗以促进黏膜愈合；
- NERD患者可考虑按需治疗或维持治疗。
