# 溃疡性结肠炎严重度分型评估

## 适用条件
- 患者处于活动期（有症状）
- 已具备完整的临床症状及实验室检查数据
- 已排除克罗恩病、感染性肠炎等其他结肠炎

## 评估步骤
1. **确认活动期状态**：患者存在腹泻、黏液脓血便、腹痛等症状。
2. **收集以下6项指标数据**：
   - 每日排便次数（次/日）
   - 便血程度（少/无 vs 明显血便）
   - 腋下体温（℃）
   - 静息脉搏（次/分）
   - 血红蛋白浓度（g/L）
   - 红细胞沉降率（血沉，mm/h）
3. **对照标准判断严重度**：
   - **轻度**：排便 ≤3次/日，便血少或无，体温 <37.8℃，脉搏 <90次/分，血红蛋白 >105 g/L，血沉 <20 mm/h
   - **重度**：排便 ≥6次/日，明显血便，体温 >37.8℃，脉搏 >90次/分，血红蛋白 <105 g/L，血沉 >30 mm/h
   - **中度**：不符合轻度或重度全部标准的其余情况
4. **整合完整诊断**：结合病变范围（直肠炎、左半结肠炎、广泛结肠炎）和临床类型（初发型、慢性复发型）形成最终分型。

## 输出结果
输出为“轻度”、“中度”或“重度”，用于指导后续治疗方案选择（如5-ASA、糖皮质激素或生物制剂）。

> 注意：本技能不适用于缓解期患者或其他类型的结肠炎。
