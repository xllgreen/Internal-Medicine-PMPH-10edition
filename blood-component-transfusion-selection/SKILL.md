# 成分输血选择与应用

## 何时使用
- 患者被诊断为以下情况之一：
  - 贫血（需红细胞）
  - 血小板减少（需血小板）
  - 凝血因子缺乏（需冷沉淀、因子Ⅷ、凝血酶原复合物等）
  - 低蛋白血症（需白蛋白）
  - 免疫缺陷（需免疫球蛋白）
- 已完成交叉配血且符合输血适应证
- 不属于急性大失血需快速补充全血容量的情形（除非联合使用多种成分）

## 如何执行
1. **确认目标血液成分**：根据实验室检查和临床表现，明确患者缺乏的具体成分（如红细胞、血小板、因子Ⅷ等）。
2. **选择对应成分制品**：
   - 红细胞类：浓缩红细胞、洗涤红细胞、冰冻红细胞或红细胞悬液
   - 血小板：单采或浓缩血小板
   - 粒细胞：仅在严重感染且粒细胞极度低下时考虑（因保存期极短）
   - 血浆及衍生物：新鲜冰冻血浆、冷沉淀、白蛋白、免疫球蛋白、纤维蛋白原、凝血酶原复合物等
3. **评估库存时间与紧迫性**：
   - 注意全血或成分血的保存时间对凝血因子活性和细胞功能的影响
   - 若临床需求紧急且成分制备无法及时完成，需重新评估是否联合使用或改用其他策略
4. **实施输注**：遵循“缺什么补什么”原则，进行针对性成分输注，实现高效、节约血源的精准治疗。
5. **避免全血输注**：除特殊情形（如同时需红细胞和大量血浆且无成分组合条件），优先使用成分血而非全血。

> 提示：成分输血是现代输血医学的核心方向，强调治疗针对性、资源节约和疗效最大化。
