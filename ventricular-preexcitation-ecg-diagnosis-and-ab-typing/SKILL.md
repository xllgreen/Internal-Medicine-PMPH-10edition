# 心室预激心电图诊断与A/B型分型

## 何时使用
- 心电图显示PR间期 < 0.12秒且QRS起始粗钝（δ波）
- 患者处于窦性心律，且有完整12导联心电图
- 需要区分A型与B型预激以辅助旁道定位

> **注意**：不适用于房颤伴快速心室率掩盖δ波的情况，也不适用于束支传导阻滞干扰QRS起始判断的情形。

## 执行步骤

1. **确认三项基本心电图特征**：
   - PR间期 < 0.12秒
   - QRS波群起始部分粗钝（称为δ波）
   - ST-T段呈继发性改变，方向与QRS主波相反

2. **根据胸导联QRS主波方向进行分型**：
   - **A型预激**：V1～V6导联QRS主波全部向上
   - **B型预激**：V1、V2导联主波向下，V4～V6导联主波向上

3. **输出结果**：
   - 明确是否符合预激综合征诊断
   - 标明为A型或B型

4. **临床提示**：
   - δ波是旁道前传的直接证据
   - A型多提示左侧旁道，B型多提示右侧旁道
   - 若怀疑房颤合并预激但δ波被掩盖，需结合宽QRS心动过速及极快心室率（如 >200次/分）综合判断
