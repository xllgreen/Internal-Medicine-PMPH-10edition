# 铁调素介导的贫血机制分析

## 何时使用
- 患者存在慢性炎症、感染、肿瘤或肾病等慢性疾病背景；
- 实验室检查显示：血清铁↓、转铁蛋白饱和度↓、铁蛋白正常或升高；
- 贫血类型疑似“慢性病贫血”（anemia of chronic disease）；
- 单纯口服或静脉补铁效果不佳。

## 执行步骤
1. **确认炎症状态**：评估是否存在促炎细胞因子升高（如IL-6↑、CRP↑）。
2. **判断铁调素水平影响**：
   - 若存在炎症，则铁调素分泌增加；
   - 铁调素抑制肠上皮细胞和巨噬细胞上的ferroportin，阻断铁释放入血。
3. **识别功能性铁缺乏特征**：
   - 血浆游离铁显著降低；
   - 铁被“锁”在储存库（肝、脾、巨噬细胞）中无法用于红细胞生成。
4. **解释临床表现**：
   - 铁蛋白可能因作为急性期反应物而假性升高；
   - 此为铁利用障碍，非绝对铁缺乏。
5. **提供治疗建议**：
   - 强调控制原发炎症或慢性病是关键；
   - 单纯补铁通常无效；
   - 可提及铁调素通路靶向疗法（如拮抗剂）处于研究阶段。

## 关键变量
- `inflammatory_status`：是否存在炎症（布尔值）
- `serum_iron`：血清铁浓度（μmol/L）
- `ferritin_inflammation_adjusted`：铁蛋白是否因炎症假性升高
- `hepcidin_level`：血清铁调素水平（若可测）

> **输出模板**：提示为功能性铁缺乏，解释为何补铁无效，并强调需治疗基础炎症或慢性病。
