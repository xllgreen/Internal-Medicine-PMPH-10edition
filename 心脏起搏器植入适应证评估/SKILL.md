# 心脏起搏器植入适应证评估

## 何时使用
当临床医生面对以下情况时启用本技能：
- 患者报告晕厥、近晕厥、明显乏力或运动耐量显著下降；
- 心电图或动态心电图提示缓慢型心律失常；
- 需判断是否达到永久性起搏器植入的I类适应证标准。

## 执行步骤
1. **确认症状与心律失常相关**：判断患者是否存在明确由缓慢型心律失常引起的症状（如晕厥、黑矇、严重乏力等）。
2. **识别心律失常类型**：检查心电监测是否显示以下任一情况：
   - 窦房结功能障碍（如严重窦性停搏、窦房传导阻滞、慢-快综合征）；
   - 症状性慢性双分支/三分支阻滞，或二度Ⅱ型、高度、三度房室传导阻滞；
   - 持续性房颤合并症状性心动过缓（平均心室率<40次/分或RR间歇>5秒）；
   - 因必须使用β受体阻滞剂等药物导致的症状性传导障碍，且无替代方案；
   - 心脏抑制性颈动脉窦综合征（颈动脉窦按摩诱发>3秒停搏伴晕厥）；
   - 直立倾斜试验中出现>3秒心脏停搏并导致反复晕厥。
3. **排除可逆性病因**：确认症状非由急性心肌梗死（48小时内）、药物中毒、电解质紊乱、睡眠呼吸暂停等可逆因素引起。
4. **做出判定**：若满足上述任一心律失常条件，且症状与心律失常有明确因果关系，则符合I类适应证，建议进行永久性起搏器植入评估，并根据心房功能和传导障碍类型选择合适起搏模式（如AAI、VVI、DDD等）。

> 注意：无症状的窦房结功能障碍或传导阻滞不符合植入指征。
