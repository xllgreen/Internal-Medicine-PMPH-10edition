# 使用Michet标准诊断复发性多软骨炎（RP）

## 适用前提
- 已观察到耳、鼻或呼吸道软骨的炎症表现。
- 需排除外伤、感染、梅毒、肉芽肿性疾病等类似表现。

## 执行步骤
1. **确认主要标准**（需为“发作性软骨炎”）：
   - 明确的发作性耳软骨炎（耳廓红肿疼痛，不累及耳垂，可致“菜花耳”）
   - 明确的发作性鼻软骨炎（可致“鞍鼻”畸形）
   - 明确的发作性喉、气管软骨炎（可致声嘶、喘鸣、窒息）

2. **确认次要标准**：
   - 眼炎（如巩膜炎、葡萄膜炎）
   - 听力下降（传导性或感音神经性）
   - 前庭功能障碍（如眩晕、平衡障碍）
   - 血清阴性多关节炎（非对称性、非侵蚀性）

3. **应用诊断规则**：
   - 若满足任意2项主要标准，即可确诊复发性多软骨炎；
   - 或满足1项主要标准 + 任意2项次要标准，也可确诊。

4. **辅助确认与鉴别**：
   - 通过影像学（如肺CT、纤维支气管镜）或活检确认气道受累；
   - 与VEXAS综合征、肉芽肿性多血管炎等疾病进行鉴别。

> 注意：“明确的发作性”指症状反复出现且与软骨结构直接相关。
