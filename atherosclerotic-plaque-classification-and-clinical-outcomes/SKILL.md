# 动脉粥样硬化斑块分类与临床转归评估

## 触发条件
当影像学检查（如IVUS、OCT）确认存在动脉粥样硬化斑块（至少为Ⅲ型病变）时，执行本技能以评估斑块稳定性及潜在临床风险。

## 执行步骤

1. **按稳定性对斑块进行分类**：
   - **稳定型斑块**：纤维帽较厚、脂质池较小、钙化明显，通常导致慢性管腔狭窄，破裂风险低。
   - **不稳定型（易损）斑块**：纤维帽薄、脂质池大、富含炎症细胞（如巨噬细胞、T细胞）、基质金属蛋白酶（MMPs）活性高，机械强度弱，易破裂。

2. **识别导致斑块不稳定的因素**：
   - 炎症反应增强（如C反应蛋白CRP、白介素-6 IL-6升高）
   - 血流动力学剪切力异常变化
   - 应激状态（儿茶酚胺水平升高）
   - 内皮功能障碍

3. **评估斑块破裂的病理后果**：
   - 斑块破裂释放组织因子和血小板活化因子 → 血小板聚集形成白色血栓
   - 炎症因子促进促凝物质表达并上调PAI-1 → 抑制纤溶系统 → 血栓演变为红色血栓
   - 最终可导致血管急性闭塞，引发心肌梗死、脑梗死等急性缺血事件

4. **参考美国心脏病学会（AHA）病理分期**：
   - Ⅰ–Ⅱ型：脂质点/条纹（可逆，不适用本技能）
   - Ⅲ–Ⅳ型：脂核形成、脂质池出现
   - Ⅴ型：纤维粥样斑块（典型狭窄性病变）
   - Ⅵ型：复合病变（含出血、溃疡、钙化或血栓）

5. **输出结果**：
   给出“斑块稳定性评分”（稳定 / 不稳定），用于指导临床决策，包括是否需强化药物治疗（如高强度他汀、双联抗血小板）或考虑血运重建（如支架置入）。

## 注意事项
- 本技能**不适用于**早期脂质条纹（Ⅰ–Ⅱ型）。
- 使用前需排除非粥样硬化性狭窄（如大动脉炎、纤维肌性发育不良）。
- 必须基于可靠的腔内影像学证据（IVUS/OCT）进行评估。
