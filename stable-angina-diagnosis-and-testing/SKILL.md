# 稳定型心绞痛诊断与辅助检查流程

## 触发条件
仅当满足以下全部条件时启动本流程：
- 胸痛由劳力、情绪、饱食或寒冷诱发；
- 疼痛持续3～5分钟，休息或含服硝酸甘油后数分钟内缓解；
- 患者年龄≥40岁，或为绝经后女性，且存在冠心病危险因素；
- **排除**：胸痛持续>30分钟、静息发作、硝酸甘油无效——此类情况应优先排查急性冠脉综合征（ACS）。

## 执行步骤

### 1. 初步临床评估
- 确认典型症状：胸骨后压迫感或闷胀感，可放射至左肩、左臂内侧或小指；
- 核实诱因明确、缓解方式典型、病程稳定（数周至数月无恶化）。

### 2. 心电图（ECG）检查
- **静息ECG**：记录基线，注意陈旧性心梗或非特异性ST-T改变；
- **发作时ECG**：观察是否出现一过性ST段压低≥0.1 mV（J点后60～80 ms）或T波倒置；若平时T波倒置，发作时“假性正常化”亦支持诊断；
- **运动负荷试验**（平板/踏车）：
  - 目标心率达预计最大心率的85%～90%（亚极量）或极量；
  - 阳性标准：运动中出现典型心绞痛 + ST段水平或下斜形压低≥0.1 mV持续≥2分钟；
  - **禁忌**：急性心梗、不稳定型心绞痛、心衰、严重心律失常；
- **Holter监测**：连续记录≥24小时，结合症状时间窗识别有症状或无症状心肌缺血。

### 3. 影像学检查
- **冠脉CTA**：
  - 评估管腔狭窄、钙化及斑块性质；
  - 若无显著狭窄，阴性预测价值高，可避免有创检查；
  - 结合CT-FFR：<0.80提示功能性心肌缺血，需进一步评估；
- **超声心动图**：
  - 静息状态下通常正常；
  - 负荷超声可检出缺血区室壁运动异常，并排除肥厚型心肌病、主动脉瓣狭窄等；
- **核素心肌灌注显像**（²⁰¹Tl 或 ⁹⁹ᵐTc-MIBI）：
  - 负荷状态下出现灌注缺损，静息时再分布充填，为典型心肌缺血表现；
- **冠脉造影**（金标准）：
  - 管腔直径减少≥70%～75%视为血流动力学显著狭窄；
  - 可联合IVUS、OCT、FFR或QFR等腔内技术综合评估斑块形态与功能意义。

### 4. 鉴别诊断
重点排除以下疾病：
- **急性冠脉综合征（ACS）**：静息胸痛、硝酸甘油无效、心肌标志物升高；
- **X综合征**：多见于女性，负荷试验阳性但冠脉造影无狭窄，预后良好；
- **心脏神经症**：刺痛/隐痛、部位不固定、伴叹息样呼吸、硝酸甘油无效；
- **非心源性胸痛**：如肋间神经痛（沿神经走行压痛）、胃食管反流（与进食相关）。

### 5. 严重度分级（CCS分级）
- **I级**：仅剧烈活动诱发；
- **II级**：快走或登楼>1层受限；
- **III级**：平地步行<200米或登1层即诱发；
- **IV级**：静息或轻微活动即发作。

## 输出结果
整合所有检查结果，完成以下任务：
1. 确认或排除稳定型心绞痛诊断；
2. 评估冠脉狭窄的解剖与功能严重程度；
3. 排除其他类似疾病；
4. 确定CCS分级，为后续药物或血运重建治疗提供依据。
