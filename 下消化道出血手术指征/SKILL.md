# 下消化道出血手术指征

## 触发条件
患者接受药物、内镜及介入治疗后仍持续出血且危及生命。

## 执行步骤
1. **确认治疗失败**：
   - 药物治疗、内镜治疗、血管介入治疗均已尝试
   - 出血持续存在（血红蛋白持续下降、需反复输血、血流动力学不稳定）

2. **评估生命危险**：
   - 收缩压<90 mmHg、心率>120次/分、意识障碍、尿量<0.5 mL/(kg·h)
   - 需大量输血（>2 U/h 或 >4 U/24h）

3. **关键原则**：
   - 无论出血病变是否确诊，只要满足上述条件即视为具备急诊手术指征

4. **紧急处理**：
   - 立即组织多学科会诊
   - 准备急诊剖腹探查或针对性手术止血
