# 阿片类中毒纳洛酮应用

## 触发条件
患者出现阿片类中毒典型症状（瞳孔缩小、呼吸抑制、昏迷）。

## 执行步骤
1. **给药途径**：
   - 静脉、肌内、皮下注射或气管内给药

2. **初始剂量**：
   - 阿片中毒者静脉注射2mg

3. **重复给药策略**：
   - 阿片依赖中毒者：间隔3～10分钟重复应用
   - 非依赖性中毒者：间隔2～3分钟重复应用

4. **总剂量上限**：
   - 若总剂量达15～20mg仍无效，应考虑合并非阿片类毒品中毒或其他病因

5. **特殊情况处理**：
   - 长半衰期阿片类（美沙酮）或强效阿片类（芬太尼）中毒：采用静脉输注维持拮抗效果

6. **注意事项**：
   - 对哌替啶中毒所致癫痫发作无效
   - 对海洛因/美沙酮所致非心源性肺水肿无效
