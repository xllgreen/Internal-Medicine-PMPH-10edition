# ECMO启动指征与初始参数设置

## 何时使用
当以下条件同时满足时，使用本技能：
- 患者为危重型新冠肺炎、ARDS或难治性低氧血症；
- 已实施标准肺保护性通气（FiO₂≥0.8、潮气量6 ml/kg理想体重、PEEP≥5 cmH₂O）；
- 俯卧位通气≥12小时/天且无效；
- 无ECMO禁忌证（如不可逆脑损伤、终末期疾病）；
- 出现以下任一血气恶化标准：
  - PaO₂/FiO₂ < 50 mmHg 持续 >3 小时；
  - PaO₂/FiO₂ < 80 mmHg 持续 >6 小时；
  - pH < 7.25 且 PaCO₂ > 60 mmHg 超过6小时，伴呼吸频率 >35次/分；
  - 呼吸频率 >35次/分、pH < 7.2 且平台压 >30 cmH₂O。

## 如何执行
1. **确认前提条件**：核实患者已接受最优机械通气和俯卧位治疗，且无ECMO禁忌证。
2. **评估ECMO指征**：若满足上述任一血气标准，立即触发多学科会诊，准备ECMO。
3. **选择ECMO模式**：
   - 仅呼吸衰竭 → VV-ECMO；
   - 合并循环衰竭 → VA-ECMO；
   - VA-ECMO出现头臂缺氧 → VAV-ECMO。
4. **设置初始呼吸机参数**（ECMO启动后同步实施）：
   - 潮气量：4～6 ml/kg（理想体重）；
   - 平台压 ≤ 25 cmH₂O；
   - 驱动压 ≤ 15 cmH₂O；
   - PEEP：5～15 cmH₂O；
   - 呼吸频率：4～10 次/分；
   - FiO₂ < 0.5。
5. **联合俯卧位通气**：若存在强吸气努力、双肺重力依赖区实变或需气道引流，积极联合俯卧位。
6. **强调尽早启动**：避免因延误导致预后恶化。

> **输出结果**：触发“ECMO多学科会诊，准备启动体外生命支持”。
