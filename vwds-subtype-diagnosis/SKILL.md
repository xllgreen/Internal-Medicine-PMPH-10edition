# 血管性血友病（vWD）分型诊断

## 何时使用
- 患者vWF:Ag <30% 和/或 vWF:RCo <30%
- 已完成基础实验室检查（vWF:Ag、vWF:RCo、FVIII:C）
- 已排除血友病A/B及其他原发性血小板功能障碍
- 不适用于获得性vWD的初始诊断

## 执行步骤

### 1. 初步分类
根据vWF缺陷性质分为：
- **量的缺陷**：1型（部分缺乏）、3型（完全缺失）
- **质的缺陷**：2型（功能异常）

### 2. 2型vWD亚型鉴别
依据以下实验结果进一步区分：
- **2A型**：高-中分子量vWF多聚体缺乏，血小板依赖性功能减弱
- **2B型**：vWF对血小板GPIb亲和力增强，导致高分子量多聚体被清除
- **2M型**：vWF介导的血小板黏附能力下降，但多聚体分布正常
- **2N型**：vWF对因子VIII亲和力显著降低，FVIII:C常<30%

### 3. 必做分型实验
执行以下关键检测以支持分型：
- 血浆vWF多聚体分析
- 瑞斯托霉素诱导的血小板聚集（RIPA）
- vWF胶原结合试验（vWF:CB）
- vWF因子VIII结合活性（vWF:FVIIIB）

### 4. 鉴别诊断要点
必须排除以下疾病：
- 血友病A或B
- 获得性vWD
- 血小板型vWD（假性vWD）

> **注意**：最终诊断需结合家族史、临床出血表现与实验室结果综合判断。
