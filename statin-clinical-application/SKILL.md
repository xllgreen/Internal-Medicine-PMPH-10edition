# 他汀类药物临床应用

## 触发条件
患者诊断为高胆固醇血症、混合性高脂血症或ASCVD，且无禁忌证。

## 执行步骤
1. **理解药理机制**：
   - 竞争性抑制HMG-CoA还原酶，减少胆固醇合成
   - 加速LDL分解代谢，抑制VLDL合成

2. **选择合适药物与剂量**：
   - 洛伐他汀：10～80mg
   - 辛伐他汀：5～40mg
   - 普伐他汀：10～40mg
   - 氟伐他汀：10～40mg
   - 阿托伐他汀：10～80mg
   - 瑞舒伐他汀：10～20mg

3. **监测不良反应**：
   - 转氨酶升高、肌痛、肌炎、CK升高
   - 极少数发生横纹肌溶解

4. **应对策略**：
   - 出现不良反应时可更换他汀种类、减少剂量、改为隔日服用，或换用非他汀类药物

5. **特别警示**：
   - 长期使用可能增加新发糖尿病风险，需定期监测血糖
   - 严禁与环孢素、雷公藤、环磷酰胺、大环内酯类抗生素、吡咯类抗真菌药联用
