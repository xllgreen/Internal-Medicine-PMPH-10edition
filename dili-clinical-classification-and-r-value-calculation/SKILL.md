# 药物性肝损伤（DILI）临床分型与R值计算

## 何时使用
- 患者有明确用药史
- 已完成ALT、ALP等肝功能检测且结果异常
- 需排除病毒性肝炎、酒精性肝病等其他肝损伤原因

## 执行步骤
1. **确认前提条件**：确保已知用药史，并已获得ALT和ALP的实测值及其对应的正常上限（ULN）。
2. **计算R值**：使用公式 R = (ALT实测值 / ALT_ULN) ÷ (ALP实测值 / ALP_ULN)。
3. **根据R值和酶学指标进行分型**：
   - **肝细胞损伤型**：ALT ≥ 3×ULN 且 R ≥ 5
   - **胆汁淤积型**：ALP ≥ 2×ULN 且 R ≤ 2
   - **混合型**：ALT ≥ 3×ULN 且 ALP ≥ 2×ULN，且 2 < R < 5
   - **肝血管损伤型**：临床表现符合肝窦阻塞综合征、布-加综合征等，不依赖R值
4. **判断急慢性**：病程≤6个月为急性DILI；>6个月肝酶仍异常或有慢性肝损伤证据者为慢性DILI。
5. **考虑辅助诊断**：若可行，肝活检发现嗜酸性粒细胞浸润、小泡型脂滴或重金属沉着可支持DILI诊断。

> 注意：胆汁淤积型DILI更易进展为慢性，需密切随访。
