# R值计算公式与DILI分型详细说明

## R值计算

R = (ALT_measured / ULN_ALT) ÷ (ALP_measured / ULN_ALP)

其中：
- ALT_measured：血清丙氨酸氨基转移酶实测值（U/L）
- ALP_measured：血清碱性磷酸酶实测值（U/L）
- ULN_ALT：实验室提供的ALT正常上限值（通常为男性40–50 U/L，女性35–45 U/L，依实验室而异）
- ULN_ALP：实验室提供的ALP正常上限值（成人通常为100–130 U/L，但受年龄、性别影响较大）

## 分型标准详解

### 肝细胞损伤型
- **生化标准**：ALT ≥ 3×ULN 且 R ≥ 5
- **组织学特征**：肝细胞坏死，汇管区可见嗜酸性粒细胞和淋巴细胞浸润
- **临床表现**：类似急性病毒性肝炎，可有乏力、恶心、黄疸

### 胆汁淤积型
- **生化标准**：ALP ≥ 2×ULN 且 R ≤ 2
- **组织学特征**：毛细胆管型胆汁淤积，胆栓形成
- **临床表现**：显著黄疸、皮肤瘙痒，ALT升高相对较轻

### 混合型
- **生化标准**：ALT ≥ 3×ULN 且 ALP ≥ 2×ULN，且 2 < R < 5
- **组织学与临床**：兼具上述两类特征

### 肝血管损伤型
- **不依赖R值**
- 包括：肝窦阻塞综合征（SOS）、布-加综合征（Budd-Chiari syndrome）、门静脉血栓等
- 机制多与药物对肝血管内皮的直接毒性相关

## 急性 vs 慢性 DILI
- **急性DILI**：起病后6个月内肝酶恢复正常
- **慢性DILI**：持续>6个月的肝酶异常，或影像/组织学显示纤维化、肝硬化等慢性改变

## 辅助诊断线索（肝活检）
支持DILI的组织学表现包括：
- 汇管区或实质内嗜酸性粒细胞浸润
- 小泡性脂肪变性（尤其见于某些抗生素或抗癫痫药）
- 重金属颗粒沉积（如金制剂、砷剂）
- 肉芽肿（如别嘌呤醇、奎尼丁）

> 注：肝活检非必需，但在诊断困难或需排除其他病因时具有重要价值。
