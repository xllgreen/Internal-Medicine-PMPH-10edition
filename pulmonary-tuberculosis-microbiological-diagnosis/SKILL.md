# 肺结核病原学确诊与痰标本采集规范

## 适用对象
- 有咳嗽、咳痰持续≥2周等肺结核可疑症状者
- 胸部影像学显示肺部异常阴影者

## 启动条件
IF 患者有咳嗽咳痰≥2周 **或** 影像学异常 THEN 必须进行痰结核分枝杆菌检查。

## 执行步骤

### 1. 痰标本采集
- **初诊患者**：至少送检3份痰标本，包括清晨痰、夜间痰和即时痰。
- **复诊患者**：每次送检2份痰标本。
- **无痰患者**：采用痰诱导技术获取标本。
> **理由**：结核分枝杆菌排菌具有间断性和不均匀性，多次采样可显著提高检出率。

### 2. 痰涂片检查
- **方法**：齐-尼（Ziehl-Neelsen）染色法 或 WHO推荐的LED荧光显微镜法。
- **敏感度阈值**：每毫升痰含5,000–10,000个细菌方可呈阳性。
- **局限性**：阳性仅表示存在抗酸杆菌，不能区分结核分枝杆菌与非结核分枝杆菌，需结合临床综合判断。

### 3. 培养法（诊断“金标准”）
- **改良罗氏法（Lowenstein-Jensen）**：耗时2–8周。
- **BACTEC-TB 960液体培养法**：约10日出结果，分离率提高10%，并可提供菌株用于药敏试验和菌种鉴定。

### 4. 分子生物学检查
- **Xpert MTB/RIF**：实时荧光定量PCR，可同时检测MTB复合群及利福平耐药性。
- **等温扩增技术（LAMP、CPA、SAT）**：主要用于MTB复合群检测。
- **mNGS/tNGS**：不依赖培养，阳性结果可作为病原学阳性依据，但必须结合临床综合判断。

### 5. 病原学阳性判定
满足以下任一条件即为病原学阳性：
- 痰涂片阳性
- 培养阳性
- 分子生物学检测阳性

### 6. 特殊情况处理
若所有病原学检测均为阴性，但同时满足以下条件，可诊断为“病原学阴性肺结核”：
- 典型肺结核影像学表现
- 相符的临床症状
- 免疫学检测阳性（如IGRA或PPD中度以上阳性）**或** 肺外组织病理证实结核

## 输出结果
输出病原学检查结果（涂+/−、培+/−、分子+/−），用于：
- 确诊肺结核
- 分类为病原学阳性或阴性
- 制定化疗方案
- 疗效评估
