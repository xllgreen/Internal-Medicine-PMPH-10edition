# 氧合指数计算与校正规则

## PaO₂/FiO₂ 计算
- 单位：mmHg
- FiO₂ 以小数形式输入（如 40% → 0.4）
- 海拔校正公式（适用于海拔 >1000 米）：
  ```
  校正值 = (PaO₂ / FiO₂) × (当地大气压 / 760)
  ```
  其中，当地大气压（mmHg）≈ 760 - (海拔米数 / 10.5)

## SpO₂/FiO₂ 使用条件
- 仅当 SpO₂ ≤ 97% 时有效
- SpO₂/FiO₂ 与 PaO₂/FiO₂ 的对应关系（基于临床验证）：
  - SpO₂/FiO₂ ≤ 315 ≈ PaO₂/FiO₂ ≤ 300
  - SpO₂/FiO₂ ≤ 235 ≈ PaO₂/FiO₂ ≤ 200
  - SpO₂/FiO₂ ≤ 148 ≈ PaO₂/FiO₂ ≤ 100

| 状态 | 支持条件 | 轻度 | 中度 | 重度 |
|------|--------|------|------|------|
| **未插管** | HFNC ≥30 L/min 或 NPPV EPAP/CPAP ≥5 cmH₂O | PaO₂/FiO₂ ≤300 或 SpO₂/FiO₂ ≤315 | — | — |
| **已插管** | 任意 PEEP | PaO₂/FiO₂ 200–300 或 SpO₂/FiO₂ 235–315 | PaO₂/FiO₂ 100–200 或 SpO₂/FiO₂ 148–235 | PaO₂/FiO₂ ≤100 或 SpO₂/FiO₂ ≤148 |
| **资源有限地区** | 无支持要求 | SpO₂/FiO₂ ≤315（SpO₂ ≤97%）即诊断ARDS | — | — |
