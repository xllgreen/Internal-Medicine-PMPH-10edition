# 原发性干燥综合征ACR/EULAR 2016诊断

## 适用场景
- 患者有至少1项干燥症状（如眼干/口干≥3个月、每日人工泪液≥3次等）
- 已排除头颈部放疗、丙肝、艾滋病、结节病等
- 未服用抗胆碱药物或已停药足够时间

## 执行步骤
1. **确认前提条件**：
   - 至少1项干燥症状
   - 无排除性疾病

2. **评分项目（累加）**：
   - 唇腺活检灶性指数≥1灶/4mm²：3分
   - 抗SSA(Ro)抗体阳性：3分
   - OSS评分≥5或van Bijsterveld≥4：1分
   - Schirmer试验≤5mm/5min：1分
   - 未刺激唾液流率≤0.1ml/min：1分

3. **诊断判定**：
   - 总分≥4分 → 诊断为原发性干燥综合征（pSS）

## 注意事项
- 继发性干燥综合征不适用此标准
- 抗胆碱药物影响泪液/唾液测试，需停药后再评估
- 抗SSA阴性但活检阳性仍可能诊断pSS
