# 压力梯度计算详细参数与示例

## 标准参数值（单位：cmH₂O）
- **壁胸膜毛细血管静水压**：30 cmH₂O
- **脏胸膜毛细血管静水压**：24 cmH₂O
- **胸腔内压**：-5 cmH₂O（计算时取绝对值 5）
- **胸腔积液胶体渗透压**：5 cmH₂O
- **毛细血管内胶体渗透压**（壁/脏胸膜共用）：34 cmH₂O

## 计算示例
### 壁胸膜压力梯度
```
30 + 5 + 5 - 34 = 6 cmH₂O → 正值，表示液体滤出
```

### 脏胸膜压力梯度
```
24 + 5 + 5 - 34 = 0 cmH₂O → 无净滤出，作用微弱
```

## 生理意义
- 液体滤过主要发生在胸腔上部（壁胸膜区域）
- 液体吸收主要位于横膈和胸腔下部纵隔胸膜
- 该模型解释了临床观察到的胸腔积液多源于壁层胸膜的现象

## 约束条件
- 不适用于胸壁创伤、穿刺损伤或手术导致的直接液体漏出
- 假设胸腔积液胶体渗透压恒定为 5 cmH₂O；若为高蛋白渗出液，需调整该值并重新评估
