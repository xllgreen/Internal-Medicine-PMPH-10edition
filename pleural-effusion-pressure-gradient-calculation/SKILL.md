# 胸腔积液形成的压力梯度计算

## 何时使用
- 需要分析胸腔积液的形成机制（如区分漏出液与渗出液的生理基础）
- 解释为何液体主要从壁胸膜滤出而非脏胸膜
- 评估胸膜腔液体动力学在疾病状态下的变化

## 执行步骤
1. 确认已知参数：毛细血管静水压（区分壁/脏胸膜）、胸腔内压、胸腔积液胶体渗透压、毛细血管内胶体渗透压。
2. 应用压力梯度公式：
   ```
   压力梯度 = 毛细血管内流体静水压 + 胸腔内负压绝对值 + 胸腔积液胶体渗透压 - 毛细血管内胶体渗透压
   ```
3. 分别计算壁胸膜和脏胸膜的压力梯度。
4. 解读结果：正值表示净滤出，零或负值表示无滤出或吸收。
5. 结合解剖位置（上部滤出、下部吸收）综合判断积液形成与清除模式。

> 注意：仅适用于非外伤性、非医源性胸腔积液机制分析。
