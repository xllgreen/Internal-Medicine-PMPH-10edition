# 血清肌酐检测临床应用

## 何时使用
- 需要快速、简便地初步评估肾小球滤过功能（GFR）
- 对病情稳定的患者进行常规肾功能筛查
- 不适用于早期肾损伤（GFR下降 <50%）的检测场景

## 执行步骤
1. **样本采集**：采集患者空腹静脉血，测定血清肌酐（Scr）浓度（单位：mg/dl）。
2. **结果解读前确认影响因素**：
   - **性别**：男性通常肌酐水平高于女性
   - **年龄**：老年人肌肉量减少，肌酐可能偏低
   - **肌肉量**：肌肉发达者肌酐偏高；消瘦、截肢或营养不良者偏低
   - **蛋白质摄入**：高蛋白饮食可短暂升高肌酐
   - **用药史**：如西咪替丁可抑制肾小管分泌肌酐，导致假性升高
3. **临床意义判断**：
   - 血清肌酐仅在肾小球滤过率（GFR）下降 ≥50% 时才显著升高
   - 因此不能用于早期肾损害筛查
   - 适用于对已知慢性肾病患者或稳定状态下的粗略肾功能评估
4. **综合判断**：该方法敏感性低，必须结合估算肾小球滤过率（eGFR）、尿蛋白或其他肾功能检查进行综合评估。

> 注意：若临床怀疑早期肾损伤，应优先选择更敏感的指标（如胱抑素C、尿微量白蛋白等），而非依赖血清肌酐。
