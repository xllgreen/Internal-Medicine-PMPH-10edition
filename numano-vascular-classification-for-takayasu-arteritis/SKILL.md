# Numano大动脉炎血管分型及临床表现评估

## 适用条件
- 患者已确诊为大动脉炎（Takayasu arteritis）
- 已完成血管影像学评估（如CTA、MRA或血管造影）
- 已排除动脉粥样硬化、纤维肌发育不良等其他血管狭窄病因

## 执行步骤

1. **确定受累血管段**：根据影像学结果，识别主动脉及其一级分支（包括主动脉弓分支、升主动脉、降主动脉、腹主动脉、肾动脉等）的病变范围。

2. **应用Numano分型标准**：
   - **I型**：仅累及主动脉弓分支（头臂干、左颈总动脉、左锁骨下动脉）
   - **II型**：累及升主动脉、主动脉弓及其分支
   - **III型**：累及降主动脉及双侧肾动脉
   - **IV型**：仅累及腹主动脉及双侧肾动脉
   - **V型**：累及主动脉全程及其所有一级分支

3. **关联临床表现与体征**：
   - I/II型：头痛、头晕、视力模糊、咀嚼无力；上肢发凉、麻木、无力；颈/桡动脉搏动减弱或消失；颈部或锁骨下窝血管杂音
   - III/IV型：顽固性高血压、腹痛、间歇性跛行；下肢血压低于上肢；腹部或背部血管杂音（III型有，IV型无）
   - V型：综合I–IV型所有症状

4. **判断疾病分期**：
   - **无脉前期（全身期）**：发热、盗汗、体重下降、关节痛，偶有血管压痛或杂音
   - **无脉期（缺血期）**：以各型对应的器官缺血表现为主

5. **输出结果**：明确Numano分型（I–V型），用于指导治疗决策、手术指征评估及预后判断。

> **注意**：约90%患者在30岁前发病，女性占80–90%，亚洲人群高发。
