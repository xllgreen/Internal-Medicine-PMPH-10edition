# 心力衰竭按左心室射血分数分类

## 适用场景
- 患者确诊为心力衰竭
- 已通过超声心动图等影像学手段测定LVEF值
- 排除未完成LVEF评估的疑似心衰患者

## 执行步骤

### 1. 获取LVEF数值
使用改良双平面Simpson法测量左心室射血分数（LVEF）。

### 2. 分类标准
根据LVEF值进行分类：
- **HFrEF（射血分数降低的心衰）**：LVEF ≤ 40%
- **HFmrEF（射血分数轻度降低型心衰）**：LVEF 41%～49%
- **HFpEF（射血分数保留型心衰）**：LVEF ≥ 50%

### 3. 特殊情况处理
- **HFimpEF（射血分数改善的心衰）**：既往LVEF ≤ 40%，治疗后LVEF > 40%
- 注意LVEF是动态指标，需定期监测避免单次测量误判

## 输出结果
输出心衰的具体亚型（HFrEF、HFmrEF、HFpEF或HFimpEF），用于指导后续治疗方案选择。
