# 溶血性贫血的发病机制与分类

## 何时使用
- 用户询问溶血性贫血的根本原因或病理机制
- 需要对疑似溶血性贫血患者进行分类诊断
- 区分遗传性（如遗传性球形红细胞增多症、G6PD缺乏症）与获得性（如自身免疫性溶血性贫血、阵发性睡眠性血红蛋白尿）类型
- 解释实验室检查结果（如LDH升高、结合珠蛋白降低、网织红细胞增多）背后的机制

## 执行步骤
1. **确认溶血存在**：评估间接胆红素升高、乳酸脱氢酶（LDH）升高、结合珠蛋白降低、网织红细胞计数升高等指标。
2. **判断溶血部位**：
   - 血管内溶血：血红蛋白尿、含铁血黄素尿、血浆游离血红蛋白升高
   - 血管外溶血：脾脏肿大、胆结石、无血红蛋白尿
3. **区分病因类别**：
   - **遗传性**：红细胞膜缺陷（如球形红细胞）、酶缺陷（如G6PD、丙酮酸激酶缺乏）、血红蛋白病（如镰状细胞病）
   - **获得性**：免疫性（如温抗体型、冷抗体型AIHA）、微血管病性（如TTP、HUS）、感染/药物/毒素诱发
4. **整合临床信息**：结合病史、家族史、药物暴露、伴随症状及特异性检查（如Coombs试验、流式细胞术检测CD55/CD59）完成分类。
