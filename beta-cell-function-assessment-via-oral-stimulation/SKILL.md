# 口服刺激下β细胞功能评估

## 适用对象
- 疑似糖尿病患者
- 需评估β细胞功能者
- 使用外源性胰岛素治疗者（仅限C肽试验）
- 需排除胰岛素抗体干扰者（仅限C肽试验）

## 前提条件
- 空腹至少8小时
- 无急性疾病或应激状态
- 口服75g无水葡萄糖或等效碳水化合物（如100g标准面粉馒头）

## 执行步骤
1. **采集基础样本**：空腹状态下采集血浆胰岛素（正常范围35～145 pmol/L）或C肽（正常基础值≥400 pmol/L）。
2. **给予刺激**：口服75g无水葡萄糖或等效碳水化合物。
3. **动态采样**：分别于服糖后30、60、120、180、240分钟采集血样，检测胰岛素或C肽水平。
4. **结果判读**：
   - 正常反应：30～60分钟达峰，峰值为基础值的5～10倍（胰岛素）或5～6倍（C肽），3～4小时恢复至基础水平。
   - 异常表现：峰值不足（<5倍基础值）、达峰延迟或恢复缓慢，提示β细胞功能受损。

## 注意事项
- 胰岛素释放试验不能单独用于IFG/IGT诊断。
- C肽释放试验不受外源性胰岛素或胰岛素抗体干扰，更适合已接受胰岛素治疗者。
- 严重肝肾功能不全可能影响C肽清除，需谨慎解读结果。

> 提示：根据患者是否使用外源性胰岛素选择胰岛素或C肽作为检测指标。
