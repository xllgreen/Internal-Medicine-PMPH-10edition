# 使用CRAB+SLiM标准诊断活动性多发性骨髓瘤

## 适用场景
- 患者已确认存在单克隆浆细胞增殖（骨髓浆细胞比例 ≥10% 或组织活检证实为浆细胞瘤）
- 需评估是否存在骨髓瘤相关终末器官损害或高风险无症状指标
- 排除冒烟性骨髓瘤、MGUS及反应性浆细胞增多症等鉴别诊断

## 执行步骤

1. **确认单克隆浆细胞增殖证据**：
   - 骨髓中单克隆浆细胞比例 ≥10%，或
   - 组织活检证实为浆细胞瘤

2. **评估是否满足以下任一类别条件**：

   ### A. CRAB 靶器官损害（满足任一项即符合）
   - **[C] 高钙血症**：校正血清钙 > 2.75 mmol/L
   - **[R] 肾功能损害**：肌酐清除率 <40 ml/min 或 血清肌酐 >177 μmol/L
   - **[A] 贫血**：血红蛋白 <100 g/L 或低于正常下限20 g/L以上
   - **[B] 骨病变**：影像学（X线、CT或PET-CT）显示1处或多处溶骨性病变

   ### B. SLiM 高风险无症状指标（满足任一项即符合）
   - **[S]** 骨髓单克隆浆细胞比例 ≥60%
   - **[Li]** 受累/非受累血清游离轻链比值 ≥100（且受累轻链浓度 ≥100 mg/L）
   - **[M]** MRI显示 >1处直径 ≥5 mm的局灶性骨髓病变

3. **特殊情形处理**：
   - 若患者M蛋白阴性，但骨髓浆细胞 ≥10%，诊断为“不分泌型多发性骨髓瘤”，仍适用本标准。

4. **输出结论**：
   - 若同时满足第1步和第2步中任一条件，则诊断为活动性（有症状）多发性骨髓瘤，需启动系统治疗。
