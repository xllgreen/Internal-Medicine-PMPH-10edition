# IBS亚型分类（基于粪便性状量表）

## 适用条件
- 患者已完成IBS诊断
- 拥有至少2周的完整排便记录，可用于粪便类型统计
- 排便记录足以支持粪便性状比例计算

## 执行步骤
1. **使用布里斯托粪便性状量表（Bristol Stool Form Scale, BSFS）对每次排便进行分类**：
   - 1型：羊粪状硬块
   - 2型：香肠状但表面凹凸
   - 3型：香肠状带裂纹
   - 4型：光滑柔软香肠状
   - 5型：软团状易排出
   - 6型：糊状
   - 7型：水样无固体

2. **统计关键粪便类型比例**：
   - 计算1型和2型粪便占总排便次数的比例（stool_type_1_or_2_ratio）
   - 计算6型和7型粪便占总排便次数的比例（stool_type_6_or_7_ratio）

3. **根据比例规则确定IBS亚型**：
   - **IBS便秘型（IBS-C）**：stool_type_1_or_2_ratio > 0.25 且 stool_type_6_or_7_ratio < 0.25
   - **IBS腹泻型（IBS-D）**：stool_type_6_or_7_ratio > 0.25 且 stool_type_1_or_2_ratio < 0.25
   - **IBS混合型（IBS-M）**：stool_type_1_or_2_ratio > 0.25 且 stool_type_6_or_7_ratio > 0.25
   - **IBS不定型（IBS-U）**：不符合上述任一标准，但仍符合IBS诊断

4. **输出结果**：返回明确的IBS亚型（IBS-C / IBS-D / IBS-M / IBS-U），用于指导后续药物选择和管理策略。

> 注意：避免基于单次或少量排便记录进行分类；建议使用至少14天的排便日记以确保分类可靠性。
