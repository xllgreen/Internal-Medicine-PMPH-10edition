# NYHA心力衰竭功能分级评估

## 何时使用
- 患者已确诊为**慢性心力衰竭**；
- 患者处于**病情稳定期**（非急性发作或急剧恶化阶段）；
- 需要评估其**日常活动耐量**、**症状严重程度**，用于：
  - 调整药物治疗方案；
  - 制定康复计划；
  - 判断预后；
  - 决定是否需高级干预（如心脏移植）。

## 如何执行
1. **询问或观察**患者在不同活动强度下是否出现典型心衰症状（如呼吸困难、乏力、心悸等）。
2. **依据以下标准进行分级**：
   - **Ⅰ级**：日常活动不受限，一般活动不引起症状；
   - **Ⅱ级**：体力活动轻度受限，休息无症状，但一般活动（如步行、爬楼梯）即可诱发症状；
   - **Ⅲ级**：体力活动明显受限，低于日常一般活动（如穿衣、洗漱）即出现症状；
   - **Ⅳ级**：无法进行任何体力活动，休息时已有症状，活动后加重。
3. **注意事项**：
   - NYHA分级基于**主观感受**，个体差异大，应结合客观检查（如6分钟步行试验、心肺运动试验）综合判断；
   - 分级可能随病情波动而变化，建议**动态重复评估**；
   - 不适用于急性心衰或病情不稳定期。
4. **输出结果**：明确记录NYHA分级（Ⅰ、Ⅱ、Ⅲ或Ⅳ级），用于后续临床决策。
