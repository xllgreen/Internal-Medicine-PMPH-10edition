# 关节液白细胞计数鉴别关节炎类型

## 适用场景
- 患者存在急性或慢性关节肿痛
- 已完成关节腔穿刺并获得关节液样本
- 需快速区分关节炎类型以指导后续诊疗

## 执行步骤
1. **立即对新鲜关节液进行白细胞计数和分类**：
   - 若白细胞计数 < 2000 × 10⁶/L → 判断为“非炎性关节炎”（如骨关节炎）
   - 若白细胞计数 > 3000 × 10⁶/L 且中性粒细胞比例 ≥ 50% → 判断为“炎性关节炎”（如类风湿关节炎、痛风、反应性关节炎等）
   - 若关节液呈脓性，且白细胞计数显著升高（通常 >10,000 × 10⁶/L）→ 提示“化脓性关节炎”

2. **根据初步判断决定下一步检查**：
   - 炎性或疑似晶体性关节炎 → 进行偏振光显微镜检查尿酸钠或焦磷酸钙结晶
   - 脓性或高度怀疑感染 → 立即送检细菌涂片和培养

3. **注意事项**：
   - 关节液必须新鲜送检，避免细胞溶解导致计数偏低
   - 化脓性关节炎早期白细胞计数可能仅轻度升高，需结合临床高度警惕
   - 假性痛风等晶体性关节炎的白细胞计数可能处于2000–3000 × 10⁶/L之间，不可仅凭计数排除

## 输出结果
返回关节炎类型（非炎性/炎性/化脓性），并明确提示是否需进行结晶检查或微生物培养。
