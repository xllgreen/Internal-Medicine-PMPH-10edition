# 肝硬化病因分类与地域化诊断流程

## 触发条件
当患者已被确诊为肝硬化，且需明确具体病因为治疗或研究目的时，启动本流程。

## 执行步骤

### 1. 识别肝硬化常见病因大类
按以下类别系统筛查：
- **病毒性肝炎**：乙型肝炎病毒（HBV）、丙型肝炎病毒（HCV）
- **脂肪性肝病**：酒精性或非酒精性脂肪性肝病
- **免疫性肝病**：自身免疫性肝炎、原发性胆汁性胆管炎等
- **药物或化学毒物**：长期使用肝毒性药物或接触工业毒物
- **胆汁淤积性疾病**：原发性或继发性
- **循环障碍**：如布-加综合征、慢性心功能不全、缩窄性心包炎
- **寄生虫感染**：血吸虫、华支睾吸虫等
- **遗传和代谢性疾病**：如Wilson病、血色病、α₁-抗胰蛋白酶缺乏症
- **隐源性肝硬化**：仅在排除所有可识别病因后考虑

### 2. 根据地域调整病因排查优先级
- **在中国及亚洲部分地区**：优先排查 **乙型肝炎病毒（HBV）感染**
- **在欧美国家**：优先考虑 **酒精性肝病** 和 **丙型肝炎病毒（HCV）感染**

### 3. 对疑似遗传代谢性疾病进行专项筛查
- **Wilson病**：检测ATP7B基因（13q14）、血清铜蓝蛋白、24小时尿铜
- **原发性血色病**：检查第6对染色体相关基因、血清铁、转铁蛋白饱和度、肝脏铁沉积
- **α₁-抗胰蛋白酶缺乏症**：测定血清α₁-抗胰蛋白酶（α₁-AT）水平，必要时进行基因分型（如PiZZ表型）

### 4. 隐源性肝硬化的诊断原则
仅在全面排除上述所有明确病因后，方可谨慎诊断为“隐源性肝硬化”，并需安排定期随访以避免漏诊潜在可治疗病因。

## 注意事项
- 本流程不适用于急性肝损伤或非肝硬化性肝病患者。
- 病因诊断直接影响治疗策略，务必系统、有序地完成评估。
