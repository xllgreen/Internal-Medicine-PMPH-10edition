# 内分泌和代谢性疾病的三级诊断框架

## 适用场景
- 患者存在疑似内分泌或代谢异常的临床表现
- 具备基础实验室和影像学检查条件
- 排除急性创伤、感染等非内分泌代谢性急症

## 执行步骤

### 1. 临床表现评估
系统收集症状与体征，重点关注内分泌腺体靶器官受累表现：
- 甲亢：怕热多汗、心悸
- 甲减：畏寒乏力、水肿
- 库欣综合征：向心性肥胖、紫纹

### 2. 功能诊断
通过激素水平测定判断腺体功能状态：
- **功能亢进**：靶腺激素水平升高，伴相应临床综合征
- **功能减退**：靶腺激素水平降低，伴相应临床综合征
- 必要时进行动态功能试验（如TRH兴奋试验、ACTH刺激试验、葡萄糖抑制试验）

### 3. 定位诊断
确定病变部位在靶腺、垂体或下丘脑：
- 测定上游调节激素（TSH、ACTH、GH等）
- 影像学检查（垂体MRI、肾上腺CT、甲状腺超声）
- 利用激素反馈机制判断（如原发性甲减TSH↑、FT4↓；中枢性甲减TSH正常或轻度↑但FT4↓）

### 4. 病因诊断
结合病史、免疫指标、基因检测、病理等明确病因：
- 自身免疫性（Graves病、桥本甲状腺炎）
- 肿瘤性（垂体瘤、嗜铬细胞瘤）
- 遗传性（先天性肾上腺皮质增生症）
- 继发性（垂体卒中导致腺垂体功能减退）

## 输出结果
生成完整的诊断路径报告，明确功能状态（亢进/减退）、病变定位（原发/继发/中枢）及可能病因，指导后续治疗策略。
