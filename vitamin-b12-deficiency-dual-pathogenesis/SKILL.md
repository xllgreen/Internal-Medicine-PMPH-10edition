# 维生素B₁₂缺乏的双重病理机制

## 何时使用
- 血清维生素B₁₂水平低于74pmol/L（100ng/ml）；
- 患者同时或分别表现出巨幼细胞贫血和/或神经系统症状（如对称性远端麻木、深感觉障碍、共济失调、抑郁、幻觉等）；
- 需鉴别B₁₂缺乏与叶酸缺乏所致的血液学改变。

## 执行步骤

1. **评估甲硫氨酸合成酶通路障碍**：
   - 确认维生素B₁₂以甲基钴胺素形式参与甲硫氨酸合成酶反应；
   - 判断该反应是否受阻（B₁₂缺乏 → 同型半胱氨酸 + N⁵-甲基FH₄ → 甲硫氨酸 + FH₄ 反应中断）；
   - 推断是否出现“叶酸陷阱”（N⁵-甲基FH₄堆积但无法转化为FH₄）；
   - 分析是否导致dTTP合成减少、DNA合成障碍，进而引发巨幼细胞贫血。

2. **评估L-甲基丙二酰-CoA变位酶通路障碍**：
   - 确认维生素B₁₂以5-脱氧腺苷钴胺素形式参与该酶反应；
   - 判断L-甲基丙二酰-CoA是否因反应受阻而堆积；
   - 分析异常脂肪酸是否掺入神经髓鞘；
   - 同时评估甲硫氨酸合成障碍是否导致S-腺苷甲硫氨酸减少，进而损害神经细胞甲基化反应。

3. **整合双重机制解释临床表现**：
   - 造血系统症状归因于DNA合成障碍（与叶酸缺乏相似）；
   - 神经系统症状归因于髓鞘异常和甲基化障碍（为B₁₂缺乏特有，叶酸缺乏不引起）。

> 注意：本技能仅适用于依赖B₁₂的酶促反应系统，不用于解释单纯叶酸缺乏所致症状。
