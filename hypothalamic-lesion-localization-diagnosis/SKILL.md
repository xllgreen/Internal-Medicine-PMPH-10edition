# 下丘脑病损定位诊断

## 适用条件
- 患者为疑似下丘脑综合征或伴有多种内分泌与神经系统症状的个体
- 已排除单一靶腺或垂体原发病
- 存在至少一项非典型内分泌症状（如体温异常、嗜睡、肥胖）
- 不适用于急性脑干或大脑皮层病变
- 需结合影像学确认结构性病损

## 执行步骤

### 1. 根据核心症状匹配下丘脑功能区
- **水代谢异常**（尿崩症、SIADH） → 病变位于**视上核、室旁核、室周器**
- **体温异常**（高热或低体温） → **下丘脑前部视前区**（高热）或**后部**（低体温）受损
- **多食肥胖** → **腹内侧核**（饱食中枢）损伤
- **厌食消瘦** → **下丘脑外侧区**（摄食中枢）或**中部外侧区**病变
- **嗜睡/昏迷** → **下丘脑后部**（觉醒中枢）损伤
- **性功能减退 + 尿崩 + ACTH/GH/PRL异常** → **腹内侧正中隆起**受损
- **记忆障碍** → **乳头体或腹内侧核**病变

### 2. 判断病损性质对临床表现的影响
- **慢性损伤性病损**（如肿瘤压迫）：导致功能丧失（如低体温、失眠）
- **急性兴奋性病损**（如炎症、出血）：导致功能亢进（如高体温、嗜睡）

### 3. 考虑年龄相关差异
- 青春期前GHRH缺乏 → 身材矮小
- 成人GHRH缺乏 → GH缺乏综合征（无身材改变）

### 4. 辅助验证手段
- 使用GnRH/TRH兴奋试验区分下丘脑与垂体病变
- 安排MRI检查以确认结构异常

## 输出建议
提示特定下丘脑区域受损，建议针对性影像学检查及激素动态试验。
