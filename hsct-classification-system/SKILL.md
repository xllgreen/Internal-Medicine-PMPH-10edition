# 造血干细胞移植分类体系

## 何时使用
- 需要为血液病患者制定造血干细胞移植方案
- 需明确供者选择策略和移植类型
- 需综合评估供受者匹配情况以指导临床决策

## 执行步骤

1. **按供者来源分类**：
   - **异体HSCT（allo-HSCT）**：供者为他人
     - 若为同卵孪生者间移植 → **同基因移植**（无GVHD和排斥风险）
     - 其余情况 → **异基因移植**
   - **自体HSCT（auto-HSCT）**：供者即患者本人

2. **按干细胞采集部位细分**：
   - 骨髓移植（BMT）
   - 外周血干细胞移植（PBSCT）
   - 脐带血移植（CBT）

3. **按供受者血缘关系分类**：
   - 血缘移植
   - 无血缘移植（UDT）

4. **按HLA配型相合程度分类**：
   - HLA相合移植
   - 部分相合移植
   - 单倍型相合（haploidentical）移植

5. **综合输出完整分类描述**，例如：“HLA相合同胞供者的PBSCT”或“单倍型相合亲缘供者的BMT”，用于指导供者选择和移植方案制定。

## 前提条件
- 已明确供者来源
- 已明确干细胞采集部位
- 已获得HLA配型结果
- 已确认供受者血缘关系

## 约束条件
- 不适用于非造血系统疾病的一般治疗
- 不适用于未完成预处理的患者
