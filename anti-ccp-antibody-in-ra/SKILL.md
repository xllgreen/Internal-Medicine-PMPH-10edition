# 抗CCP抗体在类风湿关节炎中的应用

## 适用场景
- 患者存在关节肿痛症状
- 已排除其他明确病因
- 需评估RA可能性或预后

## 执行步骤
1. **确认检测结果**：
   - 抗CCP阳性（记录滴度：低/中/高）
   - 同时获取RF结果

2. **评估诊断价值**：
   - 抗CCP特异性>95%，优于RF
   - 可在临床症状出现前阳性，有助于早期诊断
   - 不能单独确诊RA，必须结合临床表现

3. **评估预后价值**：
   - 抗CCP阳性提示更高风险出现关节侵蚀和影像学破坏
   - 与多关节受累、晨僵时间长相关

4. **临床决策建议**：
   - 若抗CCP阳性但无典型症状：密切随访
   - 若抗CCP阳性且有典型症状：支持RA诊断
   - 联合RF检测可提高敏感性

## 注意事项
- 低滴度阳性需谨慎解读，结合临床
- 不适用于已确诊其他关节炎的患者
