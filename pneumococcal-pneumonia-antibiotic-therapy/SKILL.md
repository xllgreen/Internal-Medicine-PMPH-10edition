# 肺炎链球菌肺炎的抗菌治疗方案

## 触发条件
- 患者诊断为肺炎链球菌肺炎（SP肺炎）
- 无青霉素过敏史
- 已评估病情严重程度及是否存在并发症（如脑膜炎、休克）

## 执行步骤

### 1. 根据病情严重程度选择青霉素G剂量
- **轻症**（门诊治疗，无并发症）：
  - 青霉素G 240万U/天，分3次肌内注射；或
  - 普鲁卡因青霉素60万U，每12小时肌注1次
- **中症**（需住院，无休克/脑膜炎）：
  - 青霉素G 240万～480万U/天，分次静脉滴注，每6～8小时1次
- **重症或并发脑膜炎**：
  - 青霉素G 1000万～3000万U/天，分4次静脉滴注

### 2. 处理耐药情况
- 若当地SP对青霉素不敏感率高，或MIC ≤ 8 mg/L：
  - 采用大剂量方案：青霉素G 320万U，每4小时1次（依据欧洲指南）
  - 此方案可覆盖MIC ≤ 8 mg/L菌株，并减少广谱抗生素诱导的MRSA和艰难梭菌传播

### 3. 青霉素过敏或耐药时的替代方案
- 呼吸氟喹诺酮类：莫西沙星、吉米沙星、左氧氟沙星
- 第三代头孢菌素：头孢噻肟、头孢曲松
- MDR菌株：万古霉素、替考拉宁或利奈唑胺

### 4. 疗程与停药指征
- 标准疗程：5～7天
- 停药条件：热退2～3天且主要呼吸道症状明显改善
- **注意**：不必等待肺部阴影吸收

### 5. 疗效监测
- 有效治疗后体温通常在1～3天内恢复正常
- 若72小时内无改善，需重新评估：耐药、特殊病原体、并发症或误诊

- `severity_level`：mild / moderate / severe
- `penicillin_allergy`：true / false
- `local_resistance_rate`：当地青霉素耐药率
- `complications`：是否合并脑膜炎、休克等
