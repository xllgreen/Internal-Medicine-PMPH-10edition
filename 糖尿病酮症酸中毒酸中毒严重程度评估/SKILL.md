# 糖尿病酮症酸中毒酸中毒严重程度评估

## 何时使用
在以下情况下启用本技能：
- 患者确诊或疑似糖尿病酮症酸中毒（DKA）；
- 出现典型临床表现（如深快呼吸、意识改变、脱水）；
- 动脉血气分析显示pH<7.3，需量化酸中毒严重程度。

## 执行步骤
1. **确认DKA诊断**：核实存在高血糖、血酮升高及代谢性酸中毒三联征。
2. **获取关键实验室指标**：
   - 动脉血pH值；
   - 血清碳酸氢根（HCO₃⁻）浓度；
   - 阴离子间隙（AG）。
3. **判断酸中毒严重程度**：
   - **轻度**：pH < 7.3 或 HCO₃⁻ < 18 mmol/L；
   - **中度**：pH < 7.25 或 HCO₃⁻ < 15 mmol/L；
   - **重度**：pH < 7.0 或 HCO₃⁻ < 10 mmol/L。
4. **评估临床影响**：
   - pH < 7.2：通常出现Kussmaul呼吸；
   - pH 7.0–7.1：可能抑制中枢神经系统，增加心律失常风险。
5. **输出建议**：根据分级结果决定是否需要静脉补碱（通常仅在pH<7.0且血流动力学不稳定时考虑）及监护强度（如转入ICU）。

> 注意：本技能不适用于非酮症性酸中毒（如乳酸酸中毒、尿毒症）或pH≥7.3的代偿期患者。
