# 识别消化性溃疡的典型临床表现与节律性特征

## 何时使用
- 患者主诉慢性上腹痛，且疼痛具有周期性或节律性；
- 患者有长期服用非甾体抗炎药（NSAIDs）或幽门螺杆菌（Hp）感染病史；
- 需要初步判断是否应进一步安排内镜检查以确诊消化性溃疡。

## 执行步骤

1. **评估疼痛性质**：确认疼痛是否为钝痛、隐痛、灼痛、胀痛、剧痛或饥饿样不适。

2. **判断三大特征**：
   - **慢性过程**：症状持续数月甚至十余年；
   - **周期性发作**：每次发作持续数周至数月，常在秋冬或冬春之交加重；
   - **节律性**：
     - 胃溃疡（GU）：多为餐后痛；
     - 十二指肠溃疡（DU）：多为饥饿痛或夜间痛，进食后缓解。

3. **观察缓解因素**：确认疼痛是否可被抑酸药或抗酸剂缓解。

4. **识别非典型表现**：注意部分患者仅表现为上腹胀、厌食、嗳气、反酸等消化不良症状。

5. **警惕无症状性溃疡**：若患者以出血、穿孔等并发症为首发表现，尤其见于长期服用NSAIDs者或老年人，即使无典型疼痛也应考虑溃疡可能。

6. **特殊人群调整判断**：
   - 老年人：疼痛可能无规律，伴随体重减轻或贫血；
   - 儿童：腹痛多位于脐周，常伴恶心、呕吐。

> **输出建议**：若符合上述节律性、周期性、慢性上腹痛特征，应高度怀疑消化性溃疡，需进一步内镜确诊。
