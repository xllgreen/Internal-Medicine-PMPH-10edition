# 高血压血流动力学公式详解

## 核心公式
平均动脉血压（MBP）= 心排血量（CO）× 总外周血管阻力（PR）

## 变量说明

### MBP（平均动脉血压）
- **单位**：mmHg
- **定义**：反映器官灌注压力的核心指标
- **估算方法**：通常为 舒张压 + 1/3 脉压
- **正常范围**：约 70–100 mmHg

### CO（心排血量）
- **单位**：L/min
- **定义**：单位时间内心脏泵出的血量，受心率和每搏输出量影响
- **正常范围**：成人约 4–8 L/min

### PR（总外周血管阻力）
- **单位**：mmHg·min/L（或 dyn·s·cm⁻⁵，1 mmHg·min/L ≈ 80 dyn·s·cm⁻⁵）
- **定义**：小动脉对血流的阻力，主要由血管口径和血液黏度决定
- **正常范围**：约 800–1200 dyn·s·cm⁻⁵（即 10–15 mmHg·min/L）

## 年龄相关变化模式

| 年龄阶段 | 主要变化 | 血压表现 | 病理生理机制 |
|----------|--------|--------|------------|
| 年轻人 | CO↑，PR 正常或轻度↑ | 收缩压升高为主 | 交感神经激活，心输出增加 |
| 中年人（30–50岁） | PR↑，CO 正常 | 舒张压升高明显 | 小动脉重构，阻力增加 |
| 老年人 | 大动脉弹性减退，PR 结构改变（壁/腔比值↑） | 收缩压↑、舒张压↓、脉压增大 | 动脉硬化，缓冲功能下降 |

## 临床分型与治疗指导

- **高心排型高血压**：CO 显著升高 → 优先考虑 β 受体阻滞剂
- **高阻型高血压**：PR 显著升高 → 优先考虑钙通道阻滞剂、ACEI/ARB
- **混合型**：CO 和 PR 均升高 → 联合用药策略

## 注意事项
- 公式为稳态模型，不适用于动态剧烈变化状态
- PR 通常通过计算得出（PR = MBP / CO），而非直接测量
- 在心衰、瓣膜病等情况下需谨慎解读
