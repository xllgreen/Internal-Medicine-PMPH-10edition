# 巨幼细胞贫血诊断与试验性治疗验证

## 适用场景
- 患者表现为大细胞性贫血（MCV、MCH增高）
- 外周血中性粒细胞核分叶过多（≥5叶核>5%或出现6叶以上）
- 骨髓象显示红系、粒系、巨核系“核幼浆老”的巨幼变
- 血清叶酸<6.8nmol/L（3ng/ml）和/或维生素B₁₂<74pmol/L（100ng/ml）
- 已排除骨髓增生异常综合征（MDS）、白血病等其他病因

## 执行步骤

### 1. 初步诊断确认（必须全部满足）
- 存在叶酸或维生素B₁₂缺乏的病因（如偏食、素食、胃切除、烹调不当等）及相应临床表现（贫血、舌炎、神经精神症状）
- 外周血呈大细胞性贫血（MCV、MCH增高，MCHC正常），伴中性粒细胞核分叶过多
- 骨髓象显示三系巨幼变，无其他病态造血特征
- 血清叶酸和/或维生素B₁₂水平低于诊断阈值

### 2. 启动试验性治疗验证
- **若怀疑叶酸缺乏**：口服叶酸5～10mg，每日3次
- **若怀疑维生素B₁₂缺乏**：肌注维生素B₁₂ 500μg，每周2次
- **若两者均缺乏**：必须同时补充维生素B₁₂和叶酸

### 3. 观察治疗反应
- 从治疗第5～7天开始监测网织红细胞
- 网织红细胞通常在1周左右达高峰
- 若网织红细胞显著上升（>2%或绝对值明显增加），支持巨幼细胞贫血诊断

### 4. 关键禁忌与注意事项
- **严禁对维生素B₁₂缺乏患者单独补充叶酸**，以免纠正贫血但加重神经系统损害
- 若试验性治疗无效，需重新评估是否为MDS、白血病或其他非巨幼细胞性大细胞性贫血
- 不可仅凭治疗反应确诊，必须结合实验室指标综合判断

> 提示：本技能适用于已完成初步筛查并符合触发条件的病例，不适用于未排除其他病因的大细胞性贫血。
