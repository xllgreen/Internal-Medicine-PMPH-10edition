# T2DM胰岛β细胞功能缺陷三重表现分析

## 适用对象
- 已诊断为T2DM的患者
- 糖尿病前期（IGR）个体
- 正在接受OGTT或高糖钳夹试验的受试者

## 使用前提
- 已完成葡萄糖刺激胰岛素释放试验或静脉葡萄糖耐量试验
- 具备胰岛素及胰岛素原检测数据

## 禁忌与限制
- 不适用于胰岛素抗体阳性或正在使用外源性胰岛素者
- 急性感染或应激状态下结果不可靠，应暂缓评估

## 执行步骤

### 1. 判断分泌量缺陷
- **早期阶段**：空腹胰岛素水平正常或升高，葡萄糖刺激后呈代偿性增多，但因胰岛素抵抗而处于“相对不足”状态。
- **进展阶段**：胰岛素最大分泌能力逐渐下降，最终低于正常水平。

### 2. 识别分泌模式异常（脉冲式分泌缺陷）
- 检查**第一时相胰岛素分泌**（静脉注射葡萄糖后0–10分钟）是否减弱或消失。
- 在口服葡萄糖耐量试验（OGTT）中，观察0–30分钟早时相胰岛素分泌是否延迟、减弱或缺失。
- 评估第二时相：早期是否代偿性升高且峰值后移（可能导致餐后3–5小时反应性低血糖）；晚期是否对葡萄糖及非葡萄糖刺激（如氨基酸）反应均减退。

### 3. 评估分泌质缺陷
- 计算**胰岛素原/胰岛素比值**：若该比值升高，提示β细胞加工前胰岛素能力下降。

### 4. 综合判断与输出
- 若检测显示第一时相缺失或胰岛素原比例升高，提示β细胞功能已受损。
- 建议强化血糖管理，并考虑使用具有β细胞保护作用的药物（如GLP-1受体激动剂）。

> **注意**：本技能聚焦于功能缺陷的识别与分类，不涉及具体药物剂量或治疗方案制定。
