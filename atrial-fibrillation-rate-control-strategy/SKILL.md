# 房颤心室率控制策略

## 触发条件
患者诊断为房颤且未计划立即复律，已完成心功能评估（如射血分数）。

## 执行步骤
1. **设定心室率控制目标**：
   - 无症状且左室收缩功能正常者：静息心室率＜110次/分（宽松控制）
   - 有明显症状或存在心动过速性心肌病者：静息心室率＜80次/分，中等运动时＜110次/分（严格控制）

2. **根据心功能状态选择药物**：
   - 射血分数保留的心衰（HFpEF）：β受体拮抗剂和/或非二氢吡啶类钙通道阻滞剂（维拉帕米、地尔硫䓬）
   - 射血分数下降的心衰（HFrEF）：首选β受体拮抗剂；若控制不达标或有禁忌，加用洋地黄类药物或胺碘酮

3. **注意药物禁忌证**：
   - 维拉帕米/地尔硫䓬禁用于失代偿性心衰或低血压
   - β受体拮抗剂慎用于严重支气管痉挛或严重心动过缓

4. **验证安全性**：
   - 达到目标心室率后，进行24小时动态心电图监测，评估是否存在心动过缓或心脏停搏

5. **难治性病例处理**：
   - 若药物无效且心室率持续过快，考虑房室结消融联合永久起搏器植入
