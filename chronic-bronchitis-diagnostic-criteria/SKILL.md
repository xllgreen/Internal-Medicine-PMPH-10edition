# 慢性支气管炎诊断标准

## 适用对象
- 慢性咳嗽患者
- 呼吸系统疾病疑似患者

## 使用前提
在应用本诊断标准前，确认以下条件已满足：
- 患者存在咳嗽、咳痰或喘息症状
- 症状持续时间满足年限要求

## 执行步骤
1. **确认主要症状**：患者以咳嗽、咳痰为主要表现，可伴有喘息。
2. **核实年度症状持续时间**：每年症状持续 ≥ 3个月。
3. **确认连续病程**：上述症状模式已连续出现 ≥ 2年。
4. **执行排除性诊断**：必须排除以下可引起类似症状的疾病：
   - 支气管哮喘（常有过敏史、支气管激发试验阳性）
   - 嗜酸性粒细胞性支气管炎（诱导痰中嗜酸粒细胞比例 ≥ 3%）
   - 肺结核（伴发热、盗汗、消瘦，痰抗酸杆菌阳性）
   - 支气管肺癌（刺激性咳嗽、痰中带血，胸部CT或支气管镜检查异常）
   - 特发性肺纤维化（听诊有Velcro啰音，高分辨率CT异常）
   - 支气管扩张（大量脓痰、咯血，HRCT确诊）
   - 其他：慢性咽炎、上气道咳嗽综合征、胃食管反流性疾病、心血管疾病等。
5. **得出诊断结论**：若上述所有条件均满足，则诊断为“慢性支气管炎”；否则，判定为“不符合诊断标准”。

## 输出格式
返回明确诊断结果：“慢性支气管炎”或“不符合诊断标准”。
