# 肾性水肿的病理生理分类与临床特征

## 适用条件
- 患者已确诊肾小球疾病
- 存在水肿症状
- 排除心源性、肝源性或营养不良性水肿

## 执行步骤

1. **确认肾性水肿共同基础**：水钠潴留。

2. **判断是否为肾病性水肿**：
   - 主要机制：大量蛋白尿（>3.5 g/d）→ 血浆白蛋白降低 → 血浆胶体渗透压下降 → 液体渗入组织间隙；
   - 次要机制：有效血容量减少 → 激活RAAS系统和ADH分泌增加 → 肾小管重吸收水钠增加；
   - 组织间隙蛋白含量低；
   - 水肿起始于下肢（重力依赖性）。

3. **判断是否为肾炎性水肿**：
   - 主要机制：“球-管失衡”——GFR下降而肾小管重吸收功能基本正常 → 水钠潴留；
   - 滤过分数（GFR/肾血浆流量）降低；
   - 血容量常增加 → RAAS活性受抑制、ADH减少；
   - 高血压和毛细血管通透性增加使水肿持续加重；
   - 组织间隙蛋白含量高；
   - 水肿起始于眼睑或颜面部（疏松组织）。

4. **临床区分要点**：
   - 起始部位（眼睑/颜面 vs 下肢）；
   - 是否伴大量蛋白尿（>3.5 g/d）；
   - 血浆白蛋白水平（肾病性通常 <30 g/L）。

## 输出结果
根据上述分析，明确水肿类型，并据此指导利尿、限盐及病因治疗策略（如针对蛋白尿或GFR下降）。
