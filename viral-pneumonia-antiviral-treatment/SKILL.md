# 病毒性肺炎抗病毒药物治疗

## 触发条件
当患者确诊或高度疑似病毒性肺炎，并明确或高度怀疑特定病毒感染（如流感病毒、疱疹病毒、巨细胞病毒、呼吸道合胞病毒等）时，启动本技能。

## 执行步骤

1. **确认适用对象**：
   - 确诊或高度疑似病毒性肺炎患者
   - 免疫抑制宿主（如器官/骨髓移植受者）
   - 重症高风险人群

2. **验证前提条件**：
   - 发病时间在5天内
   - 无相关抗病毒药物禁忌症
   - 已排除单纯细菌性肺炎

3. **根据病毒类型选择抗病毒方案**：
   - **呼吸道合胞病毒（RSV）、腺病毒、副流感病毒、流感病毒** → 利巴韦林
   - **疱疹病毒、水痘-带状疱疹病毒（尤其免疫缺陷者）** → 阿昔洛韦
   - **巨细胞病毒（CMV）** → 更昔洛韦
   - **甲、乙型流感病毒** → 奥司他韦
   - **免疫缺陷者疱疹病毒/水痘病毒感染** → 阿糖腺苷

4. **执行用药原则**：
   - 抗生素不得用于预防继发细菌感染
   - 一旦明确合并细菌感染（如金黄色葡萄球菌），立即加用敏感抗生素
   - 糖皮质激素应谨慎评估后“酌情使用”，避免常规应用

5. **输出结果**：提供具体药物名称、剂量、给药方式、疗程及关键注意事项。

> 注意：本技能不适用于冠状病毒（如SARS-CoV-2）引起的肺炎，除非有明确循证支持的抗病毒指征。
