# 肾病综合征蛋白尿的病理生理机制分析

## 适用场景
- 患者被诊断为肾病综合征
- 出现持续性大量蛋白尿（>3.5g/天）
- 尿蛋白成分以白蛋白等中小分子蛋白为主

## 执行步骤
1. **确认屏障损伤**：评估是否存在肾小球滤过膜的以下任一屏障功能障碍：
   - **分子屏障受损**：无法有效阻挡大分子物质
   - **电荷屏障受损**：失去对带负电荷血浆蛋白（如白蛋白）的排斥能力
2. **识别加重因素**：检查是否存在以下促进蛋白尿加重的状态：
   - 高血压 → 增加肾小球内压力
   - 高蛋白饮食 → 引起肾小球高灌注、高滤过
   - 大量输注血浆蛋白 → 增加滤过负荷
3. **解释蛋白尿特征**：说明为何尿中主要含白蛋白等中小分子蛋白，而大分子蛋白（如纤维蛋白原、α₁和α₂巨球蛋白）未出现在尿液中
4. **输出临床指导建议**：基于机制，提出针对性干预措施，如降压治疗、限制膳食蛋白摄入等

## 注意事项
- 本技能仅适用于肾小球性蛋白尿，不适用于肾小管性或溢出性蛋白尿
- 蛋白尿成分分析是判断机制的关键依据
