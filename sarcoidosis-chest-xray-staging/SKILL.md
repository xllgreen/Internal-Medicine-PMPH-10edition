# 结节病胸部X线分期标准

## 适用条件
- 患者临床怀疑结节病
- 已完成标准后前位胸部X线检查
- 需结合组织活检与其他检查排除其他肉芽肿性疾病（如结核、淋巴瘤等）

## 分期步骤
1. **观察双侧肺门淋巴结是否肿大**
2. **评估是否存在肺部浸润影**（包括网状、结节状或片状阴影）
3. **判断是否存在终末期肺纤维化表现**（如蜂窝肺、肺大疱、肺气肿）
4. **根据以下标准确定分期**：
   - **0期**：无异常X线表现
   - **I期**：双侧肺门淋巴结肿大，无肺部浸润影
   - **II期**：双侧肺门淋巴结肿大，伴肺部浸润影
   - **III期**：肺部浸润影，无双侧肺门淋巴结肿大
   - **IV期**：肺纤维化表现（蜂窝肺、肺大疱、肺气肿）

## 临床意义
- 分期与自然缓解率密切相关：
  - I期：55%～90%
  - II期：40%～70%
  - III期：10%～20%
- 本分期**不能单独用于确诊结节病**，必须满足诊断三要素：
  1. 临床与影像表现相符
  2. 组织病理显示非干酪样坏死性肉芽肿
  3. 排除其他可能病因

## 输出
返回结节病X线分期（0–IV），用于指导预后判断和治疗决策。
