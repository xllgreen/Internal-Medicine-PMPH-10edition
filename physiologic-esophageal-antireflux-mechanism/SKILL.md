# 生理性食管抗反流防御机制

## 适用场景
- 分析胃食管反流事件后为何未出现食管炎
- 评估患者是否存在生理性防御机制受损
- 解释正常解剖与功能如何协同防止病理性反流

## 执行步骤

### 1. 确认触发条件
IF 发生胃食管反流（如吞咽后或一过性下食管括约肌松弛） THEN 启动以下三重防御机制。

### 2. 检查抗反流屏障完整性
- **下食管括约肌（LES）**：确认其长度为3～4 cm，能形成高压带阻止胃内容物反流。
- **辅助解剖结构**：评估膈肌脚、膈食管韧带及His角（食管与胃底间锐角）是否完整。

### 3. 评估食管清除能力
- 观察反流发生后是否通过1～2次自发或继发性食管蠕动将反流物排入胃内（即“食管廓清”）。
- 确认唾液分泌正常，可冲洗并中和残留反流物。

### 4. 验证食管黏膜屏障功能
- 确保复层鳞状上皮完整、黏膜下血供充足。
- 依赖唾液中的碳酸氢盐中和酸性反流物，维持黏膜完整性。

## 判断结果
若上述三重机制均完整，则可有效防止病理性胃食管反流；任一环节受损可能导致反流性食管炎或其他反流相关疾病。

> **注意**：本机制不适用于严重LES功能障碍或食管-胃连接解剖结构已破坏的患者。
