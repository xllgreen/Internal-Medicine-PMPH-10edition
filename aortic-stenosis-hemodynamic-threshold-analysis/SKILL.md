# 主动脉瓣狭窄血流动力学阈值分析

## 何时使用
- 患者已确诊为主动脉瓣狭窄
- 超声心动图或其他影像学检查显示主动脉瓣口面积 ≤1.0 cm²
- 出现或疑似出现劳力性呼吸困难、心绞痛、晕厥等症状
- 需评估左心衰竭、心肌缺血或脑供血不足的病理生理基础

## 执行步骤
1. **确认关键阈值**：验证主动脉瓣口面积是否 ≤1.0 cm²（正常为 3–4 cm²）。
2. **评估左心室压力负荷变化**：
   - 判断是否存在显著的左心室-主动脉收缩期压力阶差
   - 识别左心室壁向心性肥厚（游离壁和室间隔增厚）
3. **分析舒张功能障碍**：
   - 评估左心室顺应性下降与松弛速度减慢
   - 监测左心室舒张末压（LVEDP）是否进行性升高
4. **追踪压力传导至肺循环**：
   - 确认升高的 LVEDP 是否通过二尖瓣传导至左心房
   - 判断是否导致肺静脉压、肺毛细血管楔压及肺动脉压升高
   - 识别左心衰竭症状（如劳力性呼吸困难、端坐呼吸、急性肺水肿）
5. **评估心肌氧供需失衡**：
   - 分析因左室收缩压增高、射血时间延长导致的心肌耗氧量增加
   - 判断冠状动脉灌注是否减少（主动脉舒张压降低 + LVEDP 升高）
   - 关联心绞痛症状
6. **识别脑供血不足机制**：
   - 评估心排血量是否无法随外周血管扩张而代偿性增加
   - 排查心律失常因素
   - 关联晕厥或黑朦发作

## 输出提示
当瓣口面积 ≤1.0 cm² 时，提示患者已进入有症状阶段，需高度警惕三联征（呼吸困难、心绞痛、晕厥）及心力衰竭风险，建议尽快转诊至心脏专科进一步评估干预指征。
