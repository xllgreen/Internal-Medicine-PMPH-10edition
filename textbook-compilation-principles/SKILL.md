# 内科学教材编写原则

遵循以下四项核心原则编写或修订五年制本科《内科学》教材：

1. **保持五大编写特质**：确保教材具备科学性、系统性、完整性、权威性和实用性。保留并提炼内核知识，融入最新发展前沿，调整全书结构以契合临床实践脉络，满足教学、科研与临床多方位需求。

2. **强调“三基”培养**：聚焦基础理论、基本知识、基本技能的培养。精选近年临床实践经验，重点覆盖医学生必须掌握的常见病、多发病等实用内容，以内科各专科分支为纲构建整体架构。

3. **基于循证更新内容**：在前一版基础上深化章节内容，特别是常见病诊疗部分。依据国际和国内最新指南及循证医学证据进行更新；增补热门新章节，删减不常见疾病内容，调整交叉重复部分，并统一规范疾病名称与专业术语。

4. **融合数字教育资源**：嵌入多媒体资源、PBL案例库、在线学习材料等，支持学生预习、复习与自主学习，营造“身临其境”的临床实践体验，拓展知识探索路径。

> **注意**：本技能仅适用于五年制本科临床医学专业《内科学》教材编写，不适用于非临床医学专业、专科或高职医学教育场景。
