# 肢端肥大症的鉴别诊断

## 适用场景
- 患者存在肢端肥大样外貌但GH和IGF-1水平不典型
- 需排除非垂体源性GH/IGF-1升高或其他模拟疾病

## 执行步骤
1. **进行基础生化检测**：测定血清GH、IGF-1和GHRH水平，并完成口服葡萄糖抑制试验。
   - 若GH和IGF-1升高且不被葡萄糖抑制，同时GHRH增高 → 考虑非垂体GH瘤（如类癌、胰腺癌等）
   - 若GH和IGF-1升高但GHRH正常或降低 → 支持垂体GH瘤
2. **若GH和IGF-1正常**，排除真性肢端肥大症，考虑以下鉴别：
   - 单纯性凸颌症：仅面部骨骼突出，无系统表现
   - 皮肤骨膜肥厚症：青年男性多见，X线示增生性骨关节病，垂体MRI无肿瘤
   - 妊娠面容：仅见于妊娠期，产后数周消退
3. **对身材高大者**，区分胎儿期或产后过度生长病因（如Sotos综合征、马方综合征等）
4. **综合辅助检查**：所有疑似病例均需结合垂体MRI、X线骨关节检查及详细病史（围产史、家族史）判断

## 输出要求
明确指出最可能的鉴别诊断方向，并列出所需进一步检查（如GHRH检测、垂体MRI、X线等）。
