# 急性胰腺炎严重程度分级

## 适用条件
- 患者已确诊为急性胰腺炎（AP）
- 已完成以下评估：
  - 器官功能评估（改良Marshall评分）
  - 增强CT确认胰腺坏死范围
  - 感染迹象评估（如体温、白细胞、CT气泡征等）
- 发病时间 ≥48 小时（避免在早期未出现器官衰竭时进行最终定级）

## 执行步骤

### 1. 评估三个核心维度
- **器官衰竭（OF）**：任一器官（呼吸、循环、肾脏）改良Marshall评分 ≥2 分即判定存在OF
  - 呼吸：PaO₂/FiO₂ ≤300
  - 循环：收缩压 <90 mmHg 且补液不能纠正
  - 肾脏：肌酐 ≥170 μmol/L
- **胰腺坏死范围**：通过增强CT判断
  - <30% 或 >30%
- **感染状态**：符合以下任一提示感染性坏死
  - 体温 >38.5°C
  - 白细胞 >16×10⁹/L
  - CT见气泡征

### 2. 应用分级标准
- **轻症（MAP）**：无器官衰竭，且无胰腺坏死
- **中度重症（MSAP）**：器官衰竭在 ≤48 小时内恢复，和/或存在无菌性坏死
- **重症（SAP）**：器官衰竭持续 >48 小时，或存在感染性坏死
- **危重（CAP）**：器官衰竭持续 >48 小时 **且** 存在感染性坏死

### 3. 输出结果
输出最终分类（MAP / MSAP / SAP / CAP），用于：
- 确定监护级别（普通病房 vs ICU）
- 制定抗感染、营养支持及干预策略
- 预估死亡风险与住院时长

> **注意**：若存在肠功能衰竭（如腹腔间隔室综合征）或肝衰竭（肝性脑病≥2期伴凝血障碍及黄疸），应视为多器官衰竭，纳入SAP或CAP范畴。
