# 下丘脑-垂体-靶腺轴反馈调节机制

## 何时使用
- 当循环中靶腺激素（如甲状腺素、皮质醇、性激素）浓度偏离正常范围时
- 需要解释内分泌系统如何通过反馈维持激素稳态
- 分析月经周期中雌激素引发LH峰的正反馈机制
- 区分超短、短、长负反馈及正反馈的适用场景

## 核心操作流程

### 1. 负反馈调节（主要机制）
- **执行路径**：下丘脑释放激素 → 垂体分泌促激素 → 靶腺分泌靶腺激素
- **调节逻辑**：
  - 血中靶腺激素浓度过高 → 通过「长负反馈」抑制垂体和下丘脑激素分泌
  - 垂体促激素水平升高 → 通过「短反馈」抑制下丘脑释放激素
- **目标**：将靶腺激素浓度维持在生理最适范围内

### 2. 正反馈调节（特例，见于性腺轴）
- **触发条件**：卵泡期雌激素浓度达到阈值（接近排卵）
- **执行路径**：高浓度雌激素 → 兴奋下丘脑-垂体 → FSH/LH骤然升高（LH峰）→ 促进排卵

### 3. 反馈类型识别
- **超短反馈**：下丘脑激素抑制自身分泌
- **短反馈**：垂体激素抑制下丘脑激素分泌
- **长负反馈**：靶腺激素抑制垂体和下丘脑激素分泌
- **正反馈**：靶腺激素增强下丘脑-垂体激素分泌（仅限特定生理阶段）

> **注意**：本机制仅适用于经典内分泌轴，不适用于局部自分泌/旁分泌系统或完全外源性激素干预状态。
