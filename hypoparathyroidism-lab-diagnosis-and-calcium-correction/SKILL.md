# 甲状旁腺功能减退症的实验室诊断与血钙校正

## 何时使用
- 患者有手足搐搦、肌肉痉挛或神经肌肉兴奋性增高表现；
- Chvostek征或Trousseau征阳性；
- 已完成血清钙、磷、PTH及白蛋白检测；
- 排除急性肾损伤或严重未纠正的酸碱失衡。

## 执行步骤

### 1. 判断低钙血症
- 确认多次测定血清总钙 < 2.2 mmol/L；
- 若有症状，通常血清总钙 ≤ 1.88 mmol/L 或血清游离钙 ≤ 0.95 mmol/L。

### 2. 校正血清总钙
- 使用公式：**校正血清钙 (mmol/L) = 实测血清钙 + 0.02 × [40 - 血清白蛋白 (g/L)]**；
- 此步骤用于排除因低白蛋白导致的假性低钙。

### 3. 解读PTH水平
- 在血清总钙 ≤ 1.88 mmol/L 时，PTH应升高5~10倍；
- 若PTH未升高（即使在“正常范围”），提示甲状旁腺功能减退；
- 若PTH显著升高，需考虑假性甲状旁腺功能减退（PTH抵抗）。

### 4. 综合其他指标
- 血磷：通常升高，部分正常；
- 尿钙、尿磷：排出减少；
- 血碱性磷酸酶（ALP）：正常（反映骨转换减慢）。

## 输出结果
- 校正后的血清钙值；
- PTH与血钙关系的判断；
- 是否符合甲状旁腺功能减退症的实验室诊断标准。
