# 高血压急症控制性降压

## 触发条件
患者出现高血压急症（血压>180/120mmHg伴急性靶器官损害）。

## 执行步骤
1. **分阶段降压目标**：
   - 初始1小时内：平均动脉压（MAP）降幅 ≤ 治疗前水平的25%
   - 随后2～6小时内：将血压降至约160/100 mmHg
   - 后续24～48小时内逐步降至正常

2. **药物选择原则**：
   - 必须采用静脉给药
   - 优选起效快、作用时间短、停药后效应迅速消失的药物
   - 避免使用利血平和强力利尿剂（除非合并心衰）

3. **具体药物选择与剂量**：
   - 硝普钠：起始0.3～0.5 μg/(kg·min)，最大10 μg/(kg·min)
   - 硝酸甘油：起始5～10 μg/min，主要用于伴急性心衰或ACS
   - 尼卡地平：起始0.5 μg/(kg·min)，适用于合并急性脑血管病
   - 拉贝洛尔：起始20～80 mg IV，主要用于妊娠或肾功能不全

4. **后续管理**：
   - 病情允许时尽早过渡至口服降压药
   - 1～2周内逐步将血压降至长期目标值
   - 去除诱因（疼痛、焦虑、药物滥用等）
