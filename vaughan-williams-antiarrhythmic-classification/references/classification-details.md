# Vaughan Williams 分类法详细说明

## 经典四类分类法（1975）

### I 类：钠通道阻滞剂
- **共同特征**：抑制快钠内流，降低心肌细胞兴奋性和传导速度
- **Ia 类**
  - 药物：奎尼丁、普鲁卡因胺、丙吡胺
  - 电生理效应：中度抑制 V_max，延长 APD 和 ERP
  - 临床用途：房性和室性心律失常
- **Ib 类**
  - 药物：利多卡因、美西律、苯妥英
  - 电生理效应：轻度或无抑制 V_max，缩短 APD
  - 临床用途：主要用于室性心律失常，尤其急性心肌梗死相关
- **Ic 类**
  - 药物：普罗帕酮、氟卡尼
  - 电生理效应：显著抑制 V_max，轻微延长 APD
  - 临床用途：室上性心动过速；禁用于结构性心脏病患者（增加死亡率）

### II 类：β 受体阻滞剂
- 药物：美托洛尔、比索洛尔、普萘洛尔
- 机制：降低自律性、减慢传导、延长房室结不应期
- 特点：唯一被证实可改善长期预后的抗心律失常药（尤其在心梗后和心衰患者）

### III 类：钾通道阻滞剂
- 药物：胺碘酮、索他洛尔、多非利特
- 机制：延长动作电位时程（APD）和有效不应期（ERP）
- 注意：胺碘酮具多通道作用（I、II、III、IV 类效应），但主要归为 III 类

### IV 类：钙通道阻滞剂
- 药物：维拉帕米、地尔硫䓬
- 机制：抑制 L 型钙通道，主要影响窦房结和房室结
- 用途：终止阵发性室上速，控制房颤心室率

## 拓展八类分类法（现代版）

| 类别 | 靶点/机制 | 代表药物 | 说明 |
|------|----------|--------|------|
| 0 | 窦房结 I_f 电流 | 伊伐布雷定 | 控制窦性心率，非传统抗心律失常药 |
| I (含 Id) | 晚钠电流 | 雷诺嗪 | 用于心绞痛，具抗心律失常潜力 |
| II | 交感神经调节 | 阿托品（激活）、异丙肾上腺素（激活）、β 阻滞剂（抑制） | 扩展原 II 类范畴 |
| III | 钾通道开放/阻滞 | 尼可地尔（开放剂）、胺碘酮（阻滞剂） | 更全面描述钾通道调控 |
| IV | 钙处理调节 | 洋地黄类（Na⁺/K⁺-ATPase 抑制剂 → 间接影响钙） | 将洋地黄纳入分类体系 |
| V | 机械力门控通道 | 实验阶段药物 | 应对牵张诱发的心律失常 |
| VI | 缝隙连接（Connexin） | 实验性化合物 | 影响细胞间电耦联 |
| VII | 上游调控 | 抗炎药、抗氧化剂 | 针对心律失常基质（如心房纤维化） |

## 关键变量定义
- **V_max**：动作电位 0 相最大上升速率，反映快钠通道功能状态
- **动作电位时程（APD）**：从去极化开始到复极完成的时间，决定有效不应期长度

## 使用限制
- 不适用于代谢性（如低钾、酸中毒）、电解质紊乱或机械性原因引起的心律失常
- 新分类尚未广泛用于临床指南，主要用于科研和机制教学
