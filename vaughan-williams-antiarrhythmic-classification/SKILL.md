# Vaughan Williams 抗心律失常药物分类

## 何时使用
- 需要对抗心律失常药物按电生理机制进行归类
- 制定或评估基于机制的心律失常药物治疗方案
- 解读新型多靶点抗心律失常药物的作用维度
- 区分经典与现代分类体系的适用场景

## 执行步骤

### 1. 应用经典 Vaughan Williams 四类分类法（1975年）
- **I 类**：钠通道阻滞剂
  - Ia：减慢 V_max，延长动作电位时程（如奎尼丁）
  - Ib：不减慢 V_max，缩短动作电位时程（如利多卡因）
  - Ic：显著减慢 V_max 和传导（如普罗帕酮）
- **II 类**：β 受体拮抗剂（如美托洛尔）
- **III 类**：钾通道阻滞剂，延长复极（如胺碘酮）
- **IV 类**：钙通道阻滞剂（如维拉帕米）

### 2. 如需涵盖非传统药物或多机制药物，采用拓展八类分类法
- **0 类**：起搏电流（I_f）抑制剂
- **I 类**：含新增 Id 亚类（晚钠电流抑制剂）
- **II 类**：扩展为交感神经调节剂（含阿托品、异丙肾上腺素）
- **III–VII 类**：分别对应钾通道、钙处理、机械力门控、缝隙连接及上游调控通路

### 3. 判断适用性
- 若心律失常由代谢等非电生理机制主导，本分类不适用
- 对于新型多靶点药物，可同时归属多个类别，需结合临床证据综合判断

> 提示：经典分类适用于大多数临床场景；拓展分类用于研究或复杂病例机制解析。
