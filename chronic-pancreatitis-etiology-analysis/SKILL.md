# 慢性胰腺炎病因分类与发病机制分析

## 触发条件
当满足以下条件时启动本技能：
- 患者有反复或持续性上腹痛、脂肪泻、消瘦或新发糖尿病等临床表现；
- 影像学或功能检查提示胰腺结构/功能异常（如钙化、胰管多发狭窄与扩张、假性囊肿、胰腺萎缩）；
- 已排除胰腺肿瘤及其他非胰腺性消化疾病。

## 执行步骤

### 1. 确认慢性胰腺炎诊断基础
- **临床表现**：反复或持续性上腹痛、脂肪泻、体重下降、糖尿病。
- **影像证据**：胰腺钙化、胰管狭窄与扩张交替、假性囊肿、胰腺实质萎缩。

### 2. 系统评估病因类别
按以下常见病因逐一排查，结合病史、实验室及影像结果判断最可能类型：

- **胆胰管疾病**（中国高发）：
  - 查找胆总管下段或胰胆管交界处狭窄/结石证据；
  - 判断是否因胰液流出受阻导致反复急性胰腺炎（RAP）进而进展为CP。

- **酒精相关**：
  - 评估长期饮酒史（注意：<10%酗酒者最终发展为CP）；
  - 考察是否合并吸烟（协同促进钙化）；
  - 判断是否存在酒精性RAP病史。

- **病毒感染**（如B组柯萨奇病毒）：
  - 回顾急性胰腺炎发作前是否有病毒感染史；
  - 若合并饮酒或组织修复障碍，考虑慢性化风险。

- **遗传性胰腺炎**：
  - 关注幼年发病、家族史、反复RAP；
  - 若高度怀疑，建议基因检测（PRSS1、SPINK1、CFTR等）；
  - 注意其高胰腺癌风险，需长期随访。

- **自身免疫性胰腺炎（AIP）**：
  - **I型**：检测血清IgG4（>1350 mg/L支持诊断），评估是否有多器官受累；
  - **II型**：关注导管中心性炎症及中性粒细胞微脓肿（通常需病理确诊）。

- **高钙血症**：
  - 检测血钙水平；
  - 高钙可激活胰酶并促进胰管结石形成，增加RAP风险。

- **营养因素**：
  - 评估饮食结构（如高饱和脂肪+低蛋白），尤其在热带地区患者中考虑“热带胰腺炎”。

- **特发性**：
  - 在排除上述所有病因后归为此类；
  - 区分早发型（20–30岁）与迟发型（60–70岁）；
  - 可能与轻度基因变异、解剖异常或吸烟相关。

### 3. 整合发病机制共性
- 多数病因需“急性胰腺炎前哨事件”启动慢性化进程；
- 遗传性或自身免疫性可隐匿起病，无需明确前哨事件；
- 共同通路为：反复损伤 → 持续炎症 → 纤维化 → 不可逆胰腺功能损害。

## 输出要求
输出具体病因类别、支持证据（临床、影像、实验室）、潜在机制，并据此提出针对性干预建议（如戒酒、胆道手术、免疫抑制治疗、基因咨询等）。
