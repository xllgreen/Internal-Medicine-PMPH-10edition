# 高原病发病机制分析

## 适用对象
- 从平原快速进入高原地区的人群
- 在海拔≥2400米环境中暴露且未充分适应低氧者

> **注意**：不适用于已完全适应高原者或长期世居高原人群（除非出现慢性高原病）。

## 触发条件
当满足以下条件时启动分析：
- 个体从平原快速进入海拔≥2400米地区
- 出现缺氧相关症状（如头痛、失眠、呼吸困难、意识障碍等）

## 执行步骤

### 1. 评估海拔与动脉血氧饱和度关系
根据当前海拔估算动脉血氧饱和度（SpO₂）下降程度：
- 海拔2400～2700米：SpO₂轻度降低
- 海拔3500～4000米：SpO₂ < 90%
- 海拔5000米：SpO₂ ≈ 75%
- 海拔5500米以上：严重低氧血症伴低碳酸血症
- 海拔7000米：SpO₂ ≈ 60%
- 海拔8000米：吸入气氧分压仅56mmHg

### 2. 按系统分析发病机制

#### 神经系统
- 大脑皮质对缺氧最敏感 → 脑血管扩张 → 颅内压升高 → 头痛、失眠
- 严重缺氧时ATP减少 → 钠泵功能障碍 → 细胞内水肿 → 高原脑水肿

#### 呼吸系统
- 低氧刺激颈动脉体 → 反射性通气增强 → 过度换气 → CO₂排出增多 → 呼吸性碱中毒
- 肺小动脉痉挛 → 肺动脉高压 → 毛细血管通透性增加 → 血浆渗出 → 高原肺水肿
- 同时伴随：肺泡表面活性物质减少、内皮素水平升高（可达正常2倍）、花生四烯酸/TXA₂释放加重肺损伤

#### 心血管系统
- 早期代偿：心率加快、心排血量增加
- 血液重分布：皮肤和肾血管收缩，心脑血管扩张
- 长期缺氧：肺动脉高压 → 右心室肥大（高原性心脏病）
- 红细胞增多 → 血液黏滞度升高 → 心脏负荷增加
- 儿茶酚胺、ADH、RAAS激活 → 血压升高

#### 造血系统
- 低氧刺激促红细胞生成素（EPO）分泌 → 红细胞和血红蛋白增多
- 2,3-DPG升高 → 氧解离曲线右移 → 组织释氧能力增强

### 3. 判断高原病类型与病理特征
- **急性高原反应**：无特异性病理改变，主要为功能性症状
- **高原肺水肿**：肺充血、水肿、透明膜形成、毛细血管通透性增高、散在微血栓
- **高原脑水肿**：脑皮质充血、细胞水肿、点状出血、毛细血管损害
- **慢性高原病**：右心室肥大、肺动脉硬化、心肌浊肿或坏死

### 4. 输出建议
基于上述分析，结合症状严重程度，给出以下建议之一：
- 继续观察并吸氧
- 使用乙酰唑胺或地塞米松等药物
- 立即下撤至低海拔地区
