# 医学教材分级负责编写制度

## 适用场景
- 组织大型医学教材（如《内科学》第10版）编写项目
- 涉及多位院士/教授级专家协作的学术出版工程
- 需要按学科篇章进行清晰责任划分的编撰任务

> **不适用于**：单作者著作或无明确学科分工的出版项目。

## 执行步骤

1. **设立主编团队**
   - 任命3位权威院士担任主编，负责整体学术方向把控与质量监督。

2. **配置副主编协同机制**
   - 安排4位资深教授担任副主编，协助主编协调各篇章内容，并参与关键章节审定。

3. **实施分篇责任制**
   - 按学科篇章指定负责人，每位负责人对其所辖篇章的学术内容负主要责任。典型分工包括：
     - 绪论与循环系统疾病 → 主编之一
     - 呼吸系统疾病 → 主编之一
     - 消化、泌尿、血液、内分泌代谢、风湿免疫、危重症等篇章 → 对应领域权威专家

4. **任命学术秘书**
   - 指定专职学术秘书（如主任医师），负责稿件整理、格式统一、进度跟踪与成稿汇编。

5. **建立质量保障机制**
   - 在前一版基础上“精雕细琢”，并通过评审委员会审核，确保内容权威性、一致性与可追溯性。

## 关键角色定义
- **分篇负责人**：对特定学科篇章内容负主要学术责任的专家。
- **学术秘书**：负责编务协调、稿件整合与流程管理的专职人员。

## 输出成果
建立高效、权威、可追溯的医学教材编写组织体系，支撑高质量学术出版。
