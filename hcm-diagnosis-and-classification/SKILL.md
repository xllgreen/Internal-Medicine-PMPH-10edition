# 肥厚型心肌病（HCM）诊断与分型

## 触发条件
当超声心动图或心脏磁共振（CMR）显示左心室壁增厚，且已排除高血压、主动脉瓣狭窄等负荷性原因时，启动本诊断流程。

## 执行步骤

### 1. 确认前提条件
- 已完成超声心动图（TTE）或心脏磁共振（CMR）检查
- 已排除运动员心肌肥厚、淀粉样变等系统性疾病
- 不可仅凭心电图做出诊断

### 2. 应用年龄与家族史分层诊断标准
- **成人**：
  - 无家族史/基因阴性：左室任意部位舒张末期室壁厚度 ≥15 mm → 可诊断HCM
  - 有明确家族史或基因阳性：室壁厚度 ≥13 mm → 可诊断HCM
- **儿童**：
  - 无家族史且无症状：左室厚度 z 值 >2.5 → 可诊断
  - 有家族史或基因阳性：z 值 >2 → 可诊断
- **一级亲属筛查**：
  - 成年亲属：室壁厚度 >13 mm → 可诊断
  - 儿童亲属：z 值 <2 但存在形态或心电异常 → 标记为可疑，需进一步评估，不可单独确诊

### 3. 进行分型诊断
- **按血流动力学**：
  - 梗阻型：静息或激发状态下左室流出道（LVOTO）峰压差 ≥30 mmHg
  - 非梗阻型：压差 <30 mmHg
- **按肥厚部位**：
  - 室间隔基底部肥厚（最常见）
  - 心尖部肥厚（常伴广泛对称T波倒置）
  - 左室中部肥厚（乳头肌水平）
  - 右心室肥厚（罕见）

### 4. 安排必要辅助检查
- **超声心动图**：首选检查；若静息无梗阻，需行瓦氏动作或运动负荷激发试验
- **心脏磁共振（CMR）**：用于评估特殊部位肥厚及心肌纤维化；若延迟强化（LGE）≥15%左室质量，提示猝死高风险
- **基因检测**：一线检测8个肌小节基因（如MYH7、MYBPC3）；若阴性，可考虑全外显子测序
- **心电图**：识别特征性改变（如深而窄Q波、固定ST-T改变、广泛T波倒置），但仅作辅助

### 5. 输出结果
- 明确是否符合HCM诊断标准
- 给出具体分型（梗阻/非梗阻 + 肥厚部位）
- 建议是否需进一步基因检测或高级影像学评估
