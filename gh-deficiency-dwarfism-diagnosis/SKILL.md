# 生长激素缺乏性矮小症诊断

## 适用对象
- 疑似生长激素缺乏性矮小症的儿童患者

## 前提条件
- 存在身材矮小和/或生长速度缓慢的临床表现
- 已排除其他明显病因

## 使用限制
- 不适用于非内分泌性矮小症（如家族性矮小、体质性发育延迟）的初步筛查
- 随机血GH浓度不可用于诊断

## 执行步骤

1. **评估临床特征**：确认患儿身高 < 同年龄同性别正常人均值 -2SD，且生长速度缓慢；可伴性发育障碍。

2. **进行骨龄检查**：若骨龄较实际年龄落后 ≥2 年，支持生长激素缺乏诊断。

3. **测定血IGF-1和IGFBP-3水平**：检测血清胰岛素样生长因子-1（IGF-1）及其主要结合蛋白IGFBP-3（占92%），两者降低提示GH分泌不足。

4. **实施GH激发试验**：必须进行至少两种激发试验（如胰岛素低血糖、左旋多巴、精氨酸、可乐定等），判读标准如下：
   - GH峰值 < 5 μg/L → 完全性GHD
   - GH峰值 5–10 μg/L → 部分性GHD

5. **排除其他疾病**：鉴别并排除以下可能导致生长迟缓的疾病：
   - 先天性甲状腺功能减退（呆小病）
   - 染色体异常（如Turner综合征）
   - 慢性肝肾疾病
   - 慢性感染（结核、血吸虫病、钩虫病等）

6. **病因学检查**：确诊GHD后，进一步行视野检查、蝶鞍CT或MRI排查颅内肿瘤；必要时进行染色体核型分析或基因检测；若无明确病因，则诊断为特发性GHD。

## 输出结果
- 明确是否符合生长激素缺乏性矮小症诊断标准
- 区分完全性或部分性GHD
- 确认是否已完成其他鉴别诊断的排除
