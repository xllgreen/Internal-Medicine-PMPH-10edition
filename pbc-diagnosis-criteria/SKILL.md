# 原发性胆汁性胆管炎（PBC）诊断标准

## 适用场景
- 患者出现皮肤瘙痒、乏力、黄疸、肝大等表现
- 无肝外胆管梗阻证据
- 已完成相关实验室或组织学检查（如AMA、ALP、IgM或肝活检）

## 执行步骤
1. **确认胆汁淤积的生化证据**：检查血清碱性磷酸酶（ALP）和γ-谷氨酰转移酶（γ-GT）是否明显升高。
2. **检测特异性自身抗体**：确认以下任一抗体阳性：
   - 抗线粒体抗体（AMA），滴度 >1:40 具有诊断意义
   - AMA-M₂亚型（阳性率90%～95%）
   - 抗核骨架蛋白抗体（GP210）
   - 抗肝脂蛋白抗体（SP100）
3. **评估肝组织学改变**：确认是否存在PBC典型病理分期（Ⅰ至Ⅳ期）。
4. **应用诊断规则**：满足上述三项中的任意两项即可确诊PBC。

## 注意事项
- 约95%以上PBC患者AMA阳性，部分在症状出现前6～10年即呈阳性。
- AMA阴性时，GP210或SP100可作为重要替代标志。
- 不适用于继发性胆汁性肝硬化或其他明确病因的胆道梗阻患者。

## 输出结果
- 若满足两项标准，输出“确诊PBC”
- 否则输出“需进一步排除其他疾病”
