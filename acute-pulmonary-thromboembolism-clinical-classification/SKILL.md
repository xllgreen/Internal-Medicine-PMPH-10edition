# 急性肺血栓栓塞症（PTE）临床分型

## 使用时机
- 患者已通过CTPA、V/Q显像等检查确诊为急性PTE
- 已完成血流动力学状态、右心功能影像学及心脏生物标志物（如肌钙蛋白、BNP/NT-proBNP）评估
- 需尽快完成分型以制定初始治疗策略

## 执行步骤

### 第一步：评估血流动力学稳定性
判断是否存在以下任一情况（满足即为**血流动力学不稳定**，归类为**高危PTE**）：
- 心搏骤停（需心肺复苏）
- 梗阻性休克：收缩压 < 90 mmHg，或需升压药维持 ≥90 mmHg 且伴外周器官低灌注
- 持续低血压：收缩压 < 90 mmHg 或较基线下降 ≥40 mmHg 持续 >15分钟，且排除心律失常、低血容量或感染中毒症

### 第二步：若血流动力学稳定，评估右心功能不全（RVD）与生物标志物
1. **检查RVD影像学证据**（满足任一即可）：
   - 超声心动图：
     - 右心室/左心室舒张末期内径比 > 1.0
     - 右心室游离壁运动幅度减低
     - 三尖瓣反流速度增快
     - 三尖瓣环收缩期位移 < 17 mm
   - CTPA（四腔心层面）：右心室/左心室舒张末期内径比 > 1.0
2. **检测心脏生物标志物是否升高**：
   - 肌钙蛋白（cTNI/cTNT）高于正常上限
   - BNP 或 NT-proBNP 升高

### 第三步：确定最终分型
- **高危PTE**：血流动力学不稳定 → 提示早期死亡高风险
- **中危PTE**：血流动力学稳定 +（RVD 或 生物标志物升高）
  - **中高危**：RVD 和 生物标志物升高 同时存在
  - **中低危**：仅RVD 或 仅生物标志物升高
- **低危PTE**：血流动力学稳定 + 无RVD + 无生物标志物升高

## 输出结果
返回分型结果（高危PTE / 中高危PTE / 中低危PTE / 低危PTE），并据此建议下一步治疗方向（如抗凝、溶栓或介入评估）。
