# 急性呼吸衰竭诊断分类

## 适用场景
- 患者有低氧/高碳酸血症临床表现（如呼吸困难、发绀、意识改变）
- 已完成动脉血气分析和胸部影像学检查
- 需明确呼吸衰竭类型以指导后续诊疗

## 执行步骤

### 1. 确认血气异常
- **低氧血症**：PaO₂ < 60 mmHg
- **高碳酸血症**：PaCO₂ > 50 mmHg

### 2. 判断呼吸衰竭类型
- **Ⅰ型（低氧性）**：PaO₂ < 60 mmHg，PaCO₂ 正常或降低
- **Ⅱ型（高碳酸血症性）**：PaO₂ < 60 mmHg 且 PaCO₂ > 50 mmHg

### 3. 病因初步定位
- **Ⅰ型呼吸衰竭**：
  - 胸部影像学正常 → 考虑肺血管疾病（如肺栓塞）
  - 影像学异常 → 区分肺源性（如肺炎、ARDS）与心源性肺水肿（结合病史、心电图、心脏超声等）
- **Ⅱ型呼吸衰竭**：
  - 主要机制为通气功能障碍导致 CO₂ 排出减少
  - 单纯 CO₂ 生成增加所致者罕见

### 4. 注意事项
- 血气结果受年龄、海拔、氧疗影响，必须结合临床综合判断
- 排除非呼吸系统原发疾病导致的类似表现

## 输出结果
明确呼吸衰竭类型（Ⅰ型或Ⅱ型）及可能的病因方向，指导后续检查与治疗策略。
