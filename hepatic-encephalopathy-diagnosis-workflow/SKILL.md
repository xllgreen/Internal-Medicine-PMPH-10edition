# 肝性脑病诊断依据与排除流程

## 触发条件
当满足以下任一条件时启动本流程：
- 患者出现性格改变、行为异常、意识模糊、扑翼样震颤或昏迷等精神神经症状；
- 患者有肝硬化、病毒性肝炎、长期饮酒史，或曾接受门体分流术（如TIPS）。

## 执行步骤
1. **确认四大诊断依据是否全部满足**：
   - **① 基础肝病与诱因**：存在严重肝病（如肝硬化）和/或广泛门体侧支循环，并有明确HE诱因（如上消化道出血、感染、低钾碱中毒、便秘、镇静药使用等）；
   - **② 典型临床表现**：出现性格改变、行为异常、意识模糊、扑翼样震颤、昏迷等；
   - **③ 实验室异常**：肝功能指标（ALT、AST、TBil、Alb、PT）明显异常和/或血氨升高；
   - **④ 影像学排除**：头部CT或MRI已排除脑血管意外、颅内肿瘤、脑炎等器质性脑病。

2. **对所有不明原因精神症状患者，必须执行以下排查操作**：
   - 详细询问肝病相关病史（包括饮酒、病毒性肝炎、药物使用等）；
   - 检测肝功能全套（ALT、AST、总胆红素、白蛋白、凝血酶原时间等）；
   - 测定动脉或静脉血氨水平；
   - 必要时安排头颅CT或MRI检查。

3. **进行鉴别诊断，排除以下疾病**：
   - 低血糖、糖尿病酮症酸中毒、尿毒症；
   - 脑血管意外、脑部感染（如脑炎、脑膜炎）；
   - 镇静药或酒精过量；
   - 原发性精神疾病（如精神分裂症、躁郁症）。

4. **输出判断**：
   - 若四项依据均满足且已排除其他病因，发出“高度提示肝性脑病”警报；
   - 若仅部分满足，建议进一步排查并密切监测。

> **注意**：部分肝性脑病患者肝病史不明确，以精神症状为首发表现，极易误诊为原发性精神疾病，务必保持高度警惕。
