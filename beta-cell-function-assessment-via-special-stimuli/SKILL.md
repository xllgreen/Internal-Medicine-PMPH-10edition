# 特殊刺激下β细胞功能评估

## 适用对象
- 疑似1型糖尿病（T1DM）患者
- 需全面评估β细胞储备功能者
- 混合型糖尿病鉴别诊断

## 前提条件
- 空腹状态
- 已建立静脉通路
- 备有急救措施（如葡萄糖溶液）及专业医疗监护

## 执行步骤
根据临床需求选择以下任一刺激试验：

### 1. 静脉葡萄糖-胰岛素释放试验
- 快速静脉推注葡萄糖（0.3g/kg）。
- 在0、2、4、6、8、10分钟采血测胰岛素或C肽。
- 评估葡萄糖介导的第一时相胰岛素分泌（0～10分钟快速释放）。

### 2. 高糖钳夹试验（Hyperglycemic Clamp）
- 持续静脉输注葡萄糖，将血糖维持在高水平（如10 mmol/L）。
- 监测胰岛素/C肽分泌：第一时相（0～10分钟）和第二时相（持续分泌）。
- 为评估β细胞葡萄糖敏感性的金标准之一。

### 3. 胰高血糖素-C肽刺激试验
- 静脉注射胰高血糖素1mg。
- 在0、6分钟采血测C肽。
- 正常反应：6分钟C肽较基础值升高>2倍。

### 4. 精氨酸-C肽刺激试验
- 静脉快速输注精氨酸（5g，30秒内）。
- 在0、2、4、6分钟采血测C肽。
- 评估氨基酸刺激下的胰岛素分泌能力。

## 结果解读
- T1DM：各类刺激下C肽/胰岛素分泌显著低下或缺失。
- T2DM：通常保留部分分泌能力，尤其对非葡萄糖刺激。
- 第一时相缺失是早期β细胞功能障碍的重要标志。

## 注意事项
- 不适用于常规筛查或急性并发症期患者。
- 所有操作必须在医院内由专业人员执行，并备有低血糖抢救预案。
- 优先选择C肽作为检测指标以避免外源性胰岛素干扰。
