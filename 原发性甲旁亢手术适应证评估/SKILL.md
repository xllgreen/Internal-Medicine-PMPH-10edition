# 原发性甲旁亢手术适应证评估

## 使用前提
确保满足以下条件：
- 患者已确诊为原发性甲旁亢（血清钙升高 + PTH升高）；
- 已完成甲状旁腺定位检查（如颈部超声、MIBI扫描等）；
- 无严重手术禁忌证（如严重心肺功能不全等）。

## 执行流程

### 1. 区分有症状与无症状患者
- **有症状患者**（如骨痛、肾结石、高钙相关神经精神或消化系统症状）：**直接推荐手术**。
- **无症状患者**：继续评估以下任一手术指征。

### 2. 无症状患者的手术指征（满足任一即推荐手术）
- **高钙血症**：血清钙高于正常上限 ≥ 0.25 mmol/L（≈1 mg/dL）；
- **肾脏损害**：肌酐清除率 < 60 mL/min；
- **骨骼受累**：任一部位骨密度T值 < -2.5，和/或已发生脆性骨折；
- **年龄 < 50岁**：因长期并发症风险高；
- **无法接受常规随访**：如居住偏远、依从性差等。

### 3. 手术方式选择
- 定位明确的单发腺瘤：行腺瘤切除术；
- 多发腺瘤或增生：需探查全部四枚甲状旁腺，切除病变腺体。

### 4. 术后管理要点
- **监测血钙**：警惕术后低钙血症（常见于“骨骼饥饿综合征”或剩余甲状旁腺功能暂时抑制）；
- **急性低钙处理**：出现手足搐搦时，静脉缓慢推注10%葡萄糖酸钙10～20 mL；
- **长期替代治疗**：补充钙剂和维生素D，直至PTH水平恢复。

### 5. 并发症预警
- **喉返神经损伤**：表现为声音嘶哑；
- **永久性甲状旁腺功能减退**：需终身钙剂和活性维生素D替代治疗。

> ⚠️ 注意：本技能**不适用于**继发性或三发性甲旁亢，也不适用于无法耐受手术的高龄或重症患者（除非发生高钙危象）。
