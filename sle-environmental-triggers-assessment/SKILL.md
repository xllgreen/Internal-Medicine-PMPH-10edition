# SLE环境诱发因素评估

## 适用对象
- 具有SLE遗传易感性的个体
- 已确诊SLE但病情稳定的患者
- 出现疑似SLE症状且有相关环境暴露史者

## 使用时机
- 用户提及紫外线照射、特定药物使用或病毒感染后出现皮疹、关节痛、疲劳等症状
- 用户咨询如何降低SLE发作风险
- 需要解释环境因素如何在易感个体中启动自身免疫级联反应

## 执行步骤
1. **确认易感背景**：核实个体是否存在SLE遗传易感性或既往免疫异常。
2. **识别环境暴露**：明确近期是否接触以下任一诱因：
   - 紫外线（如日晒、紫外线灯）
   - 特定药物（如肼屈嗪）
   - 病毒感染（如巨细胞病毒、新冠病毒）
3. **评估致病机制**：
   - 紫外线 → 皮肤上皮细胞凋亡 → 暴露自身抗原（如苏木紫小体）
   - 药物 → DNA低甲基化 → 异常基因表达 → 产生自身抗体
   - 病毒 → 分子模拟 → 交叉反应性B细胞活化
4. **判断免疫激活风险**：在免疫耐受减弱的背景下，上述暴露可能促使B细胞识别自身抗原，启动T细胞依赖的自身免疫反应。
5. **生成预防建议**：提示避免紫外线暴晒、慎用已知致病药物、积极预防病毒感染，以降低SLE发作风险。

> **注意**：本技能不适用于无免疫异常基础的健康人群，且并非所有SLE患者均对环境因素敏感。
