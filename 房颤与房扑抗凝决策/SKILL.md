# 房颤与房扑抗凝决策

## 触发条件
患者确诊为非瓣膜性心房颤动或持续性心房扑动，并已完成CHA₂DS₂-VASc评分和HAS-BLED评分。

## 执行步骤
1. **计算CHA₂DS₂-VASc评分**：
   - 心力衰竭/左室功能障碍（C）：1分
   - 高血压（H）：1分
   - 年龄≥75岁（A）：2分
   - 糖尿病（D）：1分
   - 脑卒中/TIA/血栓栓塞史（S）：2分
   - 血管疾病（V）：1分
   - 年龄65~74岁（A）：1分
   - 女性（Sc）：1分

2. **根据评分决定是否抗凝**：
   - 男性≥2分 或 女性≥3分：应进行抗凝治疗
   - 男性=1分 或 女性=2分：权衡获益与风险后可考虑抗凝
   - 男性=0分 或 女性=1分：通常无需抗凝

3. **评估出血风险**：
   - 进行HAS-BLED评分（≥3分为高出血风险）
   - 高出血风险不是抗凝禁忌证，应积极纠正可逆出血因素

4. **选择抗凝药物**：
   - 非瓣膜性房颤首选新型口服抗凝药（NOAC）
   - 中重度二尖瓣狭窄或机械瓣置换术后患者必须使用华法林（目标INR 2.0～3.0）

5. **特殊情形处理**：
   - 房颤持续>48小时者，复律前需抗凝3周，或经食管超声排除左心房血栓后复律，复律后继续抗凝4周
