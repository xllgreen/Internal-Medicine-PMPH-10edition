# 低血糖分层治疗与预防

## 触发条件
当确认患者发生低血糖（血糖 ≤ 3.9 mmol/L 或出现典型症状）时，立即启动本技能。

## 执行步骤

### 1. 评估严重程度
- **轻度至中度**：意识清楚，可自主吞咽 → 进入口服治疗流程。
- **重度或昏迷**：意识障碍、无法吞咽 → 立即进入静脉急救流程。

### 2. 轻中度低血糖处理
- 立即口服 **15～20g 快速吸收碳水化合物**（如糖水、含糖饮料、糖果等）。
- **15分钟后复测血糖**；若仍 ≤ 3.9 mmol/L，重复给糖一次。
- **特别注意**：若患者正在使用 α-葡萄糖苷酶抑制剂（如阿卡波糖、伏格列波糖），**必须使用纯葡萄糖**（如葡萄糖片或葡萄糖溶液），禁用蔗糖或多糖类食物。

### 3. 重度低血糖或昏迷处理
- **立即静脉注射 50% 葡萄糖溶液 60～100ml**（无需等待血糖结果）。
- 随后持续静脉滴注 **5%～10% 葡萄糖溶液** 维持血糖稳定。
- 若效果不佳或无法建立静脉通路，可考虑：
  - 胰高血糖素 0.5～1mg 肌注或静注；
  - 氢化可的松 100mg 静脉注射。
- **严禁经口喂食**，防止误吸和窒息。

### 4. 病因治疗与长期预防
- **药物相关性**：停用可疑降糖药（如磺脲类、胰岛素）。
- **器质性疾病**（如胰岛素瘤）：转诊专科评估手术指征。
- **反复发作患者**：加强血糖监测、调整个体化降糖目标、开展低血糖识别教育。
- **高危人群**（如强化治疗糖尿病患者）：避免过度降糖，设定安全血糖下限。

> **关键约束**：神志不清者禁止口服任何食物或液体；α-葡萄糖苷酶抑制剂使用者仅可用纯葡萄糖。
