# 慢阻肺病稳定期综合评估与分组治疗策略

## 适用对象
- 已确诊的稳定期慢阻肺病患者（无当前急性加重）

## 前提条件
在执行本技能前，必须确认以下数据已获取：
- 肺功能检查结果（FEV₁%pred）
- 症状评估：mMRC分级（0–4级）或CAT评分（0–40分）
- 上一年急性加重次数及是否住院
- （如考虑使用ICS）外周血嗜酸性粒细胞计数（EOS，单位：个/μl）

## 执行步骤

1. **收集三项关键数据**：
   - **急性加重史**：统计上一年中度及以上急性加重次数（中度：需抗生素或全身激素；重度：需住院或急诊）
   - **症状水平**：使用 mMRC ≥2 或 CAT ≥10 判定为“症状多”
   - **肺功能**：FEV₁%pred 用于确认气流受限存在，但不直接决定分组

2. **按规则分组并确定首选治疗**：
   - **A组（低风险，症状少）**：
     • 急性加重：0或1次中度（无住院）
     • 症状：mMRC 0–1 或 CAT <10
     • 首选治疗：一种支气管扩张剂（短效或长效）
   
   - **B组（低风险，症状多）**：
     • 急性加重：0或1次中度（无住院）
     • 症状：mMRC ≥2 或 CAT ≥10
     • 首选治疗：LABA + LAMA
   
   - **E组（高风险）**：
     • 急性加重：≥2次中度 或 ≥1次住院
     • 症状评分不纳入分组依据
     • 首选治疗：
        - LABA + LAMA
        - 若血EOS ≥300个/μl，可加用ICS

3. **注意事项**：
   - 原GOLD C/D组已合并为E组，强调急性加重风险优先于症状严重程度
   - 必须同步评估常见合并症（如心血管疾病、骨质疏松、焦虑抑郁、肺癌、糖尿病）并纳入整体管理
   - 血EOS水平用于预测ICS对预防急性加重的疗效

## 输出要求
返回患者所属分组（A / B / E）及对应的首选药物治疗方案。
