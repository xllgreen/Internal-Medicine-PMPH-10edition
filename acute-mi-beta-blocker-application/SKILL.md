# 急性心肌梗死β受体拮抗剂应用

## 触发条件
- 诊断为STEMI/AMI，发病24小时内；
- **无以下禁忌**：心力衰竭、低心排血量、心源性休克高危（如年龄>70岁、收缩压<120mmHg、心率异常）、其他β阻滞剂禁忌。

## 使用流程
1. 首选心脏选择性β受体拮抗剂（如美托洛尔）。
2. 起始剂量为目标剂量的1/4，逐步递增至目标心率（55～60次/分）。
3. 若存在剧烈胸痛或显著高血压且其他治疗无效，可静脉使用美托洛尔：
   - 每次5mg静脉推注；
   - 每次推注后观察2～5分钟；
   - 若心率<60次/分或收缩压<100mmHg，立即停药；
   - 静脉总量不超过15mg；
   - 末次静脉给药15分钟后转为口服维持。
4. 对相对禁忌但需控制心率者，可选用艾司洛尔（极短效）：
   - 输注速率 50～200μg/(kg·min)，根据反应调整。

## 禁忌情形
- 心力衰竭；
- 低心排血量状态；
- 心源性休克高危因素（年龄>70岁、收缩压<120mmHg、窦速>110次/分或心率<60次/分、发病时间延长）；
- 其他β受体拮抗剂使用禁忌。

## 目标与监测
- 目标静息心率：55～60次/分；
- 持续监测血压、心率、心功能及缺血症状；
- 避免过度减慢心率或降低血压导致灌注不足。
