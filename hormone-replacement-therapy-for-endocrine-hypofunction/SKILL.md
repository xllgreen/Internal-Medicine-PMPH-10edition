# 内分泌腺功能减退的激素替代治疗原则

## 适用条件
- 患者已确诊为内分泌腺功能减退（如1型糖尿病、原发性甲状腺功能减退症、肾上腺皮质功能减退）
- 已明确缺乏的具体激素类型
- **不适用于**激素受体或受体后缺陷所致的功能障碍
- 急性感染或严重应激未控制时需谨慎调整剂量

## 执行步骤

1. **选择合适激素**
   - 根据缺失激素类型选择对应替代物：
     - 1型糖尿病 → 胰岛素
     - 甲状腺功能减退症 → 左甲状腺素（L-T₄）
     - 肾上腺皮质功能减退 → 糖皮质激素（如氢化可的松）

2. **模拟生理节律给药**
   - 依据激素的化学性质和生理分泌模式决定给药途径（口服、皮下注射等）及时间
   - 例如：糖皮质激素应在清晨给药以模拟晨峰

3. **个体化剂量调整**
   - 基础剂量根据年龄、体重、性别设定
   - **存在应激状态（如手术、感染、创伤）时，糖皮质激素剂量须成倍增加**

4. **疗效监测**
   - L-T₄治疗：定期监测血清TSH水平
   - 胰岛素治疗：监测血糖及HbA1c

5. **管理药物相互作用**
   - 如卡马西平等肝酶诱导剂可加速糖皮质激素代谢，需相应增加剂量

6. **监控不良反应**
   - 胰岛素：警惕低血糖、体重增加
   - 糖皮质激素：注意骨质疏松、高血压、高血糖等

## 输出结果
制定包含药物选择、剂量、给药方式及监测计划的个体化激素替代方案。
