# 急性心衰利尿剂与血管扩张剂应用

## 适用场景
- **利尿剂**：急性心衰伴液体潴留证据，且无未纠正的低灌注状态。
- **血管扩张剂**：急性心衰患者收缩压≥90mmHg，且无组织低灌注。

## 利尿剂使用流程
1. 首选静脉袢利尿剂（呋塞米、托拉塞米或布美他尼）。
2. 对于未接受过口服利尿剂者：
   - 呋塞米 20～40mg 静脉注射，或
   - 托拉塞米 10～20mg 静脉注射。
3. 根据尿量和症状追加剂量，或改为静脉滴注（5～40mg/h）。
4. 剂量限制：
   - 起初6小时内呋塞米 ≤80mg；
   - 起初24小时内呋塞米 ≤160mg。
5. 若长期口服利尿剂，初始静脉剂量应 ≥ 日常口服剂量。
6. 液体负平衡目标：
   - 一般患者：500～1000ml/d；
   - 严重肺水肿：1000～2000ml/d，必要时可达3000～5000ml/d。

## 血管扩张剂使用流程
1. 小剂量起始，缓慢给药，密切监测血压。
2. **硝普钠**（动静脉扩张）：
   - 起始 0.2～0.3μg/(kg·min)，每5～10分钟递增0.5μg/(kg·min)，最大 ≤10μg/(kg·min)；
   - 使用时间 ≤72小时。
3. **硝酸酯类**（主要扩静脉）：
   - 硝酸甘油：起始 5～10μg/min，每5～10分钟递增5～10μg/min，最大 200μg/min；
   - 硝酸异山梨酯：起始 1～2mg/h，每5～15分钟递增1mg/h，最大 10mg/h。
4. **乌拉地尔**（α受体拮抗剂）：适用于合并高血压或主动脉夹层，剂量 100～400μg/min，依血压调整。
5. **重组人脑利钠肽**（rhBNP）：
   - 负荷量 1.5μg/kg 静脉推注；
   - 维持量 0.0075～0.01μg/(kg·min) 持续滴注。

## 注意事项
- 利尿剂禁用于明显低血容量（如大出血、严重脱水）。
- 血管扩张剂禁用于收缩压<90mmHg或明显低血容量者。
- 所有用药过程中需持续监测血压、尿量、症状及电解质。
