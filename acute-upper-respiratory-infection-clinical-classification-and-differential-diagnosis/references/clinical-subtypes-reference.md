# 急性上呼吸道感染临床分型详细标准

## 1. 普通感冒（Common Cold）
- **病原体**：鼻病毒（30–50%）、冠状病毒、呼吸道合胞病毒等
- **特点**：鼻部症状突出（鼻塞、流清涕、打喷嚏），低热或无热，全身症状轻

## 2. 流行性感冒（Influenza）
- **病原体**：甲型/乙型流感病毒
- **特点**：起病急骤，高热（>38.5℃）、显著头痛、肌痛、乏力，呼吸道症状相对较轻

## 3. 咽结膜热（Pharyngoconjunctival Fever）
- **病原体**：腺病毒3、7型
- **特点**：发热、咽炎、结膜炎三联征，常伴颈淋巴结肿大，多见于儿童

## 4. 疱疹性咽峡炎（Herpangina）
- **病原体**：柯萨奇病毒A组（尤其A16）
- **特点**：高热、咽痛，软腭、悬雍垂、扁桃体周围出现灰白色疱疹，破溃后形成溃疡

## 5. 急性咽扁桃体炎（Acute Tonsillopharyngitis）
- **病原体**：A组β溶血性链球菌（GABHS）、EB病毒、腺病毒等
- **特点**：咽痛剧烈，扁桃体红肿伴脓性渗出，可伴颈部淋巴结压痛
