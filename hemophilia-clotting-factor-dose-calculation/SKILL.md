# 血友病凝血因子替代治疗剂量计算

## 何时使用
当满足以下全部条件时应用本技能：
- 患者为血友病A或血友病B；
- 需进行凝血因子替代治疗（按需止血或围手术期预防）；
- 已知患者体重（kg）；
- 已明确需提升的目标凝血因子活性水平（%）；
- 确认无高滴度抑制物存在或已处理。

> **注意**：不适用于合并高滴度抑制物的患者，也不适用于非因子类药物（如艾美赛珠单抗）治疗场景。

## 如何执行
1. **确定目标活性水平**：
   - 最低止血要求：FVIII:C 或 FIX:C ≥ 20%
   - 严重出血或中型以上手术：FVIII 或 FIX 活性水平 ≥ 40%

2. **选择对应公式计算剂量**：
   - **血友病A（FVIII）**：剂量 (IU) = 体重 (kg) × 所需提高的活性水平 (%) ÷ 2
   - **血友病B（FIX）**：剂量 (IU) = 体重 (kg) × 所需提高的活性水平 (%)

3. **调整给药频率**（基于半衰期）：
   - FVIII 半衰期为 8～12 小时 → 每日2次或持续静脉输注
   - FIX 半衰期为 18～24 小时 → 每日1次即可

4. 输出结果为需输注的凝血因子总量（IU），用于临床给药指导。
