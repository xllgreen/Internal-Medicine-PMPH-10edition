# 尿路感染的诊断阈值与标准

## 何时使用
- 已采集清洁中段尿、导尿或膀胱穿刺尿标本
- 已完成尿细菌培养
- 需要根据培养菌落数（CFU/ml）、临床症状和患者特征判断尿路感染类型

## 执行步骤

1. **确认真性细菌尿**（满足任一即成立）：
   - 清洁中段尿沉渣涂片细菌数＞1个/HPF
   - 膀胱穿刺尿细菌培养阳性
   - 清洁中段尿细菌培养计数≥10⁵ CFU/ml

2. **根据临床表现和培养结果判断感染类型**：
   - **急性非复杂性膀胱炎**：典型尿路刺激征 + 培养≥10³ CFU/ml
   - **急性非复杂性肾盂肾炎**：寒战、发热、腰痛等症状 + 培养≥10⁴ CFU/ml
   - **无症状细菌尿（ASB）**：无症状 + 两次同菌种培养均≥10⁵ CFU/ml
   - **特殊情况拟诊**（如女性明显尿路刺激征）：白细胞尿 + 培养≥10² CFU/ml + 常见致病菌（如大肠埃希菌）

3. **诊断复杂性尿路感染需同时满足**：
   - 清洁中段尿培养≥10⁵ CFU/ml
   - 存在泌尿系统结构/功能异常或基础疾病（如糖尿病、留置导尿管等）

4. **排除干扰因素**：
   - 若近7天内使用过抗生素，结果可能假阴性
   - 若标本收集不规范或未低温保存超1小时，结果可能假阳性
   - 本标准不适用于结核性泌尿感染
