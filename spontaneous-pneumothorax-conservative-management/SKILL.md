# 自发性气胸保守治疗指征与吸氧机制

## 适用条件
仅在同时满足以下全部条件时启用保守治疗：
- 肺压缩 ≤ 20%（X线胸片评估）
- 无明显呼吸困难、胸痛等症状
- 初次发作，且高分辨率CT未显示明显肺大疱
- 无血胸或其他并发症

## 执行步骤
1. **卧床休息**：患者应严格卧床，并在气胸发生后24～48小时内密切监测生命体征与症状变化。
2. **高浓度吸氧**：经鼻导管或面罩以10 L/min流量持续吸氧，通过提高PaO₂、降低氮分压，促进胸腔内气体（主要为氮气）向血液弥散，加速肺复张。
3. **镇痛镇静**：可酌情给予对症镇痛或轻度镇静药物缓解不适。
4. **动态观察**：若出现呼吸频率加快、氧饱和度下降、胸痛加剧等恶化迹象，立即终止保守治疗，转为胸腔穿刺或闭式引流。

## 禁忌情形
即使满足上述条件，若患者存在以下任一情况，禁止采用保守治疗：
- 年龄较大（通常>50岁，临床判断为主）
- 合并慢性阻塞性肺疾病（COPD）等基础肺病
- 气胸量虽小但症状显著（如无法平卧、说话断续）

> 注意：气体自然吸收率约为每日1.25%～2.20%，因此需预留足够观察时间，但不得延误干预时机。
