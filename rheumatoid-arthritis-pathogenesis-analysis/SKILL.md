# 类风湿关节炎病理机制分析

## 适用场景
- 患者存在慢性对称性多关节炎且已排除其他类型关节炎
- 出现关节畸形、功能障碍或关节外表现（如皮下结节、肺间质病变、巩膜炎等）
- 需要基于组织病理结果判断RA疾病活动度或指导治疗强度

## 执行步骤

### 1. 判断滑膜炎阶段
- **急性期**：观察是否存在滑膜渗出、中性粒细胞浸润、小血管扩张及间质水肿。
- **慢性期**：确认滑膜是否增厚至5～10层以上，是否形成绒毛样突起（即血管翳）。

### 2. 识别关键病理结构
- **血管翳（pannus）**：检查是否侵入软骨或软骨下骨；若存在，提示强破坏性病变。
- **类风湿结节**：在皮肤或关节隆突部活检中，确认是否具备三层结构（中心纤维素样坏死、中层上皮样细胞环、外层肉芽组织）。

### 3. 评估关节外病理表现
- 若发现中/小动脉或静脉壁淋巴细胞浸润、内膜增生或纤维素样坏死，考虑系统性血管炎。
- 关联临床表现：如肺间质纤维化、巩膜穿孔风险、Felty综合征（脾大+中性粒细胞减少）等。

### 4. 综合判断与输出
- 若存在血管翳或类风湿结节，立即标记为高疾病活动度，提示预后不良，需强化免疫抑制治疗。
- 排除非自身免疫性关节病（如骨关节炎、痛风）及与其他系统性血管炎的混淆。

> **注意**：本技能仅适用于确诊或高度疑似类风湿关节炎的病例，不适用于非自身免疫性关节病。
