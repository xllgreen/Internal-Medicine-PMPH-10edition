# 下丘脑-垂体-靶腺轴反馈调节机制分析

## 何时使用
- 用户提供异常的甲状腺激素、皮质醇或性激素检测结果，需判断病变部位（靶腺 vs 垂体 vs 下丘脑）
- 需要解释为何某些激素升高反而抑制上游激素（负反馈）或在特定情况下促进释放（如排卵前LH峰）
- 设计TRH、CRH或GnRH兴奋试验等动态功能评估方案

## 执行步骤

1. **识别靶腺激素类型及主要反馈部位**：
   - **甲状腺激素（T3/T4）**：主要作用于**垂体**，抑制TSH分泌
   - **肾上腺皮质激素（皮质醇）和性激素（雌二醇、睾酮）**：主要作用于**下丘脑**，抑制CRH或GnRH分泌

2. **判断反馈性质**：
   - **通常为负反馈**：激素浓度升高 → 抑制上游促激素释放
   - **例外**：在女性月经周期中期，高浓度雌二醇对下丘脑前部产生**正反馈**，触发GnRH大量释放，引发LH峰促进排卵

3. **区分反馈环路类型**：
   - **长环负反馈**：靶腺激素直接抑制下丘脑或垂体
   - **短环负反馈**：垂体激素（如ACTH）通过门脉系统或全身循环反向抑制下丘脑CRH分泌

4. **考虑高级中枢影响**：靶腺激素可作用于边缘系统等下丘脑以上中枢，间接调节情绪与内分泌稳态

5. **注意特殊激素的多重作用**：
   - TRH不仅刺激TSH，还可促进PRL分泌
   - 生长抑素（SS）同时抑制GH、胰岛素和胰高血糖素

> **注意**：本机制不适用于kisspeptin主导的青春期启动阶段，且在急性应激状态下反馈调节可能被暂时抑制。
