# 激素相互作用模式分析

## 何时使用
当满足以下条件时应用本技能：
- 多种激素同时或相继作用于同一生理过程（如血糖调节、能量平衡）
- 存在共同的靶器官或共享的代谢通路
- 需解释临床表现、预测药物干预效果或设计联合激素治疗方案

**不适用于**单一激素独立作用或非激素类信号分子主导的过程。

## 执行步骤

1. **确认参与激素组合**：列出当前情境中所有参与调节的激素（如{胰岛素, 胰高血糖素, 糖皮质激素}）。

2. **明确生理结局**：确定这些激素共同影响的目标状态（如血糖升高、血糖降低、维持稳态）。

3. **判断相互作用类型**：根据各激素对生理结局的影响方向与机制，归入以下三类之一：
   - **整合作用（Integration）**：不同激素通过不同路径达成统一目标（如低血糖时，胰岛素↓减少葡萄糖利用，胰高血糖素↑促进糖原分解）。
   - **协同作用（Synergism）**：多种激素以相同方向增强效应（如严重低血糖时，肾上腺素、胰高血糖素、糖皮质激素均升血糖，效果叠加）。
   - **拮抗作用（Antagonism）**：一种激素直接对抗另一种的效应（如胰岛素降血糖 vs 肾上腺素升血糖）。

4. **结合情境动态判断**：注意同一对激素在不同生理状态下可能表现为不同类型，需依据当前代谢状态（如血糖水平）重新评估。

5. **输出结果**：明确指出当前相互作用类型，并说明其对临床解释、治疗设计或药效预测的意义。
