# 腺垂体功能减退症的病因分类

## 适用场景
- 已确认一种或多种垂体前叶激素缺乏
- 排除原发性甲状腺、肾上腺或性腺疾病
- 不适用于孤立性尿崩症或神经垂体功能障碍

## 执行步骤
1. **判断是否为获得性病因**：
   - 垂体及鞍旁肿瘤（垂体瘤、颅咽管瘤等）
   - 垂体缺血坏死（如希恩综合征）
   - 手术/创伤、放射性损伤、感染/浸润（结核、结节病等）
   - 垂体卒中、空泡蝶鞍、垂体炎
2. **若无获得性病因，考虑遗传性**：
   - POU1F1、PROP1、HESX1基因突变 → 多激素缺乏
   - Kallmann综合征、Prader-Willi综合征 → 下丘脑功能紊乱
3. **若上述均排除，归为特发性**
4. **结合机制理解**：
   - 垂体前叶依赖门脉系统供血，缺血易致不可逆损伤
   - 妊娠期腺垂体更易受缺血影响
   - 垂体柄受损阻断下丘脑激素传输
5. **影像学辅助**：首选垂体MRI；垂体炎与垂体瘤常难区分，必要时活检

## 输出要求
输出具体病因类别（获得性/遗传性/特发性）、可能致病机制，并指导后续治疗方向（手术、激素替代、免疫治疗等）。
