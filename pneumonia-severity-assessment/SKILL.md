# 肺炎严重程度评估

## 适用场景
- 患者已确诊为社区获得性肺炎（CAP）
- 已获取意识状态、尿素氮、呼吸频率、血压、年龄等基础临床数据
- 需要判断是否住院或转入ICU

## 执行步骤

### 1. 计算 CURB-65 评分（每项1分，共5分）
- **C**：意识障碍（新发精神状态改变）
- **U**：尿素氮 > 7 mmol/L
- **R**：呼吸频率 ≥ 30 次/分
- **B**：收缩压 < 90 mmHg 或舒张压 ≤ 60 mmHg
- **65**：年龄 ≥ 65 岁

### 2. 根据 CURB-65 评分给出初步处理建议
- **0–1 分**：原则上门诊治疗
- **2 分**：建议住院或严格随访下的院外治疗
- **3–5 分**：应住院治疗

### 3. 应用重症肺炎诊断标准（满足任一即视为重症）
- **主要标准（满足1项即可）**：
  - 需气管插管行机械通气
  - 感染性休克经积极液体复苏后仍需血管活性药物
- **次要标准（满足≥3项）**：
  - 呼吸频率 ≥ 30 次/分
  - PaO₂/FiO₂ ≤ 250 mmHg
  - 多肺叶浸润
  - 意识障碍/定向障碍
  - 血尿素氮 ≥ 7.14 mmol/L
  - 收缩压 < 90 mmHg 需积极液体复苏

### 4. 综合判断
结合患者年龄、基础疾病、社会经济状况、胃肠功能及治疗依从性进行个体化决策。CURB-65 和重症标准不能单独用于ICU收治决定，必须与其他临床信息整合。

## 输出结果
明确推荐治疗场所（门诊、普通病房或ICU），并启动相应监护与支持措施。
