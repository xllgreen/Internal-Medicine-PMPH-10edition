# 静脉补钾安全规范

## 适用条件
- 患者诊断为低钾血症
- 无法经口补钾
- 尿量 >500 ml/天 或 >30 ml/小时
- 肾功能正常或已评估

## 禁忌情形
- 无尿或严重肾功能衰竭未评估者
- 高钾血症患者

## 执行步骤

### 1. 控制补钾速度
- 一般维持在 **20～40 mmol/h**
- **绝对不可超过 50～60 mmol/h**，以防心脏毒性

### 2. 控制补钾浓度
- 常规静脉滴注液体含钾 **20～40 mmol/L**（相当于氯化钾 1.5～3.0 g/L）

### 3. 特殊情况处理
- 对于需限制补液量或不能口服的严重低钾患者：
  - 应行深静脉穿刺/插管
  - 使用精确静脉微量输注泵匀速输注高浓度钾溶液

### 4. 监测要求
- 输注较高浓度钾溶液时，必须进行**持续心脏监护**
- **每小时测定血钾水平**

### 5. 注意事项
- 钾进入细胞内缓慢，细胞内外平衡需约15小时或更久
- 必须严密观察输注中及输注后情况，防止一过性高钾血症

## 关键变量
- **补钾速度**（mmol/h）：每小时输入的钾离子毫摩尔数
- **补钾浓度**（mmol/L 或 g/L）：静脉液体中钾的浓度
- **尿量**（ml/h 或 ml/d）：评估肾脏排钾能力的关键指标
