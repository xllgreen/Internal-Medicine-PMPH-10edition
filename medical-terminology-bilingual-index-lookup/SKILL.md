# 医学术语中英文对照索引查询

## 何时使用
- 用户提供一个中文医学名词，要求其标准英文对应词
- 用户需要知道某术语在原始临床文献中的具体页码
- 用户遇到带缩写的英文术语，需确认其中文译名

## 执行步骤
1. **按英文字母顺序定位索引组**：索引按英文首字母（A–Z）分组，每组下列出所有对应中文术语。
2. **匹配中文术语**：在相应字母组内查找用户提供的中文术语。
3. **提取完整条目信息**：每个条目包含以下三部分：
   - 中文术语
   - 英文术语（含全称及常用缩写，如有）
   - 出现页码（单个或多个）
4. **处理多义或多译情况**：若同一英文术语有多个中文译名（如“非酒精性脂肪肝”与“非酒精性脂肪性肝病”），应一并返回所有变体及其页码。
5. **返回结果**：输出格式为：`中文术语 + 英文术语（含缩写）+ 页码`，用于用户快速定位原文内容。

> 注意：本技能仅适用于标准医学名词、专业术语及临床文献场景，不提供术语定义或解释，也不适用于非医学领域术语。
