# AVRT心电图分类与特征识别

## 触发条件
当心电图显示连续、快速、规则的QRS波群，且心率在150～220次/分时，应用本技能进行AVRT分型。

## 执行步骤

1. **观察QRS波群形态与时限**：
   - 若QRS波群形态正常、时限 ≤ 0.12秒 → 判断为**顺向型AVRT**（占AVRT 90%以上）
   - 若QRS波群宽大畸形、时限 > 0.12秒 → 判断为**逆向型AVRT**

2. **处理特殊情况**：
   - 极少数顺向型AVRT可合并束支传导阻滞或室内差异性传导，表现为宽QRS心动过速
   - 预激综合征合并房颤或房扑时，QRS波群形态多变（可正常或宽大畸形），RR间期不规则，QRS起始部可见δ波

3. **确认关键鉴别点**：
   - **顺向型AVRT**：激动经房室结下传、旁道逆传 → QRS正常
   - **逆向型AVRT**：激动经旁道下传、房室结逆传 → QRS宽大

4. **注意共同特征**：
   - 通常未见明确P波
   - 心率范围固定为150～220次/分

## 输出结果
输出AVRT类型（顺向型或逆向型）及是否合并传导异常（如束支阻滞、室内差异性传导等）。

> 提示：本技能仅适用于已知或高度怀疑存在旁道的患者，不适用于无旁道或非折返性心律失常。
