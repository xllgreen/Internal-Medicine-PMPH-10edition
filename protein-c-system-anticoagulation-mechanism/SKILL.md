# 蛋白C系统抗凝机制

## 何时使用
- 评估维生素K缺乏或华法林治疗对抗凝系统的影响
- 分析内皮损伤（如脓毒症、DIC）导致的抗凝功能障碍
- 解读遗传性蛋白C/蛋白S缺乏症的血栓倾向
- 理解凝血过程中的负反馈调控逻辑

## 执行步骤
1. **确认前提条件**：确保维生素K充足（支持PC/PS合成）、内皮细胞表达凝血酶调节蛋白（TM），且Ca²⁺存在。
2. **启动激活通路**：凝血酶与内皮表面TM以1:1结合，形成复合物。
3. **激活蛋白C**：TM-凝血酶复合物裂解蛋白C（PC），生成活化的蛋白C（APC）。
4. **协同灭活促凝因子**：APC在蛋白S（PS）作为辅因子的协助下，特异性灭活FV和FVIII。
5. **阻断凝血放大**：灭活FV/FVIII可抑制tenase和凝血酶原酶复合物形成，防止凝血酶爆发性生成。
6. **负反馈调节**：凝血酶越多，越能通过此通路自我限制，维持凝血-抗凝平衡。

> 注意：严重内皮损伤或维生素K缺乏将导致该系统失效，增加血栓风险。
